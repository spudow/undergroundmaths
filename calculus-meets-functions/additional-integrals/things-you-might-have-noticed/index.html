Things you might have noticed

Which of the integrals below can be evaluated if we know that

∫30f(x) dx=7 and ∫30g(x) dx=4?∫03f(x) dx=7 and ∫03g(x) dx=4?

How are the functions we are being asked to integrate related to f(x)f(x) and g(x)g(x)?

Are the limits of these new integrals the same as the original integrals, or are they related to the original limits in a way that we can make use of?

As you read, you may find it helpful to sketch some possible graphs of the functions involved. You could consider some particular examples that you’re comfortable working with, or try to work with more general sketch graphs of f(x)f(x) and g(x).g(x).

The limits for this integral coincide with the ones we are given, so now we need to think about the function f(x)+2g(x)f(x)+2g(x).

Consider the graph of y=f(x)+2g(x)y=f(x)+2g(x). For any xx-value the corresponding yy-value is the sum of f(x)f(x) and 2g(x)2g(x). We can think of f(x)+2g(x)f(x)+2g(x) as a sum of heights where the height in this case is the yy-coordinate of each of the functions y=f(x)y=f(x) and y=g(x)y=g(x), so we can treat the integral of f(x)+2g(x)f(x)+2g(x) as a sum of integrals,

∫30(f(x)+2g(x)) dx=∫30f(x) dx+∫302g(x) dx=∫30f(x) dx+∫30g(x) dx+∫30g(x) dx=7+4+4=15.∫03(f(x)+2g(x)) dx=∫03f(x) dx+∫032g(x) dx=∫03f(x) dx+∫03g(x) dx+∫03g(x) dx=7+4+4=15.

Can we make a statement about what this means for integrating ∫ng(x) dx∫ng(x) dx, where nn is any real number?

The limits for this integral coincide with the ones we are given, but since we don’t know anything more about the functions f(x)f(x) and g(x)g(x), we can’t say anything about the function f(x)g(x)f(x)g(x) or its integral.

The limits for the integral coincide with the ones we are given. If we assume that g(x)g(x) is positive between 00 and 33 then we can think about this as a calculation of area: we are adding 22 to the height of the curve for each value of xx in the interval [0,3][0,3], so in total we are adding a rectangle of height 22 and width 33 to the area ∫30g(x) dx∫03g(x) dx. Therefore,

∫30(g(x)+2) dx=∫30g(x) dx+6=4+6=10.∫03(g(x)+2) dx=∫03g(x) dx+6=4+6=10.

What happens if g(x)g(x) is negative for some values of xx between 00 and 33? Can we still evaluate ∫30(g(x)+2) dx=∫30g(x) dx+6∫03(g(x)+2) dx=∫03g(x) dx+6 if that happens?

We know ∫30g(x) dx=4.∫03g(x) dx=4. We also know that ∫30f(x) dx=7∫03f(x) dx=7, but how is this related to ∫03f(x) dx∫30f(x) dx?

By the Fundamental Theorem of Calculus, if the function f(x)f(x) is continuous on the interval [0,3][0,3] and F′(x)=f(x)F′(x)=f(x) then

∫30f(x) dx=F(3)−F(0).∫03f(x) dx=F(3)−F(0).

So,

∫03f(x) dx=F(0)−F(3)=−(F(3)−F(0))=−∫30f(x) dx=−7.∫30f(x) dx=F(0)−F(3)=−(F(3)−F(0))=−∫03f(x) dx=−7.

Therefore,

∫03f(x) dx×∫30g(x) dx=−7×4=−28.∫30f(x) dx×∫03g(x) dx=−7×4=−28.

Here we are integrating over an interval of the same length as before, since [−1,2][−1,2] is a translation of [0,3][0,3] by one unit to the left. This translation of the limits allows us to evaluate the integral ∫2−1f(x+1) dx∫−12f(x+1) dx because the curve has also been translated by one unit in the negative xx direction.

Therefore, ∫2−1f(x+1) dx=∫30f(x) dx=7.∫−12f(x+1) dx=∫03f(x) dx=7.

For the remaining integrals we need more information, but we could try to use the additional pieces of information to help us.

Which of the integrals can be evaluated if we have the following extra information?

  1. f(x)f(x) is symmetric about x=3x=3

If f(x)f(x) is symmetric about x=3x=3 then we know, for example, that f(4)=f(2)f(4)=f(2), f(5)=f(1)f(5)=f(1) and in general, f(3+k)=f(3−k)f(3+k)=f(3−k) for any real number k.k. This means that the line x=3x=3 splits the integral ∫60f(x) dx∫06f(x) dx in half, so ∫60f(x) dx=2∫30f(x) dx=14.∫06f(x) dx=2∫03f(x) dx=14.

How could you use manipulation of the limits of the integrals and functions to show that ∫60f(x) dx=14∫06f(x) dx=14?

  1. g(x)g(x) is an odd function

As g(x)g(x) is odd, g(−x)=−g(x).g(−x)=−g(x). Therefore

∫0−3g(x) dx=∫30g(−x) dx=−∫30g(x) dx.∫−30g(x) dx=∫03g(−x) dx=−∫03g(x) dx.
It follows that ∫3−3g(x) dx=0.∫−33g(x) dx=0.

  1. f(x)f(x) is an even function
