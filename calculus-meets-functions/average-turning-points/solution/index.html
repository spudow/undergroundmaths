Solution

While working on part (c) of Can you find… curvy cubics edition, which asked: “Can you find a cubic curve that has a local minimum when x=−1x=−1?”, two students had the following conversation:

A: I can find a cubic with a stationary point at x=−1x=−1 by having two of the xx-axis intersection points at x=−2x=−2 and x=0x=0, because the stationary point is half-way between the two intersection points. I need a third intersection point, too, not between these, so I may as well choose x=4x=4, so my cubic is y=−x(x+2)(x−4)y=−x(x+2)(x−4) (with a minus sign so the cubic is the right way up).

B: I’m not sure that’s right; is the stationary point really half-way between the xx-axis intersection points?

Can you resolve their debate?

A good start could be to check if student A or student B is right in this case.

Student A suggests the function y=−x(x+2)(x−4)y=−x(x+2)(x−4) has the required features.

Expanding this gives y=8x+2x2−x3y=8x+2x2−x3.

The xx-coordinates of the turning points are the solutions of dydx=0dydx=0, so 8+4x−3x2=08+4x−3x2=0.

We could solve this to get x=2±27⎯⎯√3x=2±273, which does not equal −1−1, or we could just substitute −1−1 into 8+4x−3x28+4x−3x2, getting dydx=1≠0dydx=1≠0. Either way, student A is wrong in this case.

If student A is right in this case, does this approach always work for finding a cubic with a given stationary (turning) point, or only sometimes?

If student B is right in this case, does student A’s approach ever work?

What exactly is student A’s approach? Can you describe it precisely?

Answering this first will ensure that we agree about what we are trying to show!

It seems that student A’s approach can be described as:

  • To have a stationary point at x=px=p, I will put two of my xx-axis intercepts at x=ax=a and x=bx=b, where pp is midway between aa and bb, and my third intercept will be at x=cx=c, where cc is not between aa and bb.

This is what we will work with.

To prove that student A’s approach doesn’t always work, we only need to find one counterexample, and we have now done this.

To find out whether student A’s approach ever works, we either need to find an example where it does or to prove that it never works.

We could test several examples, and that might give us some insight (if they all fail) towards a reason why it never works, or it will give us an example of where it does. But if they all fail, that alone does not constitute a proof that student A’s approach never works.

An alternative is to try to prove that student A’s approach never works. If our proof is successful, we will be done, and if it is not, we may discover an example which works instead.

We will use this second route, and try to prove that student A’s approach never works.

We will start by sketching a cubic with xx-axis intercepts at a<b<ca<b<c, and a turning point between aa and bb at x=px=p. (We don’t assume that pp lies midway between aa and bb; we want to work out whether this is possible.)

We also sketch the corresponding gradient function (derivative) on a separate graph, lined up; this must be a quadratic function.

In this case, it seems that pp is closer to aa than to bb.

Is this always going to be the case? Is there anything we can see from the graph of the cubic f(x)f(x) or its derivative f′(x)f′(x) which we can use to argue either way?

It might be helpful to think about ways in which the two graphs are related.

The “area under the graph” on the derivative graph equals the change in function value on the original graph. We can write this algebraically as:

∫paf′(x)dx=f(p)−f(a)and∫bpf′(x)dx=f(b)−f(p).∫apf′(x)dx=f(p)−f(a)and∫pbf′(x)dx=f(b)−f(p).
(This uses the second fundamental theorem of calculus.) The first integral is positive, as the derivative graph is above the xx-axis, while the second one is negative. Also, as f(a)=f(b)=0f(a)=f(b)=0, we can write these as
∫paf′(x)dx=f(p)and∫bpf′(x)dx=−f(p).∫apf′(x)dx=f(p)and∫pbf′(x)dx=−f(p).

We can illustrate these areas on our graph:

The (unsigned) shaded areas must be the same because the (signed) area above the graph is f(p)f(p) and the (signed) area below the graph is −f(p)−f(p).

So can pp be midway between aa and bb?

If we draw a tangent to the gradient function graph at (p,0)(p,0), as shown, we end up with two similar triangles. The one between x=ax=a and x=px=p has area less than f(p)f(p), whereas the one between x=px=p and x=bx=b has area more than f(p)f(p). But if pp were midway between aa and bb, then the two triangles would be congruent, and so have equal areas. This is a contradiction, and so it is impossible for there to be a case where the stationary point is midway between the roots of a cubic.

We might ask questions about this argument. For example:

  • This argument has been about the left-hand stationary point, the one between x=ax=a and x=bx=b. What about the stationary point between x=bx=b and x=cx=c?

  • Could the stationary point x=px=p be midway between x=ax=a and x=cx=c, perhaps?

  • We have drawn our cubic having a maximum between x=ax=a and x=bx=b. But the original question asked for a minimum at x=−1x=−1, not a maximum! So can we modify our argument for a cubic which has a minimum between x=ax=a and x=bx=b and a maximum between x=bx=b and x=cx=c instead?

In this argument, we will write down the formula for a cubic f(x)f(x) with three xx-axis intercepts (that is, three roots), then differentiate it to find its stationary points. Instead of trying to solve the resulting quadratic, which will be algebraically messy, we will simply test whether the xx value midway between two roots is (or can be) a solution of f′(x)=0f′(x)=0.

A general cubic with three xx-axis intercepts (or three roots) aa, bb and cc can be written as
f(x)=k(x−a)(x−b)(x−c)=k(x3−(a+b+c)x2+(ab+bc+ca)x−abc)f(x)=k(x−a)(x−b)(x−c)=k(x3−(a+b+c)x2+(ab+bc+ca)x−abc)

where kk is some non-zero constant.

We can differentiate this to obtain

f′(x)=k(3x2−2(a+b+c)x+(ab+bc+ca)).f′(x)=k(3x2−2(a+b+c)x+(ab+bc+ca)).

The point midway between x=ax=a and x=bx=b (which are any two roots of the cubic) is x=a+b2x=a+b2, so we substitute this into f′(x)f′(x):
f′(a+b2)=k(3(a+b2)2−2(a+b+c)(a+b2)+(ab+bc+ca)).f′(a+b2)=k(3(a+b2)2−2(a+b+c)(a+b2)+(ab+bc+ca)).
 To simplify the algebra, we multiply both sides by 44 and divide both by kk, so
4kf′(a+b2)=3(a+b)2−4(a+b+c)(a+b)+4(ab+bc+ca)=3(a2+2ab+b2)−4(a2+2ab+b2+ac+bc)+4(ab+bc+ca)=−a2+2ab−b2=−(a−b)2.4kf′(a+b2)=3(a+b)2−4(a+b+c)(a+b)+4(ab+bc+ca)=3(a2+2ab+b2)−4(a2+2ab+b2+ac+bc)+4(ab+bc+ca)=−a2+2ab−b2=−(a−b)2.

So f′(a+b2)=0f′(a+b2)=0 if and only if a−b=0a−b=0, or a=ba=b. And a=ba=b means that the cubic has a double (or triple) root at x=ax=a, so the cubic is tangent to the xx-axis at this point.

Therefore the original Can you find… curvy cubics edition question may have a solution f(x)=k(x+1)2(x−c)f(x)=k(x+1)2(x−c) for some suitable values of kk and cc, but student A’s approach can never work for distinct values of aa, bb and cc: a stationary point of a cubic can never occur midway between two distinct roots of the cubic.

There are some ways to simplify the algebra in this argument.

Firstly, we could notice that a vertical stretch of the function does not change either the xx-axis intercepts or the xx-coordinates of the stationary points, so we may as well assume k=1k=1.

Secondly, translating the function in the xx-direction does not affect the property of having a stationary point lying midway between two xx-axis intercepts. Therefore we can translate the function so that one of the intercepts is at the origin, so our cubic is now simply f(x)=x(x−b)(x−c)=x3−(b+c)x2+bcxf(x)=x(x−b)(x−c)=x3−(b+c)x2+bcx. This is easier to work with, and so the chances of making an algebraic error are reduced.

How do these arguments compare? What advantages does each one offer?
