Things you might have noticed

Quadratic gradients

For each of the following questions, describe what you think will happen before you try it out on the interactivity. Then reflect on your ideas: did the actual behaviour surprise you? And can you explain the observed behaviour? (You might want to start with very simple values for aa, bb and cc.)

  • What happens to the gradients as you change cc?

  • What happens to the gradients as you change aa?

  • What happens to the gradients as you change bb?

  • As we change cc, the curve just translates up or down, so the gradient at a given xx-value does not change.

  • The graph of the gradients is a straight line, and its gradient changes as we change aa.

    The gradient function of y=x2y=x2 is 2x2x, as we discovered in Zooming in. If we stretch this function in the yy-direction by a factor of 22, by setting a=2a=2, we double the gradients, giving a gradient function of 4x4x.

    Does the same sort of thing happen for other values of aa?

    Does changing bb affect the gradient of the gradient function line?

  • Changing bb translates the gradient function. If we add 11 to bb, then the gradient function increases by 11. This makes some sense: b=1b=1 (with a=c=0a=c=0) gives the function y=xy=x, which has gradient 11 everywhere. So if we add xx to our function, it seems reasonable that the gradient should increase by 11. For example, when we go from y=3xy=3x to y=3x+x=4xy=3x+x=4x, the gradient increases from 33 to 44.

Putting these together, to find the gradient of the quadratic function f(x)=ax2+bx+cf(x)=ax2+bx+c, we can start with x2x2, stretch it by a factor of aa to get ax2ax2 (which has what gradient function?), then add on bxbx (which does what to the gradient function?) and then finally add cc. This will give us the final gradient function.

Cubic gradients

Following on from the above ideas, we can predict what will happen for a cubic: starting with x3x3, which appears to have a quadratic gradient function (which quadratic exactly?), we stretch it to obtain ax3ax3, and this stretches the gradient function. We then add on extra terms, whose behaviour we now understand from exploring quadratics: first we add on bx2bx2, then cxcx and finally dd.

Using this idea, we should be able to predict the gradient function of any cubic f(x)=ax3+bx2+cx+df(x)=ax3+bx2+cx+d.

Can you predict what the gradient will be for quartics (equations of the form y=ax4+bx3+⋯y=ax4+bx3+⋯) or polynomials of higher degree?

Some patterns definitely seem to be emerging here! What would you expect the gradient function of x4x4 to be?
