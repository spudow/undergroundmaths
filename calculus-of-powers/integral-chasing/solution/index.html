Solution

Can you find the missing positive numbers aa to dd?

We have four integrals to look at, but only two of them contain just one of the letters—(1) and (3).

We could start with the first integral,

(1) ∫5a10x+3dx=114.(1) ∫a510x+3dx=114.

We can see that the value the integral takes is positive, so we have an area formed between 10x+310x+3 and the xx-axis.

This can only happen when 0<a<50<a<5, since aa is also positive. We now have a good feeling for the sort of value for aa we are looking for.

If we perform the integration we get the following,

(1) ∫5a10x+3dx=[5x2+3x]5a=(5×52+3×5)−(5a2+3a)=114.(1) ∫a510x+3dx=[5x2+3x]a5=(5×52+3×5)−(5a2+3a)=114.

Simplifying the above we find that,

140−5a2−3a⟹5a2+3a−26=114=0140−5a2−3a=114⟹5a2+3a−26=0

The prime factor decomposition of 2626 is 2×132×13. Trying the possible combinations leads us to (5x+13)(x−2)(5x+13)(x−2). Given that aa is positive, we must have a=2a=2.

Is this sensible?

Well, we thought above that 0<a<50<a<5 and we have found a value that fits with this.

Which integral could we tackle next?

(2) ∫92abx⎯⎯√+ax⎯⎯√dx=42(2) ∫2a9bx+axdx=42

For this integral it is a good idea to rewrite the integrand and we can also fill in the value for aa.

∫92abx⎯⎯√+ax⎯⎯√dx=∫94bx12+2x−12dx.∫2a9bx+axdx=∫49bx12+2x−12dx.

Why is it helpful that 99 and 44 are the numbers we require to substitute in?

Performing the integration and solving a linear equation in bb, we find that b=3b=3.

(3) ∫1121x5−1x2dx=c+14(3) ∫1211x5−1x2dx=c+14

This integral does not depend on any of the unknowns. It is again useful to rewrite the integrand.

∫1121x5−1x2dx=∫112x−5−x−2dx.∫1211x5−1x2dx=∫121x−5−x−2dx.

Performing the integration and solving a linear equation in cc, we find that c=10c=10.

(4) ∫c+26xba(x⎯⎯√−1x⎯⎯√)dx=abada(4) ∫6c+2xba(x−1x)dx=abada

This equation contains all four of the unknowns so is best kept til last. As before, we start by rewriting the integrand and substituting in the values for aa, bb and cc.

∫c+26xba(x⎯⎯√−1x⎯⎯√)dx=∫126x32(x12−x−12)dx=∫126x2−xdx.∫6c+2xba(x−1x)dx=∫612x32(x12−x−12)dx=∫612x2−xdx.

We can also substitute our values for aa, bb and cc into the right hand side of the original equation to give abada=2×32×d2abada=2×32×d2.

∫126x2−xdx=18d2.∫612x2−xdx=18d2.

Performing the integration and solving a quadratic equation in dd, we find that d=5d=5.

The area formed between the xx-axis and the lines x=bx=b and x=dx=d, and the curve y=(x−2a)(x+1)y=(x−2a)(x+1) is cda(a+b)cda(a+b).

To check that we have found the correct values for aa, bb, cc and dd we can evaluate the integral between the given xx values,

∫db(x−2a)(x+1)dx=∫53(x−4)(x+1)dx∫bd(x−2a)(x+1)dx=∫35(x−4)(x+1)dx

and check it is equal to cda(a+b)=5cda(a+b)=5.

But we are not told about the integral, we are told that the area formed between the curve and the xx-axis is 55. This means we need to be careful if our curve crosses the xx-axis.

Sketching the curve y=(x−4)(x+1)=x2−3x−4y=(x−4)(x+1)=x2−3x−4 reveals that the curve crosses the xx-axis at x=4x=4. So we must separately calculate the areas between the xx values 33 and 44, and 44 and 55.

We can thus split our integral up into two

∫43x2−3x−4dx,∫34x2−3x−4dx,

which will give the negative value −136−136 and

∫54x2−3x−4dx,∫45x2−3x−4dx,

which will give the positive value 176176.

Therefore, the total area formed between the curve, the lines x=3x=3 and x=5x=5, and the xx-axis is 136+176=5136+176=5.
