Review question

When is the area of a face of this pyramid a minimum?

Solution

A pyramid of given volume VV stands on a horizontal square base of edge 2x2x, and its vertex is vertically above the centre of the base. Show that the area AA of a sloping triangular face is given by

A2=9V216x2+x4,A2=9V216x2+x4,

The volume VV of a square based pyramid is

V=13bh,V=13bh,
where bb is the area of the base and hh is the height.

To see where this formula comes from (it holds for pyramids with any base, and cones too) consider a unit cube:

Now draw in the four diagonals from each of the pairs of opposite corners like this:

This makes six identical square based pyramids from each of the faces:

Each pyramid has a base of area one and is of height 1/21/2. Each one must also have a volume of 1/61/6, which is 13bh13bh.

We can extend this for cubes of a different side length, and cuboids.

Our pyramid looks like this:

So we have

V=13(2x)2h.V=13(2x)2h.

We need to find the area of a sloping triangular face (which will be the same for all four faces, since the vertex is directly above the centre of the base).

For this, we need the height yy of the face, which is related to hh and xx via

y2=h2+x2,y2=h2+x2,
which we can see from taking a vertical cross-section of the pyramid.

The area of the face is A=12(2x)y=xyA=12(2x)y=xy.

Therefore,

A2=x2y2=x2(h2+x2)=x2((3V4x2)2+x2)=x2(9V216x4+x2)=9V216x2+x4,A2=x2y2=x2(h2+x2)=x2((3V4x2)2+x2)=x2(9V216x4+x2)=9V216x2+x4,
as required.

and prove that, as xx varies, AA is least when the face is equilateral.

We can say that AA has a turning point if and only if A2A2 does, since we know AA is positive and y=x2y=x2 is an increasing function for x>0x>0.

If you know the Chain Rule for differentiating, you might think about d(A2)dx=2AdAdxd(A2)dx=2AdAdx.

So considering

d(A2)dx=4x3+9V162x3=4x39V28x3,d(A2)dx=4x3+9V162x3=4x39V28x3,
we find stationary points when d(A2)dx=0.d(A2)dx=0.
4x39V28x3=032x6=9V2.4x39V28x3=032x6=9V2.
 There will be two real roots of this equation, one negative (which doesnt interest us) and one positive, so there is one stationary point in the x>0x>0 region.

We know

d(A2)dx=32x69V28x3,d(A2)dx=32x69V28x3,
so if xx increases slightly from (9V232)16(9V232)16, then x6x6 increases slightly from 9V2329V232, (since y=x6y=x6 is an increasing function for x>0x>0), which tells us that the gradient of y=A2y=A2 is positive. Similarly if xx decreases slightly from (9V232)16(9V232)16, the gradient is negative.

So our single stationary point for x>0x>0 must be a local minimum, and we know that A2A2 (and hence AA) is minimized when x6=9V232x6=9V232.

Now h2=9V216x4=9V232x62x2h2=9V216x4=9V232x62x2, and so if x6=9V232x6=9V232, then h2=2x2h2=2x2, and y2=x2+h2=3x2y2=x2+h2=3x2.

So y=3xy=3x, and each face must be an equilateral triangle.
