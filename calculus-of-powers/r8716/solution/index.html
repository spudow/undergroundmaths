Calculus of Powers
Review question

What happens if we integrate, then differentiate?

Solution

For a positive number aa, let

I(a)=∫a0(4−2x2)dx.I(a)=∫0a(4−2x2)dx.

Then dIda=0dIda=0 when aa equals

  1. 1+5√21+52,

  2. 2⎯⎯√2,

  3. 5√−125−12,

  4. 11.

Approach 1

We want to find a stationary point of I(a)I(a), though it may not be clear exactly how to differentiate an expression in this form. As I(a)I(a) is the (signed) area of the curve above the xx-axis between 00 and aa, we want to find a point in which the area is neither increasing nor decreasing.

Let f(x)=4−2x2f(x)=4−2x2, so that I(a)=∫a0f(x)dxI(a)=∫0af(x)dx. Then if f(a)>0f(a)>0, the area must increase if aa increases by a small amount. Similarly if f(a)<0f(a)<0, the area must decrease. This is by considering that the new areas added will be entirely above and entirely below the xx-axis in each case respectively. This picture also tells us that the rate at which the area changes is smallest when f(a)f(a) is smallest. Therefore, the stationary points must occur when f(a)=0f(a)=0.

So dIda=0dIda=0 when 4−2a2=04−2a2=0. Hence a2=2a2=2, which gives that a=2⎯⎯√a=2 (as aa is positive).

Therefore the answer is (b).

Approach 2

We may recall that, in many ways, integration can be seen as the opposite of differentiation. Therefore we may initially guess that dIda=4−2a2dIda=4−2a2. To show that this is true, suppose that 4−2x24−2x2 integrates to the function F(x)F(x). That is, FF is the function satisfying F′(x)=4−2x2F′(x)=4−2x2. Then we have that

I(a)=∫a0(4−2x2)dx=[F(x)]a0=F(a)−F(0)I(a)=∫0a(4−2x2)dx=[F(x)]0a=F(a)−F(0)

and so

dIda=F′(a)=4−2a2dIda=F′(a)=4−2a2

as F(0)F(0) is a constant. Hence the derivative is 00 when a=2⎯⎯√a=2, similarly to above.

As before, the answer is (b).

In the two arguments above, you may not be completely convinced by some of the assertions that have been made. For example, in the first approach, we asserted that if f(a)>0f(a)>0, then the area must increase if aa increases by a small amount. What if ff decreases very quickly? Would there always be some amount by which we can increase aa so that the area still increases? In the second approach, if we cannot write down an expression for the integral F(x)F(x) explicitly, must it always exist? This result is known as the fundamental theorem of calculus and is explored when studying mathematics at university.
