Calculus of Powers
Review question

If we increase xx by a small amount, what happens to yy?

Solution

Given that y=4x3−7x2y=4x3−7x2, find the value of dy/dxdy/dx at the point (2,4)(2,4).

We know that the derivative of xnxn is nxn−1nxn−1. Using this, we have that

dydx=y′=4×3x2−7×2x=12x2−14x.dydx=y′=4×3x2−7×2x=12x2−14x.
So we have that the value of dydxdydx at the point (2,4)(2,4) is
y′(2)=12×22−14×2=48−28=20.y′(2)=12×22−14×2=48−28=20.

Hence find the approximate increase in xx which will cause yy to increase from 44 to 4.054.05.

Consider the following graph of a curve y=f(x)y=f(x).

The slope of the tangent is close to the slope of the chord; that is, to use the notation in the figure,
δyδx≈dydxwhen δx is small.δyδx≈dydxwhen δx is small.
 So if the xx-coordinate of the leftmost point in this figure is denoted by x0x0, then
f(x0+δx)−f(x0)δx≈f′(x0)when δx is small.(1)(1)f(x0+δx)−f(x0)δx≈f′(x0)when δx is small.

In fact, the definition of f′(x0)f′(x0) is the value towards which

f(x0+δx)−f(x0)δxf(x0+δx)−f(x0)δx
tends as δxδx tends towards zero.

In our case:

  • f(x)=4x3−7x2f(x)=4x3−7x2,
  • x0=2x0=2,
  • f(x0)=4f(x0)=4, and
  • f′(x0)=20f′(x0)=20.

If we choose δxδx so that f(x0+δx)=4.05f(x0+δx)=4.05, then, from (1)(1), we would have

4.05−4δx=f(x0+δx)−f(x0)δx≈f′(x0)=20,4.05−4δx=f(x0+δx)−f(x0)δx≈f′(x0)=20,
and so, by rearranging,
δx≈4.05−420=1/2020=1400=0.0025.δx≈4.05−420=1/2020=1400=0.0025.

If we use our calculators, we find that

f(2.0025)−f(2)=0.05010…,f(2.0025)−f(2)=0.05010…,
and so our approximation is a good one!
