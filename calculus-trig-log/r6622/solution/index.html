Calculus of Trigonometry & Logarithms
Review question

Can the integral of sin(sint)sin⁡(sin⁡t) be zero?

Ref: R6622

Solution

In the region 0<x≤2π0<x≤2π, the equation

∫x0sin(sint)dt=0∫0xsin⁡(sin⁡t)dt=0
has

  1. no solution;

  2. one solution;

  3. two solutions;

  4. three solutions.

Can we evaluate this integral? It looks to be hard - impossible even! Instead, we’ll try to pick up some clues about the shape of y=sin(sint)y=sin⁡(sin⁡t).

Clue 1. sin(sint)=0sin⁡(sin⁡t)=0 if and only if sint=0sin⁡t=0, which happens just when t=nπt=nπ, so that’s at ππ and 2π2π on our interval (and at 00).

Clue 2. sintsin⁡t is positive for 0<t<π0<t<π, taking values from 00 to 11 on the interval, so sin(sint)sin⁡(sin⁡t) is also positive on this interval, taking values from 00 to sin1sin⁡1.

Clue 3. sintsin⁡t is negative for π<t<2ππ<t<2π, taking values from 00 to −1−1 on this interval, so sin(sint)sin⁡(sin⁡t) is also negative on this interval, taking values from 00 to −sin1−sin⁡1.

So the curve has a positive section from 00 to ππ, and a negative section from ππ to 2π2π.

Clue 4. The function y=sinty=sin⁡t is periodic; is y=sin(sint)y=sin⁡(sin⁡t)? We have sin(sin(t+2π))=sin(sint)sin⁡(sin⁡(t+2π))=sin⁡(sin⁡t), so yes, sin(sint)sin⁡(sin⁡t) is periodic.

Could the period be less than 2π2π? Clues 2. and 3. suggest not. We now know the section immediately on the other side of the origin is a negative section.

Clue 5. Could sin(sint)sin⁡(sin⁡t) be odd or even? We have sin(sin−t)=sin(−sint)=−sin(sint)sin⁡(sin−t)=sin⁡(−sin⁡t)=−sin⁡(sin⁡t), and so y=sin(sint)y=sin⁡(sin⁡t) is odd, and so has rotational symmetry about the origin.

This must mean that the negative section of the curve from −π−π to 00 matches exactly with the positive section from 00 to ππ (and the section from ππ to 2π2π, by periodicity).

We have enough now to answer the question; the (positive) area below the curve from 0 to ππ matches exactly with the (negative) area under the curve from ππ to 2π,2π, so there is just one value of xx when the integral is zero (x=2πx=2π) and the answer is (b).
