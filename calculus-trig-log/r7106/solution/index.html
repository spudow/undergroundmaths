Calculus of Trigonometry & Logarithms
Review question

Can we find this integral involving the floor function?

Solution

For a real number xx we denote by [x][x] the largest integer less than or equal to xx.

Let nn be a natural number. The integral

∫n0[2x]dx∫0n[2x]dx
equals

  1. log2((2n−1)!)log2⁡((2n−1)!);

  2. n2n−log2((2n)!)n2n−log2⁡((2n)!);

  3. n2nn2n;

  4. log2((2n)!)log2⁡((2n)!),

where k!=1×2×3×⋯×kk!=1×2×3×⋯×k for a positive integer kk.

The function [x][x] is called the floor function (for obvious reasons). It can also be written as ⌊x⌋⌊x⌋.

You might wonder if there is a ceiling function, and indeed there is; ⌈x⌉⌈x⌉ represents the smallest integer greater than or equal to xx.

We can see that, in the range 0≤x≤n0≤x≤n, [2x][2x] will take all integer values from 11 to 2n2n. More precisely, we have

[2x]=kforlog2k≤x<log2(k+1).[2x]=kforlog2⁡k≤x<log2⁡(k+1).

This means we can turn the integral into a sum of rectangles which equals

(log22−log21)+2(log23−log22)+3(log24−log23)+⋯+(2n−1)(log22n−log2(2n−1)).(log2⁡2−log2⁡1)+2(log2⁡3−log2⁡2)+3(log2⁡4−log2⁡3)+⋯+(2n−1)(log2⁡2n−log2⁡(2n−1)).

There is a lot of cancellation here, which leaves us with
−log21−log22−log23−⋯−log2(2n−1)+(2n−1)log22n=2nlog22n−(log21+log22+log23+⋯+log22n)=n2n−log2((2n)!).−log2⁡1−log2⁡2−log2⁡3−⋯−log2⁡(2n−1)+(2n−1)log2⁡2n=2nlog2⁡2n−(log2⁡1+log2⁡2+log2⁡3+⋯+log2⁡2n)=n2n−log2⁡((2n)!).

So the answer is (b).

An alternative way to think about the area is as horizontal strips instead of vertical strips.

The strip between height k−1k−1 and kk has length n−log2kn−log2⁡k, and so this is its area, too.

Adding these up for k=1k=1 through to k=2nk=2n gives the same answer.

The strip between height 2n−12n−1 and 2n2n has zero width and zero area.
