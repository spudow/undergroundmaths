Solution

These statements and explanations can be used to create a chain of reasoning to find the gradient function of f(x)=axf(x)=ax. Sort the statements into a logical argument, either writing your own explanations of the steps, or using the explanations provided.

f′(x)=limh→0(f(x+h)−f(x)h)f′(x)=limh→0(f(x+h)−f(x)h)

This is the general formula for differentiating from first principles.

f′(x)=limh→0(ax+h−axh)f′(x)=limh→0(ax+h−axh)

This is an expression for the gradient of f(x)=ax.f(x)=ax.

Remember the expression in the brackets gives the gradient of the chord between xx and x+hx+h. Taking the limit as h→0h→0 gives us the gradient of the function at xx.

f′(x)=limh→0(ax(ah−1h))f′(x)=limh→0(ax(ah−1h))

The expression is factorised as axax is a common factor.

f′(x)=axlimh→0(ah−1h)f′(x)=axlimh→0(ah−1h)

As axax is not affected by hh it can be taken outside of the limit.

The value of hh is tending to 00, so any expressions not involving hh are unaffected by this limiting process.

The expression in the limit gives the gradient of a chord between x=0x=0 and x=hx=h on f(x)=axf(x)=ax.

Think about how the expression ah−a0hah−a0h could be represented on the graph of f(x)=axf(x)=ax. What does it show?

f′(x)=ax×f′(0)f′(x)=ax×f′(0)

As h→0h→0 the limit will give the gradient function at x=0x=0.

The final step implies that limh→0ah−a0h=f′(0)limh→0ah−a0h=f′(0).

This is not a simple limit to find as both the numerator and denominator tend to zero as h→0h→0. However, if we look at how we find f′(x)f′(x) from first principles we should see some similarities.

f′(x)=limh→0f(x+h)−f(x)h and limh→0ah−a0hf′(x)=limh→0f(x+h)−f(x)h and limh→0ah−a0h
Instead of finding the gradient for a generic point xx, the limit on the right gives us the gradient for the specific point x=0x=0, i.e. f′(0)f′(0).

What does your final answer tell you about the gradient function of f(x)f(x)?

  • What does the graph of f′(x)f′(x) look like?

f′(x)=ax×f′(0)f′(x)=ax×f′(0)

The gradient function is the original function multiplied by a constant. This tells us the graph of f′(x)f′(x) will look very similar to f(x).f(x).

In fact it will be a transformation of the original function; a stretch in the yy direction, scale factor f′(0).f′(0). (Or a stretch in the xx direction, scale factor 1f′(0).1f′(0).)

What do the graphs tell you about f′(0)f′(0) when a=2a=2 and when a=4a=4?

  • How will the gradient function change as aa changes?

We can see from the graphs above that as aa gets bigger, f(x)=axf(x)=ax gets steeper more quickly and so the gradient at zero will be larger. Therefore, as aa gets bigger, f′(x)f′(x) will also get steeper more quickly. What will happen as aa gets smaller?

  • There is a special case of the gradient function of axax. Can you think what it might be? To find out more look at A special case.

The gradient function itself could be steeper than the original function. This will happen if the stretch factor is bigger than one, i.e. f′(0)>1f′(0)>1. It could also be less steep, i.e. when f′(0)<1.f′(0)<1. This implies there will be a special case in the middle when f′(0)=1f′(0)=1 and the gradient function and original function will be exactly the same!
