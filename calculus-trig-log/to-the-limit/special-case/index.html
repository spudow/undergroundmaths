A special case

We have found that the gradient function of axax can be written as

f′(x)=ax×f′(0),f′(x)=ax×f′(0),

i.e. the gradient function is the function itself multiplied by a constant, and this constant is the gradient of axax at x=0x=0. There will be a special case when f′(0)=1f′(0)=1 as then the gradient function of axax would be itself, axax.

Does the gradient function look as you expected?

We find that f′(0)≈1f′(0)≈1 when a=2.72a=2.72. This means that the gradient function becomes f′(x)=axf′(x)=ax, so axax and its gradient function are identical.

Of course this value of aa is an approximation, but does it look familiar?

It turns out that the exact value of aa for which the gradient function is the same as the original function is a=2.7181...a=2.7181..., in fact it is a=ea=e. You have probably met ee before as the exponential function exex and as the base of natural logarithms. Here we have found that if f(x)=exf(x)=ex, then f′(x)=exf′(x)=ex.
