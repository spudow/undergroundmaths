Things you might have noticed

In Mapping a derivative, we saw that we can think of a derivative as a local scaling. What happens if we now want to find the derivative of a composite of two functions?

  • What does the mapping diagram of the composite h(x)=g(f(x))h(x)=g(f(x)) look like if f(x)=x2+3f(x)=x2+3 and g(x)=x⎯⎯√g(x)=x (so that h(x)=x2+3⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√h(x)=x2+3), and we centre our mapping diagram at 11 on the input numberline?

  • How does the diagram change as we zoom in?

  • What does this tell us about the derivative h′(1)h′(1)?

What is going on here?

  • The composite function h(x)h(x) appears to be a local scaling, as all of the lines meet at a point. From the diagram, the scale factor appears to be 1212.

  • The function f(x)=x2+3f(x)=x2+3 is locally a scaling with scale factor 22. We can see this either by looking at the mapping diagram carefully, or by noting that the derivative of f(x)f(x) is f′(x)=2xf′(x)=2x, so that the scale factor at x=1x=1 is f′(1)=2f′(1)=2.

  • The function g(x)=x⎯⎯√g(x)=x is also locally a scaling when we zoom in. The scale factor is a little harder to make out reliably from the diagram, but we can use what we know about differentiation to calculate it. We have g′(x)=12x⎯⎯√g′(x)=12x. The function gg is centred on x=f(1)=4x=f(1)=4 in this diagram, so the local scale factor is g′(f(1))=g′(4)=124⎯⎯√=14g′(f(1))=g′(4)=124=14.

  • Composing these, we obtain a local scaling with scale factor 2×14=122×14=12, multiplying the scale factors as we saw in the Warm-up section. This calculated composite scale factor agrees with what we observed about the function h(x)h(x).

So we conclude that h′(1)=12h′(1)=12.

  • Generalising this, given any composite function h(x)=g(f(x))h(x)=g(f(x)), how can we find h′(a)h′(a) for a given aa?

Here is a somewhat generic sketch of a composite mapping diagram for h(x)=g(f(x))h(x)=g(f(x)), where we have shown the arrows and lines for h(x)h(x) but not for the individual functions f(x)f(x) or g(x)g(x):

The function f(x)f(x) gives a local scaling with scale factor f′(a)f′(a), the derivative of ff at aa.

The function g(x)g(x) is centred on f(a)f(a), so it gives rise to a local scaling with scale factor g′(f(a))g′(f(a)), the derivative of gg at f(a)f(a).

Since h(x)h(x) is locally the composition of these two scalings, we see that h′(a)h′(a) is also a local scaling. The scale factor of the composition is the product of the scale factors of ff and gg, so h′(a)=f′(a)×g′(f(a))h′(a)=f′(a)×g′(f(a)).

This is the chain rule, which can therefore be written:

if h(x)=g(f(x)), then h′(x)=g′(f(x)).f′(x).if h(x)=g(f(x)), then h′(x)=g′(f(x)).f′(x).

There is another common way of writing the chain rule, which we can get by setting u=f(x)u=f(x) and y=h(x)y=h(x), so that y=g(f(x))=g(u)y=g(f(x))=g(u). We then have f′(x)=dudxf′(x)=dudx and g′(f(x))=g′(u)g′(f(x))=g′(u), which we can write as dydudydu. Substituting this into the previous version then gives us:

dydx=dydu×dudx.dydx=dydu×dudx.
