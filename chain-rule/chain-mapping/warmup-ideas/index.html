Warm-up ideas

In Mapping a function, we explored mapping diagrams for linear functions. What happens if we now compose two functions?

  • How could we draw a mapping diagram to show the composition of the function f(x)=32xf(x)=32x with the function g(x)=2xg(x)=2x to get the function h(x)=g(f(x))h(x)=g(f(x))?

We can represent this composite by using three numberlines, with arrows from the left to middle one representing f(x)f(x), and then arrows from the middle to the right representing g(x)g(x), like this:

Here, we have centred the left-hand numberline on 22; the numberlines are centred in such a way that the arrow starting from 22 is horizontal, as before.

We can then show the composite function h(x)=g(f(x))h(x)=g(f(x)) by joining the starts to the ends of the arrow chains, like this:

This shows how the function hh takes values from the left-hand numberline to values on the right-hand numberline.

As in Mapping a function, we can also extend the arrows beyond the numberlines. For clarity, we only do this for the composite function hh:

The lines all meet in a point, showing that h(x)h(x) is a scaling with a scale factor of 33. (This is the scaling factor from the left-hand to right-hand numberline.) As h(x)=g(f(x))=g(32x)=3xh(x)=g(f(x))=g(32x)=3x, this is to be expected.

We can also check that the formula we worked out in Mapping a function for the location of the focal point gives us the expected location: that formula gives the location as 11−3=−1211−3=−12 times the distance between the numberlines. This matches what we see in the diagram, because the numberlines of interest here are the left-hand and right-hand ones.

  • We described linear functions as scalings. How are the scalings represented by the three functions ff, gg and hh related to each other?

f(x)=32xf(x)=32x is a scaling with scale factor 3232 and g(x)=2xg(x)=2x is a scaling with scale factor 22. The composite function h(x)=3xh(x)=3x is a scaling with scale factor 33.

In general, if we compose a scaling with scale factor ss together with a scaling with scale factor tt, we will get a scaling with scale factor stst. In our particular case, we have 32×2=332×2=3.

Here, we used linear functions with no constant term and centred our mapping diagram on 00. Would we have obtained the same results if we had started with the functions f(x)=32x+4f(x)=32x+4 and g(x)=2x−3g(x)=2x−3, still with h(x)=g(f(x))h(x)=g(f(x)), and centred our mapping diagram on 77 (as input value)?
