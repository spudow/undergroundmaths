Differentiating exponentials
Add to your resource collection

Printable/supporting materials Fullscreen mode Teacher notes

We already know that the exponential function exex is its own tangent/gradient function. However, there are other exponential functions such as 2x2x, 3x3x and so on.

Are their tangent/gradient functions the same or does something else happen?

What can we say about 1x1x or 0.5x0.5x, for example?

Here is a series of structured questions leading to finding the derivative of axax with respect to xx for a positive number aa.

If ek=2ek=2, what is kk?

We can apply the natural logarithm to both sides of our equation to find that lnek=ln2ln⁡ek=ln⁡2. Using the laws of logarithms we find

lnek=klne=k,ln⁡ek=kln⁡e=k,
so that k=ln2k=ln⁡2.

Which is steeper, exex or 2x2x?

Can we write 2x2x in the form ekxekx, where kk is a real number?

We can write 22 as eln2eln⁡2 and so 2x=(eln2)x=exln22x=(eln⁡2)x=exln⁡2.

Does this support the idea that exex is steeper than 2x2x?

What is the derivative of ekxekx with respect to xx, where kk is a real number?

Using the substitution u=kxu=kx and the knowledge that eueu when differentiated with respect to uu is eueu, the chain rule gives

ddx(ekx)=ddu(eu)×dudx=eu×k=kekx.ddx(ekx)=ddu(eu)×dudx=eu×k=kekx.

How can we differentiate 2x2x with respect to xx?

We can rewrite 2x2x as exln2exln⁡2 and differentiate exln2exln⁡2 using our answer to the question above (since ln2ln⁡2 is a real number). So we have the following:

ddx(2x)=ddx(exln2)=(ln2)exln2=2xln2.ddx(2x)=ddx(exln⁡2)=(ln⁡2)exln⁡2=2xln⁡2.

How does this change for 3x3x?

Which is steeper, exex or 3x3x and how do your calculations support your answer?
