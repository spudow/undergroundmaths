  • write down a substitution that could have been used to solve each integral?

To help us find the correct substitutions let us think about the chain rule. We should be familiar with how we differentiate a composite function.

ddx(g(f(x)))=g′(f(x))×f′(x)ddx(g(f(x)))=g′(f(x))×f′(x)

If we integrate both sides we can write

g(f(x))=∫g′(f(x))×f′(x)dx.g(f(x))=∫g′(f(x))×f′(x)dx.

This integral is made up of a composite function multiplied by the differential of the inner function.

If you spot this pattern in an integral, you may be able to integrate it by inspection without needing to substitute. However, as integrals get more complex, it will help to find a substitution that will reduce the integral to

g(u)=∫g′(u)du.g(u)=∫g′(u)du.

Taking a closer look at the integrals above, we find that B, C and F all include a function of x3−3xx3−3x, as well as a multiple of 3x2−33x2−3, which is the derivative of x3−3xx3−3x. This suggests that the substitution u=x3−3xu=x3−3x will work for all three integrals.

Try using this substitution to check that it works. Are there any other options for the substitution?

For G we see that 1sin3x1sin3⁡x is the composite function. As sin3xsin3⁡x is more clearly written as (sinx)3(sin⁡x)3 we see that sinxsin⁡x is the inner function and so u=sinxu=sin⁡x is a suitable substitution.

For H it is less obvious, but remember 1sinx1sin⁡x is a composite function of 1x1x and sinxsin⁡x, so we can use the same substitution of u=sinxu=sin⁡x.

Why can’t we use the substitution u=cosxu=cos⁡x? What happens if we do?
