The first thing you might notice is that three of the equations are based on trigonometric functions while the other three are based on powers of tt.

What might that tell us about the kind of curves they represent?

Match up each equation (blue card) with

  • the location of a stationary point on the curve (green card),

To find the stationary points on a curve, we want to find dydxdydx and when it is equal to zero.

Taking equation (E1), we have
x=1+4cost⟹dxdt=−4sint,y=t+3sint,dydt=1+3cost.x=1+4cos⁡t,y=t+3sin⁡t⟹dxdt=−4sin⁡t,dydt=1+3cos⁡t.
 We can combine these using the chain rule.
dydx=dy/dtdx/dt=1+3cost−4sintdydx=dy/dtdx/dt=1+3cos⁡t−4sin⁡t
 This is zero when
1+3cost=0⟹cost=−13.1+3cos⁡t=0⟹cos⁡t=−13.
 Substituting this back into the equation tells us the curve is stationary when
x=1+4(−13)=−13.x=1+4(−13)=−13.

This therefore goes with card (S4).

Note that we quietly ignored the sintsin⁡t in the denominator of dydxdydx when setting it equal to zero. We should really check that the denominator is not zero at the same point as the numerator is zero, because if it were the gradient would be undefined. In this case, that doesn’t happen (sint≠0sin⁡t≠0 when cost=−1/3cos⁡t=−1/3) so we are OK.

  • the location of a critical point on the curve where the tangent is vertical (pink card),

At a critical point the tangent is vertical and dydxdydx is undefined. This will happen when our expression for dydxdydx is a fraction with zero in the denominator and non-zero in the numerator.

Taking equation (E5), we have
x=t3−t⟹dxdt=3t2−1,y=t3−3t,dydt=3t2−3.x=t3−t,y=t3−3t⟹dxdt=3t2−1,dydt=3t2−3.
 We could combine these to write dydxdydx as a fraction, but in fact we are only interested in when dxdt=0dxdt=0.

As above, we should also check that dydt≠0dydt≠0 at the same value of tt. In this case, they are quadratics in tt differing only by a constant, so they can never be zero at the same time.

So the critical points happen at
3t2−1⟹xandy=0⟹t=±13⎯⎯√=±(133⎯⎯√−13⎯⎯√)=±233⎯⎯√=±(133⎯⎯√−33⎯⎯√)=±833⎯⎯√3t2−1=0⟹t=±13⟹x=±(133−13)=±233andy=±(133−33)=±833

This therefore goes with card (V4).

  • its graph (white card).

We can use the above features of the curve to work out which of the graphs it goes with. Taking equation (E2) as an example,

x=2cost,y=sin2t,x=2cos⁡t,y=sin⁡2t,
we find it has

  • stationary points at x=±2⎯⎯√x=±2,
  • critical points at (2,0)(2,0) and (−2,0)(−2,0).
Also, since we know the range of the sinsin and coscos functions, we can deduce that
−2≤x≤2and−1≤y≤1−2≤x≤2and−1≤y≤1

so the whole curve must be contained within a rectangle. From these clues, the only graph card it can match is (C2).

If we wanted an extra clue to help identify the graph, we could find the xx-intercepts and/or yy-intercepts. This curve, for instance, has xx-intercepts at x=0x=0 and x=±2x=±2, which confirms the match with (C2).
