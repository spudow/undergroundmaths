Chain Rule & Integration by Substitution

Powerful derivatives

Solution

In Binomials are the answer!, we proved that the derivative of xnxn is nxn−1nxn−1 for any positive integer nn.

Using exponentials and/or logarithms, can you now prove that the derivative of xaxa (where x>0x>0 and aa is any real number) is axa−1axa−1?

If we write x=elnxx=eln⁡x, then xaxa becomes ealnxealn⁡x. If we write u=alnxu=aln⁡x, then we can differentiate this using the chain rule (in the form dydx=dydu×duydxdydx=dydu×duydx, with y=euy=eu):
ddx(ealnx)=ddu(eu)×ddx(alnx)=eu×ax=ealnx×ax=xa×ax=axa−1ddx(ealn⁡x)=ddu(eu)×ddx(aln⁡x)=eu×ax=ealn⁡x×ax=xa×ax=axa−1

as we wanted.

This idea of writing xaxa as ealnxealn⁡x is actually the simplest and most useful way to define arbitrary powers. If we tried to define powers in a different way, it would turn out to be quite difficult to prove that the derivative of xπxπ is πxπ−1πxπ−1, for example.

You may also have thought of proving this result using the general binomial theorem using a similar approach to that in Binomials are the answer!, as follows.

The derivative of xaxa is the limit of (x+h)a−xah(x+h)a−xah as h→0h→0. Then using the binomial theorem, we have
(x+h)a=xa(1+hx)a=xa(1+a⋅hx+a(a−1)2⋅(ha)2+⋯).(x+h)a=xa(1+hx)a=xa(1+a⋅hx+a(a−1)2⋅(ha)2+⋯).

Then subtracting xaxa and dividing by hh gives

(x+h)a−xah=xa(a⋅1x+a(a−1)2⋅ha2+⋯)(x+h)a−xah=xa(a⋅1x+a(a−1)2⋅ha2+⋯)
As h→0h→0, the right hand side tends to xa⋅ax=axa−1xa⋅ax=axa−1, as required.

It turns out that there is a subtle difficulty with this approach. It uses the general binomial theorem for (1+x)a(1+x)a, but how do we prove that? The usual way to do so is to use Taylor’s theorem, but that requires us to already know the derivative of xaxa…. So this would be a circular argument! But now that we can differentiate xaxa without reference to the binomial theorem, we can use this to prove the general binomial theorem, and all works nicely.
