Chain Rule & Integration by Substitution

Can we use integration by substitution on these integrals?

  1. By using the substitution u=1−x2u=1−x2, or otherwise, evaluate
    ∫120x(1−x2)2dx.∫012x(1−x2)2dx.
If we take u=1−x2u=1−x2, then
dudx=−2x⟹du=−2xdxdudx=−2x⟹du=−2xdx
 and therefore, by substitution,
∫120x(1−x2)2dx=∫1−(12)21−0du−2u2=12∫134duu2=12[u−1−1]134=−12(1−43)=16.∫012x(1−x2)2dx=∫1−01−(12)2du−2u2=12∫341duu2=12[u−1−1]341=−12(1−43)=16.

Note how we can make an integral look simpler by factoring out constants, including negatives which we can take care of by swapping the limits over.
