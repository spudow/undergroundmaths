Review question

Can we find where the normal is parallel to y=xy=x?

Solution

A curve has parametric equations x=5asecθ,y=3atanθx=5asec⁡θ,y=3atan⁡θ, where −12π<θ<12π−12π<θ<12π and aa is a positive constant. Find the coordinates of the point on the curve at which the normal is parallel to the line y=xy=x.

Differentiating with respect to θθ, we find that
dxdθdydθ=5asecθtanθ=3asec2θ.dxdθ=5asec⁡θtan⁡θdydθ=3asec2⁡θ.

Thus

dydx=3asec2θ5asecθtanθ=35sinθ.dydx=3asec2⁡θ5asec⁡θtan⁡θ=35sin⁡θ.

The normal to the curve therefore has gradient

−53sinθ.−53sin⁡θ.

The normal is parallel to y=xy=x when

−53sinθ=1⟹sinθ=−35.−53sin⁡θ=1⟹sin⁡θ=−35.

By sketching a graph of y=sinθy=sin⁡θ, we can see that there’s a unique value of θθ in the range −π2<θ<π2−π2<θ<π2 for which sinθ=−35sin⁡θ=−35.

The blue line in the diagram shows that this corresponds to a positive value of cosθcos⁡θ.

Hence, we get that

cos2θ=1−sin2θ=1−(−35)2=1625⟹cosθ=45cos2⁡θ=1−sin2⁡θ=1−(−35)2=1625⟹cos⁡θ=45
as cosθcos⁡θ is positive.

Alternatively, using the sketch above, we can see that we are dealing with a 3−4−53−4−5 triangle, and so the value of cosine we need is cosθ=45cos⁡θ=45.

Thus our coordinates for this value of θθ are
xy=5asecθ=5acosθ=5a×54=25a4,=3atanθ=3a×sinθcosθ=3a×(−35)×(54)=−9a4.x=5asec⁡θ=5acos⁡θ=5a×54=25a4,y=3atan⁡θ=3a×sin⁡θcos⁡θ=3a×(−35)×(54)=−9a4.
