Things you may have noticed

Unrestricted domains

In this exercise we are using domain to mean all the possible values of xx. In other contexts, we may artificially restrict the domain of a function. For instance we might only be interested in the function f(x)=x2f(x)=x2 in the domain 0≤x<20≤x<2.

Similar inequalities

With some of the functions you have to be careful to distinguish between, for instance, x>0x>0 and x≥0x≥0.

Multiple inequalities

One of the functions has a range that is between 00 and 11, including 11 but not including 00. This can be written neatly as 0<f(x)≤10<f(x)≤1.

Another has a range that includes all numbers >0>0 and all numbers ≤−1≤−1 but not numbers in between. This has to be written as two separate inequalities.

Turning points of x+1xx+1x

Finding the range of f(x)=x+1xf(x)=x+1x depends on knowing the value of the function at its turning points (maximum and minimum) and this is not given on the card. There is more than one way to find these values; here are a few.

Using calculus

We could differentiate the function and thus find when its gradient is zero.

Using an inequality

By substitution we can find that f(1)=2f(1)=2. Is there any positive xx value for which f(x)<2f(x)<2? If so,
x+1xx2+1<2<2x .x+1x<2x2+1<2x .

Rearranging and factorising leads to an inequality with no real solutions, so 22 must be the minimum value of f(x)f(x) for positive xx. The local maximum can be found by symmetry.

When rearranging the inequality, why does it matter that the question specified positive xx values?

By adding gradients

Notice that f(x)f(x) is the sum of two functions: xx and 1x1x. So the gradient of f(x)f(x) is the sum of the gradients of xx and 1x1x (it is possible to prove this, but we aren’t going to do so here!).

y=xy=x has gradient 11, so the turning points of f(x)f(x) will be where y=1xy=1x has gradient −1−1. Can you find the xx values where this happens?
