Solution

The function

a(x)=x+1a(x)=x+1
can be combined in a number of ways with each of the functions
b(x)=2x−1,b(x)=x2−5x+8,b(x)=x2−5x−6b(x)=2x−1,b(x)=x2−5x+8,b(x)=x2−5x−6
 to create a set of new functions. What functions are created by adding, subtracting and multiplying? Can you sketch them in the cells of the grid below?

You might prefer to choose from the graphs A to L which are provided as cards to print out.

Top row, b(x)=2x−1b(x)=2x−1

Here, a(x)a(x) and b(x)b(x) are both straight line functions. Adding and subtracting them will create other straight line functions and the correct ones can be found by checking their axis intercepts.

Multiplying two straight line functions will usually create a quadratic function.

In this case, the result is y=(x+1)(2x−1)y=(x+1)(2x−1) which is an upward facing parabola with xx-intercepts at −1−1 and +12+12. So it must be graph K.

Can you find an example of straight line functions that do not create a quadratic when multiplied?

Middle row, b(x)=x2−5x+8b(x)=x2−5x+8

Adding, we get y=(x+1)+(x2−5x+8)=x2−4x+9y=(x+1)+(x2−5x+8)=x2−4x+9. This is an upward facing parabola, so it could be either I, J or K. It has a yy-intercept of 99 so it must be graph I.

To check, we can see that it does not factorise, but we could write it in completed square form,

y=(x−2)2+5y=(x−2)2+5
which shows that its vertex is at (2,5)(2,5) and it does not cross the xx-axis.

Subtracting, we get y=−x2+6x−7y=−x2+6x−7. This is downward facing with yy-intercept of −7−7 so it could be graph F or G.

Completing the square, y=−(x−3)2+2y=−(x−3)2+2, so its vertex is at (3,2)(3,2) and it therefore crosses the xx-axis. It must be graph F.

When we multiply a straight line function by a quadratic we expect to get a cubic function. Here, y=(x+1)(x2−5x+8)y=(x+1)(x2−5x+8). We can see that this meets the xx-axis at x=−1x=−1 and that it has a yy-intercept of 88. All three of graphs D, H and L meet the xx-axis at some negative value of xx, but only H has a positive yy-intercept. So this must be graph H.

Note that multiplying out the two brackets here would not be particularly helpful.

To check our choice of graph H, we could note that the graph has no other xx-intercepts. What does this tell us about the quadratic factor, (x2−5x+8)(x2−5x+8)?

Bottom row, b(x)=x2−5x−6b(x)=x2−5x−6

Adding, we get a quadratic that factorises.
y=x2−4x−5=(x−5)(x+1)y=x2−4x−5=(x−5)(x+1)

It has xx-intercepts at −1−1 and 55 and yy-intercept at −5−5, so it must be graph J.

Subtracting, we get
y=−x2+6x+7=−(x−7)(x+1)y=−x2+6x+7=−(x−7)(x+1)

This is downward facing with xx-intercepts at −1−1 and 77 and yy-intercept at 77, so it must be graph E.

Multiplying, we get another cubic. This time we can factorise it further.
y=(x+1)(x2−5x−6)=(x+1)(x−6)(x+1)y=(x+1)(x2−5x−6)=(x+1)(x−6)(x+1)

It has a repeated root at x=−1x=−1 and another root at x=6x=6 so it must be graph D.
