Hence sketch the curves

(iii) y=|||x−ax|||,…(iii) y=|x−ax|,…

As we already have a sketch of the graph y=x−axy=x−ax, we need only consider the effect that taking the absolute value has on the graph.

Every point on the curve with a positive yy co-ordinate is fixed and every point with a negative yy co-ordinate is reflected in the xx-axis. This gives us the following sketch.

Hence sketch the curves

…,(iv) y2=x2−a2x2.…,(iv) y2=x2−a2x2.

Approach 1

We start by rewriting our equation as

y=±1−a2x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.(1)(1)y=±1−a2x2.

Firstly, we note that if −a<x<a−a<x<a, then 1−a2x2<01−a2x2<0 and so (1)(1) has no solutions. This means that our curve only has points with x≥ax≥a or x≤−ax≤−a. We will consider the sketch of y=1−a2x2y=1−a2x2, from (ii), in this range and aim to plot the square roots. This is, for each point (x,y)(x,y) on this curve, we want to include the points (x,±y⎯⎯√)(x,±y) in our sketch of (1)(1).

If y<1y<1, then the vertical distance decreases and, if y>1y>1, the vertical distance increases. Therefore, we get the following graph for the positive root y=+1−a2x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y=+1−a2x2.

To plot the other root y=−1−a2x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y=−1−a2x2, we can simply add the reflection in the xx-axis.

We now need only consider the points (±a,0)(±a,0) and the angles that the curves make at these points. Comparing to the graph of y=x⎯⎯√y=x, we would not expect the curve to be sharp at these points. Hence, we will include the information that the curve intersects the xx-axis at right-angles in our final sketch.

Similarly to before, the equations of the asymptotes are y=±1y=±1 and the points of intersection are at x=±ax=±a.

The full details as to why this argument works will be touched on at university-level, but there are a number of ways that we can convince ourselves of this result none-the-less. For example, would we expect these points to be sharp if we rewrote the curve in the form x=±a1−y2√x=±a1−y2? Alternatively, what would we expect if we used calculus to calculate the gradient of the curve at these points? What does this tell us?

Approach 2

We begin by rewriting our equation as

y2+(ax)2=1.(2)(2)y2+(ax)2=1.

As this looks similar to the equation for a circle, we will begin by considering the graph of the unit circle y2+x2=1y2+x2=1.

To get the graph of equation (2)(2), we want to swap the xx for axax. This is equivalent to finding another point x′x′ such that xx′=axx′=a (i.e. point x′x′ for which the geometric mean of xx and x′x′ are on the lines x=±a⎯⎯√x=±a). This is equivalent to inversion in the lines x=±a⎯⎯√x=±a and so each half of the circle can be projected outwards in the xx-direction, as shown.
