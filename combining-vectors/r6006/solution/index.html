Combining Vectors
Review question

Which point PP is on both this line and this plane?

Solution

The line ll has the equation

r=505+λ210,λ∈ℝ.r=(505)+λ(210),λ∈R.

  1. Show that ll lies in the plane whose equation is
    r.−120=−5.r.(−120)=−5.

We need to show that every vector rr that satisfies the line equation also satisfies the plane equation.

Substituting in, we find

505+λ210.−120=5+2λλ5.−120=−5−2λ+2λ=−5.[(505)+λ(210)].(−120)=(5+2λλ5).(−120)=−5−2λ+2λ=−5.

This is true for every λλ, so we’ve shown that the line does indeed lie in the given plane.

  1. Find the position vector of LL, the foot of the perpendicular from the origin OO to ll.

Let OL−→=505+λ210.OL→=(505)+λ(210). We need

505+λ210.210=0[(505)+λ(210)].(210)=0

which is true if and only if 10+4λ+λ=0,10+4λ+λ=0, or λ=−2.λ=−2. Thus LL is the point 1−25(1−25).

  1. Find an equation of the plane containing OO and ll.

We know that OL−→OL→ is in the plane, and putting λ=0λ=0, we know also that OM−→−OM→ is in the plane, where OM−→−OM→ = 505(505).

Any point in the plane can be expressed as a linear combination of these vectors. So an equation of the plane is

r=α505+β1−25.r=α(505)+β(1−25).

Alternatively, we could write the equation as r.n=0r.n=0 where nn is the normal to the plane. Substituting in the two points LL and MM tells us that nn is a multiple of 1−2−1(1−2−1), so the plane equation can be written as

r.1−2−1=0.r.(1−2−1)=0.

  1. Find the position vector of the point PP where ll meets the plane ππ whose equation is
    r.122=11.r.(122)=11.

Let OP−→−=505+λ210.OP→=(505)+λ(210). If PP is on the plane, then OP−→−.122=11,OP→.(122)=11, and so

505+λ210.122=11.[(505)+λ(210)].(122)=11.

This implies

15+4λ=11,15+4λ=11,

from which we conclude λ=−1λ=−1. Thus

OP−→−=505−210=3−15.OP→=(505)−(210)=(3−15).
