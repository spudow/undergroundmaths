Counting & Binomials
Review question

Can we use a binomial expansion to evaluate (1934)6(1934)6?

Solution

Expand (x−y)6(x−y)6 by the binomial theorem…

Using the binomial theorem, we have

(a+b)n=∑k=0n(nk)an−kbk.(a+b)n=∑k=0n(nk)an−kbk.
Substituting in the values n=6n=6, a=xa=x and b=−yb=−y, we get

(x−y)6=(60)x6(−y)0+(61)x5(−y)1+(62)x4(−y)2+(63)x3(−y)3+(64)x2(−y)4+(65)x1(−y)5+(66)x0(−y)6=x6(−y)0+61x5(−y)1+6⋅52⋅1x4(−y)2+6⋅5⋅43⋅2⋅1x3(−y)3+6⋅52⋅1x2(−y)4+61x1(−y)5+x0(−y)6=x6−6x5y+15x4y2−20x3y3+15x2y4−6xy5+y6.(x−y)6=(60)x6(−y)0+(61)x5(−y)1+(62)x4(−y)2+(63)x3(−y)3+(64)x2(−y)4+(65)x1(−y)5+(66)x0(−y)6=x6(−y)0+61x5(−y)1+6⋅52⋅1x4(−y)2+6⋅5⋅43⋅2⋅1x3(−y)3+6⋅52⋅1x2(−y)4+61x1(−y)5+x0(−y)6=x6−6x5y+15x4y2−20x3y3+15x2y4−6xy5+y6.

Alternatively we can use the fact that the 6th line of Pascal’s triangle is 11, 66, 1515, 2020, 1515, 66, 11 and just write the final result down, being careful with signs. (If you do this, it’s a good idea to state that you’ve used the 6th6th line of Pascal’s triangle.)

…and use the result to evaluate (1934)6(1934)6 correct to the nearest thousand.

Using x=20x=20 and y=14y=14, we can substitute into the binomial expansion to get
(1934)6=206−6×205×14+15×204×(14)2−20×203×(14)3+15×202×(14)4−6×20×(14)5+(14)6.(1934)6=206−6×205×14+15×204×(14)2−20×203×(14)3+15×202×(14)4−6×20×(14)5+(14)6.

We need this result to the nearest thousand, so we can save ourselves some work and ignore the very last term straight away.

We can then work backwards through the terms, calculating their rough size to see whether we can ignore them in the approximation.

It’s helpful to write numbers in their prime factorisation (to help with cancelling) or in powers of 1010 so we can keep tabs on how big the terms are.

  • (14)6=2−12(14)6=2−12 — ignore

  • 6×20×(14)5=3×10×2−8=302566×20×(14)5=3×10×2−8=30256 — ignore

  • 15×202×(14)4=3×5×102×2−6=15006415×202×(14)4=3×5×102×2−6=150064 — ignore

  • 204×(14)3=104×2−2=100004=2500204×(14)3=104×2−2=100004=2500 — keep this.

The last three terms must be included in the approximation.

  • 15×204×(14)2=3×5×104=15000015×204×(14)2=3×5×104=150000

  • 6×205×14=24×3×105=48000006×205×14=24×3×105=4800000

  • 206=26×106=64000000206=26×106=64000000.

So to the nearest thousand,
(1934)6≈64000000−4800000+150000−2500=59347500≈59348000.(1934)6≈64000000−4800000+150000−2500=59347500≈59348000.

The next term, 150064150064, is positive, so we definitely need to round up.

A computer tells us that the exact answer is

(1934)6=59347523.320556640625.(1934)6=59347523.320556640625.
It is interesting to note that including one more term would have given us the answer correct to the nearest whole number.
