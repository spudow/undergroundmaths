  1. Evaluate the following. Can you do this without calculating each factor separately?

    1. (log39)(log981)(log3⁡9)(log9⁡81)

    2. (log381)(log819)(log3⁡81)(log81⁡9)

Using the generalised version of the result in the warm-up,

  1. (log39)(log981)=log381=4(log3⁡9)(log9⁡81)=log3⁡81=4

  2. (log381)(log819)=log39=2(log3⁡81)(log81⁡9)=log3⁡9=2.

  1. Write the following as a single logarithm. What conditions must xx and yy satisfy?

    1. (log105)(log5x)(log10⁡5)(log5⁡x)

    2. (log10x)(logxy)(logy3)(log10⁡x)(logx⁡y)(logy⁡3)

Using the same idea as in question 1, (a) (log105)(log5x)=log10x(log10⁡5)(log5⁡x)=log10⁡x.

For part (b), one approach is

(log10x)(logxy)(logy3)=(log10y)(logy3)=log103(log10⁡x)(logx⁡y)(logy⁡3)=(log10⁡y)(logy⁡3)=log10⁡3

Since xx and yy both occur as bases of logarithms in these expressions, xx and yy must both be positive, but can’t be 11.

  1. Simplify the following. What conditions must xx and yy satisfy?

    1. (logxy)(logy10)(log8x)(logx⁡y)(logy⁡10)(log8⁡x)

    2. (logxy)(log8x)(logy8)(logx⁡y)(log8⁡x)(logy⁡8)

  1. (logxy)(logy10)(log8x)=(logx10)(log8x)=(log8x)(logx10)=log810(logx⁡y)(logy⁡10)(log8⁡x)=(logx⁡10)(log8⁡x)=(log8⁡x)(logx⁡10)=log8⁡10.

  2. (logxy)(log8x)(logy8)=(log8x)(logxy)(logy8)=(log8y)(logy8)=log88=1(logx⁡y)(log8⁡x)(logy⁡8)=(log8⁡x)(logx⁡y)(logy⁡8)=(log8⁡y)(logy⁡8)=log8⁡8=1.

Again, since xx and yy both occur as bases of logarithms in these expressions, xx and yy must both be positive, but can’t be 11.

  1. The value of log23log2⁡3 is approximately 1.61.6. Can you use this to approximate log427log4⁡27? What about log827?log8⁡27?

If we start by using the law for manipulating logs of powers, we get log427=3log43log4⁡27=3log4⁡3.

From the definition of logarithms, we know that log43log4⁡3 is smaller than log23log2⁡3, so we can expect the answer to be less than 4.84.8. In fact, we can say more than this, because log43<1<log23log4⁡3<1<log2⁡3, so we should expect the answer to be less than 33.

From the warm-up we know that log43=12log23log4⁡3=12log2⁡3. Therefore log427=3(log42)(log23)≈3×12×1.6=2.4log4⁡27=3(log4⁡2)(log2⁡3)≈3×12×1.6=2.4.

We know that log827log8⁡27 will be smaller than log427log4⁡27. By manipulating powers and changing the base, we get log827=3(log82)(log23)=log23≈1.6log8⁡27=3(log8⁡2)(log2⁡3)=log2⁡3≈1.6.

Note that from the second part we have log2333=log23log23⁡33=log2⁡3. In general, logakbk=logablogak⁡bk=loga⁡b for any positive aa, bb with a≠1a≠1 and any power kk. How could you prove this?

  1. What is (logab3)(logba3)(loga⁡b3)(logb⁡a3)?

Using laws for manipulating logs of powers,

(logab3)(logba3)=9(logab)(logba)=9logaa=9.(loga⁡b3)(logb⁡a3)=9(loga⁡b)(logb⁡a)=9loga⁡a=9.

  1. What is the relationship between logabloga⁡b and logbalogb⁡a?

In questions 3 and 7 we had something of the form (logab)(logba)=logaa(loga⁡b)(logb⁡a)=loga⁡a, so (logab)(logba)=1(loga⁡b)(logb⁡a)=1, or equivalently logab=1logbaloga⁡b=1logb⁡a. Is there any danger of dividing by 00 here?

  1. Solve logx7=log7xlogx⁡7=log7⁡x.

Using the relationship in question 8, solving logx7=log7xlogx⁡7=log7⁡x is equivalent to solving (log7x)2=1(log7⁡x)2=1. Therefore we need to solve log7x=1log7⁡x=1 and log7x=−1log7⁡x=−1, which give us x=7x=7 and x=17x=17 as solutions.
