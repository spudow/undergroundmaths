Graph solutions

This is the graph of y=log5xy=log5⁡x

Use this graph to help you to draw graphs of

  1. log25xlog25⁡x
  2. log5√xlog5⁡x
  3. log15xlog15⁡x
  4. log10xlog10⁡x
  5. logx5logx⁡5

Try to make your graphs as accurate as you can, but use the graph of y=log5xy=log5⁡x to help you, rather than a calculator.

Here are graphs 1 and 2 drawn using values from the y=log5xy=log5⁡x. The relationships between each graph and the graph of y=log5xy=log5⁡x are described below.

  1. log25x=(log255)(log5x)=12log5xlog25⁡x=(log25⁡5)(log5⁡x)=12log5⁡x, so the graph of y=log25xy=log25⁡x is a stretch of the graph y=log5xy=log5⁡x parallel to the yy-axis and scale factor 1212.

  2. log5√x=(log5√5)(log5x)=2log5xlog5⁡x=(log5⁡5)(log5⁡x)=2log5⁡x, so the graph of y=log5√xy=log5⁡x is a stretch of the graph y=log5xy=log5⁡x parallel to the yy-axis and scale factor 22.

Thinking about the definition of logarithms, as we did in the warm-up, these results make sense. For example, we’d expect the power that we’d raise 55 to in order to get a given value of xx to be twice as much as the power we’d need to raise 2525 to.

  1. As 15=5−115=5−1, the power that we’d raise 1515 to in order to get a given value of xx is −1−1 times the power we’d need to raise 55 to. Alternatively, using the idea from the warm-up,
    log15x=(log155)(log5x)=−log5xlog15⁡x=(log15⁡5)(log5⁡x)=−log5⁡x
     and so the graph of y=log15xy=log15⁡x is a reflection in the xx-axis of the graph of y=log5xy=log5⁡x.
  1. More work is needed for the graph of y=log10xy=log10⁡x. We know that log10x=log5xlog510log10⁡x=log5⁡xlog5⁡10, so we need to divide by log510log5⁡10 to get the graph of y=log10xy=log10⁡x. The given graph doesn’t go up as far as x=10x=10, but log510=log55+log52=1+log52log5⁡10=log5⁡5+log5⁡2=1+log5⁡2. From the graph, log52≈0.42log5⁡2≈0.42. Now, 1÷1.42=100142=5071≈4970=0.71÷1.42=100142=5071≈4970=0.7 so the graph of y=log10xy=log10⁡x is a stretch of y=log5xy=log5⁡x parallel to the yy-axis, scale factor 0.70.7.
  1. For x>0x>0 with x≠1x≠1, (log5x)(logx5)=log55=1log5⁡x)(logx⁡5)=log5⁡5=1, so logx5logx⁡5 is the reciprocal of y=log5xy=log5⁡x. We can use this to plot particular points to get an idea of the shape of the graph y=logx5y=logx⁡5.

Alternatively, we can use the behaviour of the function log5xlog5⁡x to sketch the behaviour. For x>1x>1, as xx increases, log5xlog5⁡x increases and is positive, so logx5logx⁡5 decreases and is positive. For xx close to 11, log5xlog5⁡x is close to 00, and the graph of y=logx5y=logx⁡5 has an asymptote at x=1x=1. For 0<x<10<x<1, log5xlog5⁡x is negative. It increases to 00 as xx increases to 11, and tends to −∞−∞ as xx decreases to 00. The graph of y=logx5y=logx⁡5 gets steeper and closer to the yy-axis as xx decreases to 00, but it doesn’t actually meet the yy-axis because logx5logx⁡5 isn’t defined for x=0x=0.
