Some things to think about

  • Try to make your extra logarithms as general as possible. For instance if you want loga3loga⁡3 to be less than log53log5⁡3 what conditions must aa satisfy?
  • How could you include fractions or irrational numbers in your logarithms?

In order to fill in the box at the left of the grid, we require a logarithm that is smaller than log53log5⁡3. Two possible solutions are log63log6⁡3 and log52log5⁡2. Both of these are less than log53log5⁡3.

In order to give a more general logarithm for this cell, let’s think about when log5a<log53log5⁡a<log5⁡3. We know aa cannot be zero or negative because then it would not be possible to calculate its logarithm. In addition to this, log5alog5⁡a will only be smaller than log53log5⁡3 when a<3a<3. So we need 0<a<30<a<3.

Notice that aa does not have to be a whole number—it could be a fraction or an irrational number. Note too that the value of the logarithm can be negative.

Alternatively, still thinking about the same cell, we could try a logarithm of the form logb3logb⁡3. This would be smaller than log53log5⁡3 if b>5b>5. Is it possible for the base of the logarithm, bb, to be a fraction or an irrational number? If so, are there any other values of bb that would make logb3logb⁡3 smaller than log53log5⁡3?

Now suppose we wrote logb4logb⁡4 in the box at the bottom of the grid. This must be greater than log34log3⁡4 so we need b<3b<3, but this time there are other constraints on bb because you cannot use 11, 00 or any negative number as the base of a logarithm. Also, if the base, bb is between 00 and 11, then logb4logb⁡4 will be negative, so the correct constraint would be 1<b<31<b<3.

To understand fractional bases of logarithms, it might help to think about log124log12⁡4.

(12)−2=(2)2=4 so log124=−2(12)−2=(2)2=4 so log12⁡4=−2

Can you sketch the graph of y=logx4y=logx⁡4? Try using graphing software such as Desmos (find logaloga on its functions menu). You might also find the resource Changing bases helpful for thinking about the behaviour of logarithmic functions with different bases.
