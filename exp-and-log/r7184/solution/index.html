Exponentials & Logarithms
Review question

Which of these log and trig expressions is the largest?

Solution

Which of the following numbers is largest in value? (All angles are given in radians.)

  1. tan(5π4)tan⁡(5π4),

  2. sin2(5π4)sin2⁡(5π4),

  3. log10(5π4)log10⁡(5π4),

  4. log2(5π4)log2⁡(5π4).

Let’s consider the numbers in order.

a: Using the fact that tantan has period ππ, we have

tan(5π4)=tan(π+π4)=tan(π4)=1.tan⁡(5π4)=tan⁡(π+π4)=tan⁡(π4)=1.

b: Since sin(π−x)=sin(x)sin⁡(π−x)=sin⁡(x), we have
sin(5π4)⟹sin2(5π4)=sin(π−5π4)=sin(−π4)=−sin(π4)=−12⎯⎯√=12.sin⁡(5π4)=sin⁡(π−5π4)=sin⁡(−π4)=−sin⁡(π4)=−12⟹sin2⁡(5π4)=12.

c: Now, since π<4π<4, we know that 5π4<55π4<5 and so clearly 5π4<105π4<10. Therefore,

log10(5π4)<1.log10⁡(5π4)<1.

d: Similarly, π>3⟹5π4>154>2π>3⟹5π4>154>2. Hence,

log2(5π4)>1.log2⁡(5π4)>1.
So this expression has the largest value of the four.

The answer is (d).
