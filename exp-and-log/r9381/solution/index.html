Review question

Can we find bounds for log23log2⁡3?

Solution

Observe that 23=823=8, 25=3225=32, 32=932=9 and 33=2733=27. From these facts, we can deduce that log23log2⁡3, the logarithm of 33 to base 22, is

  1. between 113113 and 112112;

  2. between 112112 and 123123;

  3. between 123123 and 22;

  4. between 22 and 33.

We are told that

23<32<33<25.23<32<33<25.

As log2xlog2⁡x is an increasing function (which means that log2x<log2ylog2⁡x<log2⁡y if and only if x<yx<y, or the gradient of the graph y=log2xy=log2⁡x is always positive), we can take logarithms to base 22 of these inequalities. This gives

3<2log23<3log23<5.3<2log2⁡3<3log2⁡3<5.

From the first inequality we get

32<log23,32<log2⁡3,
and from the last inequality we get
log23<53.log2⁡3<53.

Combining these gives

32<log23<53,32<log2⁡3<53,
so the answer is (b).

An alternative approach might say, let log23=klog2⁡3=k, so 3=2k3=2k. Now 9=22k>89=22k>8, so 2k>32k>3.

We also have 27=23k<32=2527=23k<32=25, so 3k<53k<5.

We use here that y=2xy=2x is an increasing function.
