Solution

Some equations involving powers or indices can be solved using logarithms… but not all.

Think about how you could go about solving the following equations. Sort them according to the tools or methods you would use.

(A) 3x=813x=81

We know that
81=9×9=3481=9×9=34
 so we can say that x=4x=4 using only our knowledge of powers of 33. Note that we could instead use logarithms to solve this equation,
log33xx=log381=log381log3⁡3x=log3⁡81x=log3⁡81

so we could classify this equation as “either indices or logarithms”.

(B) x5=50x5=50

To solve this equation, remember that the inverse of raising to a power is a root or a fractional power. So we can solve it using indices,

x=5015=50⎯⎯⎯⎯√5 .x=5015=505 .

(C) 3x=433x=43

This looks very similar to the first equation, but since 4343 is not an exact power of 33 we have to use logarithms if we want an exact solution.

Have a go at estimating the solution.

(D) 52x−5x−6=052x−5x−6=0

The laws of indices tell us that 52x=(5x)252x=(5x)2 so this is a quadratic equation which we can factorise
(5x)2−(5x)−6(5x−3)(5x+2)5x=0=0=3 or −2 .(5x)2−(5x)−6=0(5x−3)(5x+2)=05x=3 or −2 .

Now, 5x5x cannot be negative so we have only one solution, for which we shall need logarithms.

x=log53x=log5⁡3
So we could classify this equation as requiring rearrangement and logarithms.

(E) 5x+4x=85x+4x=8

We don’t know a way to solve this exactly. How could we find an approximate solution?
What’s wrong with this?
log(5x+4x)log5x+log4xxlog5+xlog4x=log8=log8=log8=log8log5+log4log⁡(5x+4x)=log⁡8log⁡5x+log⁡4x=log⁡8xlog⁡5+xlog⁡4=log⁡8x=log⁡8log⁡5+log⁡4

(F) 5x+2×51−x=75x+2×51−x=7

It may not be immediately obvious what kind of equation this is, so we want to manipulate it into a more useful form. What can we do to the equation so there is no −x−x in the index of the second term?

We can split the second term,
51−x=5×5−x51−x=5×5−x
 and then multiply the equation through by 5x5x,
(5x)2+2×5×5−x×5x(5x)2+10=7×5x=7×5x(5x)2+2×5×5−x×5x=7×5x(5x)2+10=7×5x
 which is a quadratic in 5x5x that we can factorise,
(5x)2−7(5x)+10(5x−2)(5x−5)5x=0=0=2 or 5 .(5x)2−7(5x)+10=0(5x−2)(5x−5)=05x=2 or 5 .

One of the solutions can now be written down using indices while the other requires logarithms. So we can classify this equation as requiring rearrangement, indices and logarithms.

(G) 32x−3=2432x−3=24

This could be treated as a quadratic in 3x3x, but a bit of rearrangement reveals a simpler method to solve it.
32x2x=27=332x=272x=3

It’s always worth looking for simpler methods before embarking on complicated ones. In this case, gathering the constant terms together was the inviting first step.

(H) 22x−9×2x+8=022x−9×2x+8=0

This is another quadratic, this time in 2x2x.

22x=(2x)222x=(2x)2
so
(2x−8)(2x−1)=0 .(2x−8)(2x−1)=0 .

This can be solved using only rearrangement and indices.

(I) 2x−3⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=52x−3=5

Logarithms are not going to help, but squaring both sides will.

(J) 5x−x5=35x−x5=3

This is another example of an equation we don’t know how to solve exactly. We could use a numerical method such as trial and improvement to find an approximate solution.

How many solutions are there? Most numerical methods work best if you know roughly where the solution is. A sketch of y=5xy=5x and y=x5y=x5 might help.

(K) 163x=8163x=8

What’s gone wrong here?
163x23x232x23x=8=1=1=2x=3163x=823x=1232x=123=2xx=3
Take the logarithm of both sides and rearrange…
3xlog16x=log8=3log16log8 .3xlog⁡16=log⁡8x=3log⁡16log⁡8 .
Cube root of both sides…
161x=8⎯⎯√3=2161x=83=2
 then raise both sides to the power xx,
161x=2x=4 .161=2xx=4 .

Have the two methods given the same answer? Which method do you prefer?

(L) (1316)3x=34(1316)3x=34

We are going to have to use logarithms on this one.

Why would indices not give us a solution?

There is a fundamental difference between equations involving the unknown, xx, raised to a fixed power (such as x5x5) and equations involving a constant raised to an unknown power (such as 5x5x). How is this difference reflected in the methods required to solve the equations?

Note that we have been a bit imprecise about the meanings of the overlapping regions and about what we mean by rearrangement, so your solution may look quite different.
