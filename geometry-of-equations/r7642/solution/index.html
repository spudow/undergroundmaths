Geometry of Equations
Review question

Can we maximise the profit for this tractor-making business?

Ref: R7642
Printable/supporting materials Fullscreen mode Teacher notes

Solution

An engineering firm makes two types of tractor, the ‘Rome’ and the ‘Brussels’. The number of each made per week is limited by storage capacity on site, the number of workers employed and manufacturing costs, as detailed below:

The maximum total storage capacity for a week’s production is for 1010 tractors. However, there is not enough room to store more than 88 Brussels tractors from any week’s production and, in order to meet demand, at least 33 Rome tractors must be made each week.

In any given week, 55 workers are required to make one Rome tractor and 22 are needed to make one Brussels tractor. There are 4040 workers altogether, but not all of them have to be used. It costs £16000£16000 to make a Rome tractor and £12000£12000 to make a Brussels tractor. A total of £144000£144000 is available each week.

  1. Using the symbols RR for the number of Rome and BB for the number of Brussels tractors made per week, write down 55 inequalities arising from the information above.

Let’s go through the description line by line.

The maximum total storage capacity for a week’s production is for 1010 tractors.

So we can’t produce more than 1010 tractors per week. This means that

R+B≤10.R+B≤10.

However, there is not enough room to store more than 88 Brussels tractors from any week’s production and, in order to meet demand, at least 33 Rome tractors must be made each week.

So we can’t make more than 88 Brussels tractors, i.e.,

B≤8,B≤8,
and we must make more than 33 Rome tractors, i.e.,
R≥3.R≥3.

In any given week, 55 workers are required to make one Rome tractor and 22 are needed to make one Brussels tractor. There are 4040 workers altogether, but not all of them have to be used.

This tells us that the total number of workers per Rome tractor and the total number of workers per Brussels tractor must not exceed 4040. Mathematically, this means that

5R+2B≤40.5R+2B≤40.

It costs £16000£16000 to make a Rome tractor and £12000£12000 to make a Brussels tractor. A total of £144000£144000 is available each week.

So the money the Rome tractors cost and the Brussels tractors cost cannot exceed £144000£144000 per week. In other words,

16000R+12000B≤144000.16000R+12000B≤144000.

There are also two further inequalities implied by the situation: we cannot make a negative number of tractors, so we must have

R≥0andB≥0.R≥0andB≥0.
In this case, the first of these is already implied by R≥3R≥3, but the second is not. In general, it is worth being explicit about this sort of restriction.

  1. Illustrate these inequalities on a graph in which the horizontal axis represents the number of Rome tractors made per week and the vertical axis represents the number of Brussels tractors made per week. Show the feasible region clearly.

We should bear in mind that although we will be drawing continuous straight lines, there is an implicit assumption that the firm makes whole tractors: they are not making 2.7432.743 Brussels tractors, for example. So at the end, we will only be interested in those points with integer coordinates which lie within the feasible region.

Let’s first draw a set of suitable axes to work on. We should use graph paper to answer this question if possible.

There are two sensible ways to decide on suitable axes:

  • Work out where each of the lines representing the inequalities will meet the axes, and then draw axes large enough for these.

  • Note that as they can’t make more than 1010 tractors in total each week, it follows that R≤10R≤10 and B≤10B≤10. So drawing both axes from 00 to 1010 will certainly allow us to show the feasible region.

Our axes will have RR from 00 to 1010 and BB from 00 to 2020, so that all of the non-horizontal lines meet both axes on our graph, but this is not required.

Let’s start with the simplest inequalities: R≥3R≥3 and B≤8B≤8. First, we draw the lines R=3R=3 and B=8B=8 and shade the excluded regions.

Next, we’ll add in the inequality R+B≤10R+B≤10. We want our diagram to be accurate, so let’s check where the line R+B=10R+B=10 intersects the lines we already have. When R=3R=3, B=7B=7, and when B=8B=8, R=2R=2. So we expect R+B=10R+B=10 to cross below the intersection point of R=3R=3 and B=8B=8 like this:

The line 4R+3B=364R+3B=36 intersects RR-axis at R=9R=9 and the BB-axis at B=12B=12. (And it intersects the line B=10B=10 at R=1.5R=1.5.)

The feasible region is the region which remains unshaded (and lies on or above the RR-axis).

  1. The profit on one Rome tractor is £3000£3000 and the profit on one Brussels tractor is £2000£2000. Find the number of each type which should be made weekly in order to maximise the profit, and find the maximum profit.

We know that RR and BB must be whole numbers, so we can use our diagram to find all possible solutions, by working out what the profit is for each possible grid point and find the largest value. The profit is £1000(3R+2B)£1000(3R+2B), so we need only calculate 3R+2B3R+2B at each point. Clearly, for each possible value of RR, we only need to calculate 3R+2B3R+2B for the greatest possible value of BB.

So we check the points (R,B)=(3,7)(R,B)=(3,7), (4,6)(4,6), (5,5)(5,5), (6,4)(6,4), (7,2)(7,2) and (8,0)(8,0).

An alternative method for finding the maximum profit is to draw the line 3R+2B=c3R+2B=c on our diagram for some value of cc. Now using a ruler (or plotting the problem in Desmos or GeoGebra and using a slider), move the ruler while keeping its gradient constant so that the edge of the ruler represents the line 3R+2B=c3R+2B=c as cc increases. We want to find the largest possible cc at an integer point before we leave the feasible region.

For our problem, the maximum profit is earned at the point (6,4)(6,4), with the profit £3000×6+£2000×4=£26000£3000×6+£2000×4=£26000.
