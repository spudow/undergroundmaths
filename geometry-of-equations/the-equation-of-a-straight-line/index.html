The equation of a straight line
Add to your resource collection

Printable version Fullscreen mode

How can we find the equation of a straight line?

We already know how to do this: once we know the gradient mm and the yy-intercept cc, we can just write down

y=mx+cy=mx+c
and we are done.

But what if we don’t know the yy-intercept? In this piece, we’ll explore several scenarios and some related methods of approaching this question.

How can we find the equation from the gradient and a point on the line?

There are many times in mathematics where we know the gradient of a straight line and the coordinates of some point on the line, and we want to find its equation. When we get to Calculus of Powers, we will meet a very common example of this type of problem: how do we find the equation of the tangent to a curve?

In the meantime, though, we will just suppose that we know the gradient and a point on the line, and want to find the equation of this line.

We’ll start with a specific example: Find the equation of the line with gradient 33 passing through (1,2)(1,2).

A first approach

We know that the line has equation y=mx+cy=mx+c, and we know that m=3m=3, so the equation is y=3x+cy=3x+c.

Now when x=1x=1, we must have y=2y=2, as the point (1,2)(1,2) lies on the line. (Remember that the equation tells us the rule which every point on the line has to obey: “the yy-coordinate is 33 times the xx-coordinate plus cc”.)

So we can substitute in x=1x=1 and y=2y=2 to get
⟹⟹2=3×1+c2=3+cc=−1 .2=3×1+c⟹2=3+c⟹c=−1 .

So the equation is y=3x−1y=3x−1.

In fact, this method will always work no matter which co-ordinates and what gradient we are given!

We are about to look at some other methods which can be used instead and often appear to be slicker, but this one is solid and reliable.

A second approach: thinking about gradients

On this interactivity, one point is fixed at (1,2)(1,2). Move the second point (x,y)(x,y) so that it lies on the line with gradient 33 passing through (1,2)(1,2). As you do so, think about how you know that your point (x,y)(x,y) lies on this line.

Presumably, you ensured that the gradient between (x,y)(x,y) and (1,2)(1,2) remained equal to 33. Recall the formula for gradient: it is the change in yy-coordinate (y−2y−2) divided by the change in xx-coordinate (x−1x−1), so if we write this algebraically, we find that the point (x,y)(x,y) has to satisfy the condition
y−2x−1=3.(1)(1)y−2x−1=3.

And this gives us another way of writing our straight line!

To convert it into a more familiar form, we can multiply both sides by x−1x−1 to get:
y−2=3(x−1),(2)(2)y−2=3(x−1),

then expand the brackets to get

y−2=3x−3.y−2=3x−3.
Adding 22 to both sides finally gets us back to y=3x−1y=3x−1.

There is actually a reason why we prefer the form (2)(2) over the original (1)(1). Think about the point (1,2)(1,2) itself, which lies on the line. If we try substituting those values into (1)(1), we end up with 00=300=3, which has no mathematical meaning, as we are not allowed to divide by zero.

On the other hand, if we substitute x=1x=1 and y=2y=2 into (2)(2), we get 0=00=0, which is perfectly correct.

We can apply this method in general. If we have a line which has gradient mm and passes through the point (x1,y1)(x1,y1), then our general point (x,y)(x,y) must satisfy the equation
y−y1x−x1=my−y1x−x1=m
 (which is just (1)(1) rewritten with the numbers replaced by letters). Then multiplying by x−x1x−x1 as before (and remembering that we need brackets when we do this!) gives

y−y1=m(x−x1).y−y1=m(x−x1).

This is a very useful form to know, as it allows us to immediately write down an equation for a line when we know its gradient and a point on the line. We can then multiply out and rearrange to get the equation in the form y=mx+cy=mx+c if we wish to do so.

How can we find the equation from two points on the line?

We know how to find the gradient of the line between two points. Once we have that, we can then use one of the methods we’ve just been discussing to find the equation of the line.

For example, let’s find the equation of the straight line joining (1,3)(1,3) and (7,1)(7,1).

The gradient is

change in ychange in x=1−37−1=−26=−13change in ychange in x=1−37−1=−26=−13

So the equation of the line with gradient −13−13 passing through (1,3)(1,3) is

y−3=−13(x−1),y−3=−13(x−1),
which we can rearrange to get y=−13x+103y=−13x+103.

In general, if we have the two points (x1,y1)(x1,y1) and (x2,y2)(x2,y2), then the gradient of the line joining them is y2−y1x2−x1y2−y1x2−x1. If we use the equation of form (1)(1) that we had above, we obtain the very symmetrical form for the line:

y−y1x−x1=y2−y1x2−x1.y−y1x−x1=y2−y1x2−x1.
If we now convert this into a form not using fractions, by multiplying both sides by (x−x1)(x2−x1)(x−x1)(x2−x1), we get
(x2−x1)(y−y1)=(y2−y1)(x−x1),(x2−x1)(y−y1)=(y2−y1)(x−x1),
 which again has a pleasing symmetry to it.

Finally, if we expand the brackets and rearrange, we can write this in yet another way:

(y2−y1)x−(x2−x1)y=x1y2−x2y1.(y2−y1)x−(x2−x1)y=x1y2−x2y1.

It is not worth trying to memorise any of these general formulae; work them out as you need them!

A substitution method

An alternative method is to substitute the coordinates of the two points into the equation y=mx+cy=mx+c. In the example of the points (1,3)(1,3) and (7,1)(7,1), we get the simultaneous equations
31=m+c=7m+c3=m+c1=7m+c

Subtracting these equations gives 2=−6m2=−6m so m=−13m=−13. Substituting this back into one of the equations allows us to find that c=103c=103, as before.

Another way to write the equation: ax+by+c=0ax+by+c=0

Sometimes, the numbers which appear in the equation of a straight line written as y=mx+cy=mx+c can be somewhat awkward and involve fractions, for example our answer y=−13x+103y=−13x+103 in the last section. One thing that can make the equation look nicer in this case is to multiply through by an integer to make all of the numbers integers. In this case, we can multiply through by 33 to get 3y=−x+103y=−x+10, which now has no fractions.

Once we are rearranging equations like this, though, it might seem harder to compare two equations to decide if they represent the same line or parallel lines or so forth. For example, we could have rearranged this to get x=10−3yx=10−3y or multiplied by 66 instead to get 6y=−2x+206y=−2x+20, which both represent the same line, but look somewhat different.

One way of being a bit more consistent is to rearrange the equation to always put everything on the left hand side, getting

x+3y−10=0.x+3y−10=0.

One advantage of this way of writing straight lines is that it includes vertical lines such as x=4x=4, which can be written as x−4=0x−4=0 (or x+0y−4=0x+0y−4=0 to be explicit about the absence of yy). But it has the disadvantage that the same line can still be written in more than one way (for example, 2x+6y−20=02x+6y−20=0 in this case).

So it is often still easier to work with the y=mx+cy=mx+c form, especially if you are trying to decide if lines are parallel or perpendicular.

Challenge: Given two lines a1x+b1y+c1=0a1x+b1y+c1=0 and a2x+b2y+c2=0a2x+b2y+c2=0, how can you tell if they are parallel or perpendicular, without converting them back into the form y=mx+cy=mx+c?

  • They are parallel if the ratio a1:b1a1:b1 equals the ratio a2:b2a2:b2, which will be the case if and only if a1b2=a2b1a1b2=a2b1. If neither b1b1 nor b2b2 is zero, then they are parallel if a1/b1=a2/b2a1/b1=a2/b2. The first form is better, as it works even if one or both of b1b1 or b2b2 is zero.

  • They are perpendicular if the gradient −a1/b1−a1/b1 is the negative reciprocal of the gradient −a2/b2−a2/b2, which is the case if and only if a1a2+b1b2=0a1a2+b1b2=0. Again, this latter form works even if some of the coefficients are zero.

Alternative perspectives: Thinking about translations

We know that a straight line through the origin with gradient mm has equation y=mxy=mx. If we now wish to look at the straight line through (x1,y1)(x1,y1) with the same gradient, we can achieve this by either translating our line through the origin or by translating our coordinate system.

We’ll begin by looking at what happens if we translate our coordinate system, and come back to thinking about translating the line itself later.

In the following interactive graph, the original axes are shown in black and a new set of axes are in blue. The red point is shown with its coordinates according to both the original black axes (which we’ve called (x,y)(x,y)) and the new blue axes (called (X,Y)(X,Y)).

The red line is a straight line of gradient mm passing through the blue origin, so has equation Y=mXY=mX.

We discovered earlier that (X,Y)=(x−x1,y−y1)(X,Y)=(x−x1,y−y1), so if we substitute this into Y=mXY=mX, we get

y−y1=m(x−x1),y−y1=m(x−x1),
as we had before.

This idea of translating a coordinate system was taken much further by Einstein when he developed his Special Theory of Relativity. But that is a story for another day!

This form of the equation is also useful because it tells us that y−y1y−y1 is directly proportional to x−x1x−x1. Directly proportionality is such a nice thing itself that it is sometimes useful to write the equation of a line in this form.

Finally, we can think about translating the line instead of translating the coordinate system. When we translate an object by the vector (x1y1)(x1y1), the coordinates of all of its points have x1x1 added to the xx-coordinate and y1y1 added to the yy-coordinate.

If we consider the line with gradient mm passing through the origin, it has equation y=mxy=mx. To keep the same sort of notation as we used earlier, we will pick a point (X,Y)(X,Y) on this line, so that Y=mXY=mX.

If we now translate this line by (x1y1)(x1y1), so that the origin moves to (x1,y1)(x1,y1), the point (X,Y)(X,Y) will move to (X+x1,Y+y1)(X+x1,Y+y1), as shown in the following interactivity.

You can change the value of (x1y1)(x1y1) by moving the blue circle, and you can move the black point (X,Y)(X,Y) along the black line y=mxy=mx.

You can also change the gradient of the black line y=mxy=mx by moving the slider.

So if (x,y)(x,y) are the coordinates of a point on the translated line, we have

x=X+x1,y=Y+y1whereY=mX.x=X+x1,y=Y+y1whereY=mX.
Rearranging to get X=x−x1X=x−x1 and Y=y−y1Y=y−y1, the relationship Y=mXY=mX becomes
y−y1=m(x−x1),y−y1=m(x−x1),
 which is the same equation as before.
