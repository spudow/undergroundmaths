Inverse trigonometric function

Printable version

The inverse trigonometric functions are the inverse functions of the trigonometric functions and are either written as arcsinxarcsin⁡x, arccosxarccos⁡x and so on, or as sin−1xsin−1⁡x, cos−1xcos−1⁡x and so on.

As none of the trigonometric functions are one-to-one (injective), we cannot take their inverses without first restricting their domains. The domains that are normally chosen give rise to the following inverse trigonometric functions.

The range given in the above table is sometimes called the “principal value range” of the inverse trigonometric function.

So while an equation such as tanx=1tan⁡x=1 has infinitely many solutions (π4π4, 5π45π4, −3π4−3π4 and so on), when we write tan−11tan−1⁡1, we mean the single value which lies in the principal value range, which in this case is π4π4.
