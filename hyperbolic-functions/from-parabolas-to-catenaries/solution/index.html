Solution

Setting up the problem: Tension in a rope

This diagram shows a rope hanging. We have drawn axes so that the rope hangs in the (x,y)(x,y)-plane with the yy-axis vertically upwards and the lowest point of the rope on the yy-axis.

Here, T0T0 is the tension at the lowest point of the rope, while TT is the tension at a point PP which is at a distance of ss along the rope from the lowest point. At PP, the rope makes an angle of ψψ with the horizontal.

The rope has weight ww per unit length.

Prove that tanψ=(wT0)stan⁡ψ=(wT0)s.

Resolving vertically and horizontally on the portion of rope between the lowest point and PP, we find:
(↑)(→)Tsinψ−wsTcosψ−T0=0=0R(↑)Tsin⁡ψ−ws=0R(→)Tcos⁡ψ−T0=0

(The weight of the rope is its length ss times the weight per unit length ww.) Rearranging to get Tsinψ=wsTsin⁡ψ=ws and Tcosψ=T0Tcos⁡ψ=T0, we can divide to get tanψ=(wT0)stan⁡ψ=(wT0)s as required.

From now on, we’ll make our life easier by assuming that w/T0=1w/T0=1, so that the equation becomes tanψ=stan⁡ψ=s.

If the rope has the equation y=f(x)y=f(x), prove that f′(x)=sf′(x)=s.

The gradient of the curve at PP is f′(x)=tanψf′(x)=tan⁡ψ, so this follows immediately from tanψ=stan⁡ψ=s.

An iterative approach

We’ll start by assuming a very simple (wrong) formula for the rope length ss, use that to work out a first approximation for f(x)f(x), and improve it step-by-step.

We will also assume that the rope has a yy-intercept of 11.

Step 1

Assume that s=xs=x, that is, assume the weight is evenly distributed horizontally. (This is the case for ropes holding up bridges.)

What is f1(x)f1(x) in this case, where f′1(x)=sf1′(x)=s?

Using s=xs=x, we can integrate f′1(x)=xf1′(x)=x to get f1(x)=12x2+cf1(x)=12x2+c. Using f1(0)=1f1(0)=1, we get f1(x)=1+12x2f1(x)=1+12x2.

Step 2

Now assume that the weight is distributed uniformly along the shape given by the answer to Step 1, that is to say, ss is now the arc length of y=f1(x)y=f1(x).

What is f2(x)f2(x) approximately in this case, where f′2(x)=sf2′(x)=s? Work out an answer as a polynomial, valid for small values of xx, including one term more than in f1(x)f1(x).

The relevant version of the formula for arc length in this context is

∫1+(f′1(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=∫1+x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx.∫1+(f1′(x))2dx=∫1+x2dx.

We would like a polynomial approximation to this, so we expand the square root using the binomial theorem:
1+x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(1+x2)12=1+12x2+(12)(−12)2!(x2)2+⋯=1+12x2−18x4+⋯1+x2=(1+x2)12=1+12x2+(12)(−12)2!(x2)2+⋯=1+12x2−18x4+⋯

So integrating gives us a formula for ss:

s=x+16x3−140x5+⋯s=x+16x3−140x5+⋯
Finally, we can now solve the equation f′2(x)=sf2′(x)=s, where f2(0)=1f2(0)=1, to get
f2(x)=1+12x2+124x4−1240x6+⋯f2(x)=1+12x2+124x4−1240x6+⋯
 We only want one term more than f1(x)f1(x), though, so we get
f2(x)=1+12x2+124x4.f2(x)=1+12x2+124x4.

Step 3

Now assume that the weight is distributed uniformly along the shape given by the answer to Step 2, that is to say, ss is now the arc length of y=f2(x)y=f2(x).

What is f3(x)f3(x) approximately in this case, where f′3(x)=sf3′(x)=s? Work out an answer as a polynomial, valid for small values of xx, including one term more than in f2(x)f2(x).

We follow the same procedure as in step 2.

We start by finding ss, working up to the term in x4x4:
s=∫1+(f′2(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=∫1+(x+16x3)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=∫1+12(x+16x3)2+(12)(−12)2!(x+16x3)4+⋯=∫1+12(x2+26x4+136x6)−18(x4+⋯)+⋯=∫1+12x2+124x4+⋯=x+16x3+1120x5+⋯s=∫1+(f2′(x))2dx=∫1+(x+16x3)2dx=∫1+12(x+16x3)2+(12)(−12)2!(x+16x3)4+⋯=∫1+12(x2+26x4+136x6)−18(x4+⋯)+⋯=∫1+12x2+124x4+⋯=x+16x3+1120x5+⋯

and since f′3(x)=sf3′(x)=s, it follows that

f3(x)=1+12x2+124x4+1720x6=1+x22!+x44!+x66!,f3(x)=1+12x2+124x4+1720x6=1+x22!+x44!+x66!,
giving one term more than for f2(x)f2(x).

In the limit

  1. Can you predict f4(x)f4(x)? What about f5(x)f5(x)? And how will the sequence of polynomials f1(x)f1(x), f2(x)f2(x), f3(x)f3(x), … continue further?

    Following the pattern, it would seem likely that
    f4(x)f5(x)…fn(x)=1+x22!+x44!+x66!+x88!=1+x22!+x44!+x66!+x88!+x1010!=1+x22!+x44!+⋯+x2n(2n)!f4(x)=1+x22!+x44!+x66!+x88!f5(x)=1+x22!+x44!+x66!+x88!+x1010!…fn(x)=1+x22!+x44!+⋯+x2n(2n)!

  2. What is the power series, f(x)f(x), that this sequence is tending to? What do we call this function?

    It appears to be

    f(x)=1+x22!+x44!+⋯+x2n(2n)!+⋯f(x)=1+x22!+x44!+⋯+x2n(2n)!+⋯

    This function is coshxcosh⁡x, the hyperbolic cosine. It is defined either by this power series or by the identity

    coshx=ex+e−x2.cosh⁡x=ex+e−x2.

    Interestingly, this power series is valid for all values of xx, not only values less than 11.

  3. Can you prove that this limit function f(x)f(x) satisfies the equation f′(x)=sf′(x)=s, where ss is the arc length of y=f(x)y=f(x)?

    The limit function, y=f(x)=coshxy=f(x)=cosh⁡x, has arc length from 00 to xx given by
    s=∫x01+(f′(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=∫x01+(sinhx)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=∫x0coshxdx=[sinhx]x0=sinhxs=∫0x1+(f′(x))2dx=∫0x1+(sinh⁡x)2dx=∫0xcosh⁡xdx=[sinh⁡x]0x=sinh⁡x

    And f′(x)=sinhxf′(x)=sinh⁡x, so f′(x)=sf′(x)=s.

    We have been a bit sloppy with notation here, using xx as both one of the limits and also as the variable in the integral. If we were to be more careful, we would have changed the variable in the integral to a different letter such as tt.

  4. In the above procedure, what would Step nn have been? Can you prove that Step nn would have given the function fn(x)fn(x) which you predicted?

    Step nn would read:

    “Now assume that the weight is distributed uniformly along the shape given by the answer to Step n−1n−1, that is to say, ss is now the arc length of y=fn−1(x)y=fn−1(x).

    What is fn(x)fn(x) approximately in this case, where f′n(x)=sfn′(x)=s? Work out an answer as a polynomial, valid for small values of xx, including one term more than in fn−1(x)fn−1(x)."

    Since the construction is by iteration, the proof is by induction.

    We assume that the previous step (Step n−1n−1) gave us the expected function, that is, we assume that

    fn−1(x)=1+x22!+⋯+x2(n−1)(2(n−1))!.fn−1(x)=1+x22!+⋯+x2(n−1)(2(n−1))!.
    Then to find fn(x)fn(x), we begin by working out the arc length of y=fn−1(x)y=fn−1(x) from 00 to xx:
    s=∫x01+(f′n−1(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx.s=∫0x1+(fn−1′(x))2dx.
     But this integral is going to be very hard to calculate directly: we will need to differentiate fn−1(x)fn−1(x), square it, and then use the binomial theorem to expand the square root.

    However, we can use what we know about fn−1(x)fn−1(x) to sidestep many of these difficulties: fn−1(x)fn−1(x) is the expansion of coshxcosh⁡x up to the term in x2n−2x2n−2.

    Therefore, f′n−1(x)fn−1′(x) is the expansion of sinhxsinh⁡x up to the term in x2n−3x2n−3. (We can work this out directly by differentiating fn−1(x)fn−1(x) if we wish.) In fact, it even agrees up to the term in x2n−2x2n−2, which is zero.

    Therefore 1+(f′n−1(x))21+(fn−1′(x))2 will agree with the expansion of 1+sinh2x=cosh2x1+sinh2⁡x=cosh2⁡x up to at least the term in x2n−2x2n−2.

    When we then use the binomial theorem to expand 1+(f′n−1(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√1+(fn−1′(x))2 as a power series in xx, it will have to agree with coshxcosh⁡x up to at least the term in x2n−2x2n−2, since the expression inside the square root agrees up to at least this term.

    Thus when we integrate this to get f′n(x)=sfn′(x)=s, it will agree with ∫coshxdx=sinhx∫cosh⁡xdx=sinh⁡x up to at least the term in x2n−1x2n−1. (And the constant term in the integral is zero, as the integral is from 00 to xx, which agrees with the zero constant term in the expansion of sinhxsinh⁡x.)

    Then we integrate this final expression to get fn(x)fn(x), so this will agree with ∫sinhxdx=coshx∫sinh⁡xdx=cosh⁡x up to at least the term in x2nx2n, and the constant term is also correct as fn(0)=1fn(0)=1.

    Since we let fn(x)fn(x) be the polynomial with just one term more than fn−1(x)fn−1(x), we find that fn(x)fn(x) is the polynomial approximation to coshxcosh⁡x with terms up to x2nx2n, that is,

    fn(x)=1+x22!+x44!+⋯+x2n(2n)!fn(x)=1+x22!+x44!+⋯+x2n(2n)!
    as we wanted.

  1. Why do you think we assumed that the curve has a yy-intercept of 11?

    The limiting function is coshxcosh⁡x, and cosh0=1cosh⁡0=1. Had we chosen a different yy-intercept, say 00, the limiting function would have been coshx−1cosh⁡x−1, which would have been a little bit less pretty.

A direct approach

Can you solve the equation f′(x)=sf′(x)=s directly?

Using the formula for arc length, this differential equation becomes

f′(x)=∫x01+(f′(t))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dt.f′(x)=∫0x1+(f′(t))2dt.
(Here we’re being good and not using xx as the variable in the integration, to avoid confusing ourselves.) This equation may seem very hard at first, but we might notice that f(x)f(x) itself never appears, just its derivative.

So let’s write p(x)=f′(x)p(x)=f′(x), to get

p(x)=∫x01+(p(t))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dt,p(x)=∫0x1+(p(t))2dt,
which looks marginally simpler as there’s no differentiation involved.

We still have this pesky integral, but we can get rid of that by differentiating both sides with respect to xx, and making use of the fundamental theorem of calculus:

p′(x)=1+(p(x))2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.p′(x)=1+(p(x))2.
No integral left!

We could try squaring both sides to get rid of the square root, but then we’d have (p′(x))2(p′(x))2, a squared derivative, which is not nice.

If we write p′(x)p′(x) as dpdxdpdx things might look more familiar (and we’ll write pp in place of p(x)p(x), too):

dpdx=1+p2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.dpdx=1+p2.
This is a separable differential equation! So separating the variables, we get
∫11+p2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dp=∫1dx.∫11+p2dp=∫1dx.
 The right hand side is easy. For the left hand side, we use the substitution p=sinhup=sinh⁡u, so dpdu=coshudpdu=cosh⁡u, giving
∫11+sinh2u⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√coshudu=x,∫11+sinh2⁡ucosh⁡udu=x,
 which simplifies to
∫1du=x,∫1du=x,
 so u=x+cu=x+c, giving arsinhp=x+carsinh⁡p=x+c or p=sinh(x+c)p=sinh⁡(x+c).

Now we can also work out from our original differential equation that p(0)=0p(0)=0, so c=0c=0 and p=sinhxp=sinh⁡x.

But p=f′(x)p=f′(x), so f′(x)=sinhxf′(x)=sinh⁡x, giving f(x)=coshx+Cf(x)=cosh⁡x+C.

Finally, using f(0)=1f(0)=1 gives C=0C=0, so f(x)=coshxf(x)=cosh⁡x.

More generally, can you solve the equation f′(x)=saf′(x)=sa?

The argument is very similar to the above; one just has to take care with the extra aa term. The solution is f(x)=acosh(xa)f(x)=acosh⁡(xa).
