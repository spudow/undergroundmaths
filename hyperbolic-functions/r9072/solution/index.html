Hyperbolic Functions
Review question

Can we find this circle's integer and non-integer rational points?
Add to your resource collection

Ref: R9072
Printable/supporting materials Fullscreen mode Teacher notes

Solution

  1. The point with coordinates (a,b)(a,b), where aa and bb are rational numbers, is called:

  • an integer rational point if both aa and bb are integers;

  • a non-integer rational point if neither aa nor bb is an integer.

  1. Write down an integer rational point and a non-integer rational point on the circle x2+y2=1x2+y2=1.

An integer rational point? This is easy - (0,1)(0,1), for example.

A non-integer rational point? We can use Pythagoras’ Theorem, which tells us that a 3−4−53−4−5 triangle is right-angled.

Thus a 3/5−4/5−13/5−4/5−1 triangle is right-angled, and (35,45)(35,45) is a non-integer point on the circle.

  1. Write down an integer rational point on the circle x2+y2=2x2+y2=2. Simplify
    (cosθ+m⎯⎯⎯√sinθ)2+(sinθ−m⎯⎯⎯√cosθ)2(cos⁡θ+msin⁡θ)2+(sin⁡θ−mcos⁡θ)2
     and hence obtain a non-integer rational point on the circle x2+y2=2x2+y2=2.

Once again, finding an integer rational point on the circle is easy: (1,1)(1,1), for example.

Multiplying out, we find that
(cosθ+m⎯⎯⎯√sinθ)2+(sinθ−m⎯⎯⎯√cosθ)2=cos2θ+2m⎯⎯⎯√cosθsinθ+msin2θ+sin2θ−2m⎯⎯⎯√cosθsinθ+mcos2θ=(1+m)(cos2θ+sin2θ)=1+m.(cos⁡θ+msin⁡θ)2+(sin⁡θ−mcos⁡θ)2=cos2⁡θ+2mcos⁡θsin⁡θ+msin2⁡θ+sin2⁡θ−2mcos⁡θsin⁡θ+mcos2⁡θ=(1+m)(cos2⁡θ+sin2⁡θ)=1+m.

So let’s put m=1m=1, and the above becomes (cosθ+sinθ)2+(sinθ−cosθ)2=2(cos⁡θ+sin⁡θ)2+(sin⁡θ−cos⁡θ)2=2, which looks like the equation of our circle. So the point (x,y)=(cosθ+sinθ,sinθ−cosθ)(x,y)=(cos⁡θ+sin⁡θ,sin⁡θ−cos⁡θ) must lie on the given circle.

So if we choose θθ to be the angle in the same 3/5−4/5−13/5−4/5−1 right-angled triangle as before, we find
xy=cosθ+sinθ=45+35=75=sinθ−cosθ=35−45=−15.x=cos⁡θ+sin⁡θ=45+35=75y=sin⁡θ−cos⁡θ=35−45=−15.

Check: (−15)2+(75)2=2(−15)2+(75)2=2.

Thus a non-integer rational point on the circle x2+y2=2x2+y2=2 is (75,−15)(75,−15).

  1. The point with coordinates (p+2⎯⎯√q,r+2⎯⎯√s)(p+2q,r+2s), where pp, qq, rr and ss are rational numbers, is called:

  • an integer 22-rational point if all of pp, qq, rr and ss are integers;

  • a non-integer 22-rational point if none of pp, qq, rr and ss is an integer.

  1. Write down an integer 22-rational point, and obtain a non-integer 22-rational point, on the circle x2+y2=3x2+y2=3.

This time we let m=2m=2. Thus (cosθ+2⎯⎯√sinθ)2+(sinθ−2⎯⎯√cosθ)2(cos⁡θ+2sin⁡θ)2+(sin⁡θ−2cos⁡θ)2 is on the circle x2+y2=3x2+y2=3.

Setting θ=0θ=0 we find (1,−2⎯⎯√)(1,−2), an example of an integer 22-rational point.

Using cosθ=45,sinθ=35cos⁡θ=45,sin⁡θ=35 (as before) we find the non-integer 22-rational point

(45+352⎯⎯√,35−452⎯⎯√).(45+352,35−452).

Check: (45+352⎯⎯√)2+(35−452⎯⎯√)2=3.(45+352)2+(35−452)2=3.

  1. Obtain a non-integer 22-rational point on the circle x2+y2=11x2+y2=11.

Note that x2+y2=11=1+10x2+y2=11=1+10 and we want a 2⎯⎯√2 rather than a 10⎯⎯⎯⎯√10, so some scaling is required.

Consider
(ccosθ+dm⎯⎯⎯√sinθ)2+(csinθ−dm⎯⎯⎯√cosθ)2=c2(cos2θ+sin2θ)+d2m(cos2θ+sin2θ)=c2+d2m.(ccos⁡θ+dmsin⁡θ)2+(csin⁡θ−dmcos⁡θ)2=c2(cos2⁡θ+sin2⁡θ)+d2m(cos2⁡θ+sin2⁡θ)=c2+d2m.
 Setting m=2m=2 again to get the square root correct, we need (integer) cc and dd such that
11=c2+2d2=32+2×12,11=c2+2d2=32+2×12,
 so c=3c=3 and d=1d=1. Thus
pq=3cosθr=3sinθ=sinθs=−cosθp=3cos⁡θr=3sin⁡θq=sin⁡θs=−cos⁡θ

so taking the same values for cosθcos⁡θ and sinθsin⁡θ as before we find an example of a non-integer 2-rational point on the circle x2+y2=3x2+y2=3:

(125+352⎯⎯√,95−452⎯⎯√).(125+352,95−452).

  1. Obtain a non-integer 22-rational point on the hyperbola x2−y2=7x2−y2=7.

There are few different ways to approach this. Here are three. Note that in all three we have a certain amount of freedom in how we choose our numbers.

Approach 1—Simple algebra

We know that if (p+2⎯⎯√q,r+2s⎯⎯⎯⎯√)(p+2q,r+2s) lies on the hyperbola x2−y2=7x2−y2=7, then

p2+2q2−r2−2s2+22⎯⎯√(pq−rs)=7.p2+2q2−r2−2s2+22(pq−rs)=7.

Thus we could sensibly choose p2+2q2−r2−2s2=7,pq−rs=0.p2+2q2−r2−2s2=7,pq−rs=0.

The second equation here suggests putting p=ks,r=kqp=ks,r=kq, whereupon the first equation becomes (k2−2)(s2−q2)=7(k2−2)(s2−q2)=7.

Now k=3k=3 is the obvious choice, giving s2−q2=1s2−q2=1, which yields (s+q)(s−q)=1(s+q)(s−q)=1.

Putting s+q=2s+q=2 and s−q=12s−q=12 gives s=54,q=34s=54,q=34, and thus p=154,r=94p=154,r=94.

Thus (154+2⎯⎯√34,94+2⎯⎯√54)(154+234,94+254) is our non-integer 2-rational point, as required.

Approach 2—Trigonometry

Instead of the identity cos2θ+sin2θ=1cos2⁡θ+sin2⁡θ=1, we instead use sec2θ−tan2θ=1sec2⁡θ−tan2⁡θ=1. We therefore try
xy=esecθ+fm⎯⎯⎯√tanθ=etanθ+fm⎯⎯⎯√secθ.x=esec⁡θ+fmtan⁡θy=etan⁡θ+fmsec⁡θ.

Then

x2−y2=e2−f2m.x2−y2=e2−f2m.
As before, m=2m=2 so we can make 77 with the integers e=3e=3 and f=1f=1. Then
(x,y)=(3secθ+2⎯⎯√tanθ,3tanθ+2⎯⎯√secθ).(x,y)=(3sec⁡θ+2tan⁡θ,3tan⁡θ+2sec⁡θ).
 Using cosθ=45cos⁡θ=45 and sinθ=35sin⁡θ=35 again, we find secθ=54sec⁡θ=54 and tanθ=34tan⁡θ=34 and so we have
(x,y)=(154+342⎯⎯√,94+542⎯⎯√).(x,y)=(154+342,94+542).

Approach 3—Hyperbolic functions

We can also approach this problem using the hyperbolic functions, which makes sense as x2−y2=7x2−y2=7 is a hyperbola. Note the identity
cosh2θ−sinh2θ=1,cosh2⁡θ−sinh2⁡θ=1,
 so try
xy=ecoshθ+fm⎯⎯⎯√sinhθ=esinhθ+fm⎯⎯⎯√coshθ.x=ecosh⁡θ+fmsinh⁡θy=esinh⁡θ+fmcosh⁡θ.
 Then
x2−y2=e2−f2m.x2−y2=e2−f2m.
 As before, m=2m=2 so we can make 77 with the integers e=3e=3 and f=1f=1. Then
(x,y)=(3coshθ+2⎯⎯√sinhθ,3sinhθ+2⎯⎯√coshθ).(x,y)=(3cosh⁡θ+2sinh⁡θ,3sinh⁡θ+2cosh⁡θ).
 Then exploiting the definitions of coshcosh and sinhsinh we have
coshθ=eθ+e−θ2sinhθ=eθ−e−θ2.cosh⁡θ=eθ+e−θ2sinh⁡θ=eθ−e−θ2.
 We can take θ=log2θ=log⁡2 to find an example of a non-integer 22-rational point:
cosh(log2)sinh(log2)=elog2+e−log22=2+122=54=2−122=34cosh⁡(log⁡2)=elog⁡2+e−log⁡22=2+122=54sinh⁡(log⁡2)=2−122=34

and so we obtain

(x,y)=(154+342⎯⎯√,94+542⎯⎯√).(x,y)=(154+342,94+542).
