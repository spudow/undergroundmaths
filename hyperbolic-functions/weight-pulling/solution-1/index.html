Solution 1

I slowly pull a weight along a (rough) table using a piece of string, moving my hand in a straight line.

This applet shows the situation in an ideal setting: the weight is at PP, and the end of the string is at AA. Move AA to see what happens. (Note that if you move AA backwards again in this applet, the weight simply retraces its path, which is not what would happen in reality.)

You can use the slider (aa) to adjust the length of the string.

What is the shape of the path that the weight traverses?

From the applet or by thinking about the problem, how does the line APAP relate to the path?

How can you use this to solve the problem?

The line APAP appears to be a tangent to the path.

This is true, since the weight is always being pulled in the direction of the string. As it is moving slowly (due to friction), it will move in the direction it is being pulled.

If we draw in the angle of APAP, we see the following:

If we let the coordinates of PP be (x,y)(x,y), then we have

dydx=−tanψ.dydx=−tan⁡ψ.
But we also know that the length of APAP is aa, so we also have
asinψ=yasin⁡ψ=y
 from the right-angled triangle with hypotenuse APAP.

Since sinψ=y/asin⁡ψ=y/a, we have cosψ=1−sin2ψ⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=1−y2/a2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√cos⁡ψ=1−sin2⁡ψ=1−y2/a2. Thus we have, from dy/dx=−tanψdy/dx=−tan⁡ψ:
dydx=−y/a1−y2/a2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=−ya2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.dydx=−y/a1−y2/a2=−ya2−y2.
 This is a separable differential equation, so we get
∫a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√ydy=∫−1dx=−x+c.∫a2−y2ydy=∫−1dx=−x+c.
 We can attack the left integral by using the substitution y=asinθy=asin⁡θ, so dy/dθ=acosθdy/dθ=acos⁡θ. This gives
∫a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√ydy=∫a2−a2sin2θ⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√asinθacosθdθ=∫acosθasinθacosθdθ=a∫cos2θsinθdθ=a∫1−sin2θsinθdθ=a∫cosecθ−sinθdθ=a(−ln(cosecθ+cotθ)+cosθ)∫a2−y2ydy=∫a2−a2sin2⁡θasin⁡θacos⁡θdθ=∫acos⁡θasin⁡θacos⁡θdθ=a∫cos2⁡θsin⁡θdθ=a∫1−sin2⁡θsin⁡θdθ=a∫cosec⁡θ−sin⁡θdθ=a(−ln⁡(cosec⁡θ+cot⁡θ)+cos⁡θ)

(The integral of cosecθcosec⁡θ can be obtained by writing

cosecθ=cosecθ(cosecθ+cotθ)cosecθ+cotθcosec⁡θ=cosec⁡θ(cosec⁡θ+cot⁡θ)cosec⁡θ+cot⁡θ
and then noting that the numerator of this is minus the derivative of the denominator.)

As cosθ=1−sin2θ⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=1−(y/a)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√cos⁡θ=1−sin2⁡θ=1−(y/a)2, we can write the final integral in terms of yy:
∫a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√ydy=−aln(ay+1−(y/a)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y/a)+a1−(y/a)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=−aln(a+a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y)+a1−(y/a)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√∫a2−y2ydy=−aln⁡(ay+1−(y/a)2y/a)+a1−(y/a)2=−aln⁡(a+a2−y2y)+a1−(y/a)2

and so we deduce that

−aln(a+a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y)+a1−(ya)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=−x+c.−aln⁡(a+a2−y2y)+a1−(ya)2=−x+c.
At x=0x=0, y=ay=a, so −aln1+0=−0+c−aln⁡1+0=−0+c, giving c=0c=0.

So the equation of the path is

x=aln(a+a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y)−a2−y2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.x=aln⁡(a+a2−y2y)−a2−y2.

The path is called a tractrix.

You may well find that the answer you get using this approach is different from that obtained from Approach 2. Are they equivalent, or is one of the approaches flawed?
