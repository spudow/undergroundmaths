Introducing Calculus
Review question

Can we use the trapezium rule to estimate this integral?

Solution

When the trapezium rule is used to estimate the integral

∫102xdx∫012xdx

by dividing the interval 0≤x≤10≤x≤1 into NN subintervals the answer achieved is

  1. 12N(1+121/N+1)12N(1+121/N+1),

  2. 12N(1+221/N−1)12N(1+221/N−1),

  3. 1N(1−121/N−1)1N(1−121/N−1),

  4. 12N(521/N+1−1)12N(521/N+1−1).

The trapezium rule says this:

The area under the curve f(x)f(x) between the points x0x0 and xnxn, with this interval divided into nn smaller intervals of width h=(xn−x0)nh=(xn−x0)n is given by

∫xnx0f(x)dx≈12h[(y0+yn)+2(y1+y2+⋯+yn−1)],∫x0xnf(x)dx≈12h[(y0+yn)+2(y1+y2+⋯+yn−1)],

where yi=f(xi)yi=f(xi).

So dividing the interval 0≤x≤10≤x≤1 into NN subintervals for our problem gives h=1Nh=1N, and the trapezium rule gives

∫102xdx≈121N[(20+21)+2(21N+22N+...+2N−1N)]∫012xdx≈121N[(20+21)+2(21N+22N+...+2N−1N)]

We have what looks like a series inside the brackets. Do we know any ways of simplifying series?

21N+22N+...+2N−1N21N+22N+...+2N−1N

is a geometric series with a first term of 21N21N, a common ratio of 21N21N, and N−1N−1 terms so we can write it as,

21N(1−2N−1N)1−21N21N(1−2N−1N)1−21N

using the formula for the sum of a geometric progression.

The whole expression can now be written as

12N[3+2(21N(1−2N−1N)1−21N)].12N[3+2(21N(1−2N−1N)1−21N)].

There are many ways to rearrange this expression, but three answers have a 11 in them, so we can use that as a starting point.

12N[3+2(21N(1−2N−1N)1−21N)]=12N[1+2(1−21N1−21N)+2(21N(1−2N−1N)1−21N)]=12N[1+2(1−21N)+2(21N−2)1−2N−1N]=12N(1+221N−1)12N[3+2(21N(1−2N−1N)1−21N)]=12N[1+2(1−21N1−21N)+2(21N(1−2N−1N)1−21N)]=12N[1+2(1−21N)+2(21N−2)1−2N−1N]=12N(1+221N−1)

So the answer is (b).
