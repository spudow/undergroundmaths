Solution

Can you rationalise the denominator of the following fractions?

  1. 11+2⎯⎯√311+23

As suggested in the Suggestion section, we could try multiplying 1+2⎯⎯√31+23 by various things:

(1+2⎯⎯√3)(1−2⎯⎯√3)(1+2⎯⎯√3)(1−4⎯⎯√3)=1−4⎯⎯√3=1+2⎯⎯√3−4⎯⎯√3−2.(1+23)(1−23)=1−43(1+23)(1−43)=1+23−43−2.

Neither of these worked fully, but they did offer some good things: the first eliminated the 2⎯⎯√323 term, while the second had 2⎯⎯√3×4⎯⎯√3=223×43=2.

One thing we could do is to combine them and use both 2⎯⎯√323 and 4⎯⎯√343. As we don’t know how many of each to use, we will try multiplying by 1+a2⎯⎯√3+b4⎯⎯√31+a23+b43 and see where the algebra takes us. (If it doesn’t work, we could always then try replacing 11 by a variable to see whether that helps.)

We have
(1+2⎯⎯√3)(1+a2⎯⎯√3+b4⎯⎯√3)=1+a2⎯⎯√3+b4⎯⎯√3+2⎯⎯√3+a4⎯⎯√3+2b=(1+2b)+(1+a)2⎯⎯√3+(a+b)4⎯⎯√3.(1+23)(1+a23+b43)=1+a23+b43+23+a43+2b=(1+2b)+(1+a)23+(a+b)43.

So if we take a=−1a=−1 and b=1b=1, both of the cube root terms will disappear, and we will be left with 1+2b=31+2b=3.

So to rationalise our fraction, we multiply both the numerator and denominator by 1−2⎯⎯√3+4⎯⎯√31−23+43 to find that

11+2⎯⎯√3=1−2⎯⎯√3+4⎯⎯√32⎯⎯√33.11+23=1−23+43233.

When we rationalise a denominator with a square root such as 1+2⎯⎯√1+2, we multiply it by 1−2⎯⎯√1−2, since we have the algebraic identity

a2−b2=(a+b)(a−b),a2−b2=(a+b)(a−b),
the well-known difference of two squares identity.

There are similar identities for the sum and difference of two cubes:
a3+b3a3−b3=(a+b)(a2−ab+b2)=(a−b)(a2+ab+b2).a3+b3=(a+b)(a2−ab+b2)a3−b3=(a−b)(a2+ab+b2).

So in our case, we are starting with 1+2⎯⎯√31+23, so if we use the first identity, we find that

(1+2⎯⎯√3)(1−2⎯⎯√3+4⎯⎯√3)=13+(2⎯⎯√3)3=3,(1+23)(1−23+43)=13+(23)3=3,
giving
11+2⎯⎯√3=1−2⎯⎯√3+4⎯⎯√32⎯⎯√3311+23=1−23+43233
 as before.

You might wonder why there isn’t a similar identity for a2+b2a2+b2 to parallel that for a3+b3a3+b3.

If a3+b3=0a3+b3=0, then a3=−b3a3=−b3, so a=−ba=−b is a possibility. This means that a+ba+b is a factor of a3+b3a3+b3 (using the factor theorem).

However, if a2+b2=0a2+b2=0, then a2=−b2a2=−b2, which we cannot square root in the reals (because −1−1 does not have a real square root). If, however, we allow ourselves to use complex numbers, then we get a=±iba=±ib (where i2=−1i2=−1), so a±iba±ib are factors of a2+b2a2+b2. Thus we can write

a2+b2=(a+ib)(a−ib),a2+b2=(a+ib)(a−ib),
which turns out to be a very useful identity when working with complex numbers.

As discussed in the Suggestion section, we could find a polynomial satisfied by x=1+2⎯⎯√3x=1+23, and then rearrange it in order to find a number we could multiply xx by to obtain an integer.

We’ll look at the methods from the Irrational roots solution for finding a polynomial satisfied by 1+2⎯⎯√1+2, and see whether we can adapt them to our case.

Irrational roots approach 1

This looks like it will be pretty horrendous in our context, so we won’t try using it here!

Irrational roots approach 2

This involved rearranging x=1+2⎯⎯√x=1+2 to isolate the square root. We could do the same here.

If x=1+2⎯⎯√3x=1+23, then x−1=2⎯⎯√3x−1=23.

We can now cube this to get (x−1)3=2(x−1)3=2.

Expanding gives

x3−3x2+3x−1=2,x3−3x2+3x−1=2,
so
x3−3x2+3x=3.x3−3x2+3x=3.
 So x(x2−3x+3)=3x(x2−3x+3)=3, which is what we needed.

Thus if we multiply the denominator of our fraction, x=1+2⎯⎯√3x=1+23, by
x2−3x+3=(1+2⎯⎯√3)2−3(1+2⎯⎯√3)+3=1+22⎯⎯√3+4⎯⎯√3−3−32⎯⎯√3+3=1−2⎯⎯√3+4⎯⎯√3x2−3x+3=(1+23)2−3(1+23)+3=1+223+43−3−323+3=1−23+43

we will get 33, hence

11+2⎯⎯√3=1−2⎯⎯√3+4⎯⎯√32⎯⎯√33.11+23=1−23+43233.

Irrational roots approach 3

This involved translating a graph. We want a graph which has a root at x=1+2⎯⎯√3x=1+23. If we begin with the graph of y=x3−2y=x3−2, which has a root at 2⎯⎯√323, and then translate it 11 unit to the right, we end up with a graph which has a root at x=1+2⎯⎯√3x=1+23.

So the graph y=(x−1)3−2y=(x−1)3−2 has a root at 1+2⎯⎯√31+23, and our required polynomial is (x−1)3−2=x3−3x2+3x−3(x−1)3−2=x3−3x2+3x−3.

We can then continue as above to find that x(x2−3x+3)=3x(x2−3x+3)=3, and hence to rationalise the denominator of our fraction.

Irrational roots approach 4

This involved squaring x=1+2⎯⎯√x=1+2 and then eliminating 2⎯⎯√2. We could try the same here.

We have:
xx2x3=1+2⎯⎯√3=1+22⎯⎯√3+4⎯⎯√3=3+32⎯⎯√3+34⎯⎯√3x=1+23x2=1+223+43x3=3+323+343

We can combine the x3x3 and x2x2 equations to eliminate the 4⎯⎯√343 terms:

x3−3x2=−32⎯⎯√3.x3−3x2=−323.
Then we can use the xx equation to eliminate the 2⎯⎯√323 term from this:
x3−3x2+3x=3,x3−3x2+3x=3,
 and we have our required polynomial equation.

The rest continues as in “Irrational roots approach 2”.

  • Which of these approaches do you prefer?

  • Which of these approaches can be most easily adapted for the other problems or other situations?

For the remaining questions, we will only offer one approach to solving them, but as we have seen, there are several other possibilities.

  1. 12+2⎯⎯√312+23

This looks very similar to the first problem. We’ll use the algebraic identity for a3+b3a3+b3 to answer this one.

We have

23+(2⎯⎯√3)3=(2+2⎯⎯√3)(4−22⎯⎯√3+4⎯⎯√3)23+(23)3=(2+23)(4−223+43)
and the left hand side equals 1010, so
12+2⎯⎯√3=4−22⎯⎯√3+4⎯⎯√32⎯⎯√310.12+23=4−223+432310.

  1. 12⎯⎯√3+4⎯⎯√3123+43

In this case, we notice that the denominator has a factor of 2⎯⎯√323, so we can write

12⎯⎯√3+4⎯⎯√3=12⎯⎯√3(1+2⎯⎯√3).123+43=123(1+23).
We can rationalise the second factor using our answer to question 1, and we can multiply the numerator and denominator by 4⎯⎯√343 to rationalise the 2⎯⎯√323 term as in the warm-up problem, giving:
12⎯⎯√3(1+2⎯⎯√3)=4⎯⎯√32×1−2⎯⎯√3+4⎯⎯√33=4⎯⎯√3−2+22⎯⎯√32⎯⎯√36.123(1+23)=432×1−23+433=43−2+223236.

  1. 11−2⎯⎯√3+4⎯⎯√311−23+43

We saw in our answer to question 1 above that

(1+2⎯⎯√3)(1−2⎯⎯√3+4⎯⎯√3)=3.(1+23)(1−23+43)=3.
So just as 11+2⎯⎯√3=1−2⎯⎯√3+4⎯⎯√32⎯⎯√3311+23=1−23+43233, we have
11−2⎯⎯√3+4⎯⎯√3=1+2⎯⎯√32⎯⎯√33.11−23+43=1+23233.

Taking it further: What difference does a sign make?

  1. 11+2⎯⎯√3+4⎯⎯√311+23+43

In approach 2 (algebraic identities) to the first problem above, we saw two useful identities. One of them was

a3−b3=(a−b)(a2+ab+b2).a3−b3=(a−b)(a2+ab+b2).
If we take a=2⎯⎯√3a=23 and b=1b=1, we obtain
2−1=(2⎯⎯√3−1)(4⎯⎯√3+2⎯⎯√3+1),2−1=(23−1)(43+23+1),
 so that
11+2⎯⎯√3+4⎯⎯√3=2⎯⎯√3−1,11+23+43=23−1,
 and we don’t even have a fraction left!

  1. 11+2⎯⎯√3−4⎯⎯√311+23−43

In this case, there are no algebraic identities which are obviously useful, because the signs do not match the ones we know. So we’ll use one of the polynomial equation approaches for this instead. It is also not obviously a translation of a simple graph, so we will make use of the fourth polynomial equation approach.

We have (with a bit of algebra, and using the facts that (2⎯⎯√3)2=4⎯⎯√3(23)2=43, (4⎯⎯√3)2=22⎯⎯√3(43)2=223 and 2⎯⎯√3×4⎯⎯√3=223×43=2):
xx2x3=1+2⎯⎯√3−4⎯⎯√3=−3+42⎯⎯√3−4⎯⎯√3=−13+32⎯⎯√3+64⎯⎯√3.x=1+23−43x2=−3+423−43x3=−13+323+643.
 We can eliminate the 4⎯⎯√343 term from these equations to get expressions only involving 2⎯⎯√323, for example:
x3+6xx2−x=−7+92⎯⎯√3=−4+32⎯⎯√3.x3+6x=−7+923x2−x=−4+323.

Then we can eliminate 2⎯⎯√323 by subtracting 33 times the second of these two equations from the first to give:

(x3+6x)−3(x2−x)=5,(x3+6x)−3(x2−x)=5,
so that x3−3x2+9x=5x3−3x2+9x=5, giving x(x2−3x+9)=5x(x2−3x+9)=5.

Thus we wish to multiply the numerator and denominator by

x2−3x+9=(−3+42⎯⎯√3−4⎯⎯√3)−3(1+2⎯⎯√3−4⎯⎯√3)+9=3+2⎯⎯√3+24⎯⎯√3x2−3x+9=(−3+423−43)−3(1+23−43)+9=3+23+243
to get
11+2⎯⎯√3−4⎯⎯√3=3+2⎯⎯√3+24⎯⎯√32⎯⎯√35.11+23−43=3+23+243235.

The point of these problems is not so much to work out the answers to these particular questions, but rather to show that we can, in principle, work out the answers. Computer algebra systems which do these calculations use techniques such as these behind the scenes to do their work.

Can you find any fractions for which you cannot rationalise the denominator?

The approach we have taken has involved finding a polynomial with the denominator as a root. There are numbers which are not the root of any polynomial with integer coefficients, such as 1+π1+π (see Squaring the circle), so a fraction such as 11+π11+π cannot have its denominator rationalised.
