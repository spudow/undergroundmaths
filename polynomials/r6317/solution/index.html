Polynomials & Rational Functions
Review question

Can we sketch the graph of y=(4x−1)/(x+5)y=(4x−1)/(x+5)?

Solution

  1. Sketch the graph of
    y=4x−1x+5.y=4x−1x+5.

In sketching the graph, we’ll look at four different features of the function:

  1. its intercepts with the axes;
  2. its asymptotic behaviour;
  3. its signs in various regions; and
  4. its turning points, if it has any.

Intercepts

The yy-intercept is

y=y(0)=4×0−10+5=−15.y=y(0)=4×0−10+5=−15.

The xx-intercepts are the values of xx such that

0=4x−1x+5;0=4x−1x+5;
there’s one such value, x=14x=14.

Asymptotic behaviour

There’s a single vertical asymptote when the denominator is zero, the line x=−5x=−5.

If we rewrite the function as

4x−1x+5=4−21x+54x−1x+5=4−21x+5
it is easier to see that as x→∞x→∞, y→4y→4 and that as x→−∞x→−∞, y→4y→4 so there’s a single horizontal asymptote, y=4y=4.

Signs

The sign of yy can only change when xx moves across a vertical asymptote or an xx-intercept.

The xx-axis falls into three regions, x<−5x<−5, −5<x<14−5<x<14, and x>14x>14, and in each of these regions yy has a constant sign.

We can determine the sign of yy in each of these regions by looking at the signs of the numerator and denominator separately.

  • In the first region, 4x−1<04x−1<0 and x+5<0x+5<0, so yy is the quotient of two negative expressions, and is therefore positive.
  • In the second region, 4x−1<04x−1<0 and x+5>0x+5>0, and so y<0y<0.
  • In the third region, 4x−1>04x−1>0 and x+5>0x+5>0, and so y>0y>0.

Turning points

By considering the function in the form

4x−1x+5=4−21x+54x−1x+5=4−21x+5
we see that it is a transformation of y=1xy=1x. Therefore there are no turning points.

Hence, or otherwise, find the set of values of xx for which

−1<4x−1x+5<1.−1<4x−1x+5<1.

The sketch shows that we need the set of xx-values from aa to bb, where aa is the solution of −1=4x−1x+5−1=4x−1x+5, and bb is the solution to 4x−1x+5=14x−1x+5=1.

We have
−1=4x−1x+5⟹−x−5=4x−1⟹x=−45.−1=4x−1x+5⟹−x−5=4x−1⟹x=−45.
 and also
4x−1x+5=1⟹4x−1=x+5⟹x=2.4x−1x+5=1⟹4x−1=x+5⟹x=2.

Thus a=−45a=−45, and b=2b=2, and the set of values for xx that we need is {x∈ℝ:−45<x<2}{x∈R:−45<x<2}.

  1. Find the set of values of xx for which
    3x3+2x2+6≥19x.3x3+2x2+6≥19x.

Now

3x3+2x2+6≥19x⟺3x3+2x2−19x+6≥0,3x3+2x2+6≥19x⟺3x3+2x2−19x+6≥0,
so if we let p(x)=3x3+2x2−19x+6p(x)=3x3+2x2−19x+6, then we have to find the set of values of xx for which p(x)≥0p(x)≥0.

Can we factor the polynomial p(x)p(x)?

By the factor theorem, (x−a)(x−a) is a factor of p(x)p(x) if and only if p(a)=0p(a)=0.

Let’s try a few different values for xx: every cubic equation has at least one real root, and if this is an integer, we have a chance of finding it here.

  • p(0)=6p(0)=6;
  • p(1)=3+2−19+6=−8p(1)=3+2−19+6=−8;
  • p(2)=3×8+2×4−19×2+6=24+8−38+6=0p(2)=3×8+2×4−19×2+6=24+8−38+6=0.
So our first factor is (x−2)(x−2). There are thus real constants αα, ββ and γγ such that
3x3+2x2−19x+6=p(x)=(αx2+βx+γ)(x−2)=αx3−2αx2+βx2−2βx+γx−2γ=αx3+(β−2α)x2+(γ−2β)x−2γ.3x3+2x2−19x+6=p(x)=(αx2+βx+γ)(x−2)=αx3−2αx2+βx2−2βx+γx−2γ=αx3+(β−2α)x2+(γ−2β)x−2γ.

By equating coefficients, we have that α=3α=3, γ=−3γ=−3, and β−2α=2β−2α=2, i.e. β=8β=8. Thus,

p(x)=(x−2)(3x2+8x−3).p(x)=(x−2)(3x2+8x−3).
Factorising, we have
3x2+8x−3=(3x−1)(x+3).3x2+8x−3=(3x−1)(x+3).
 Therefore
p(x)=(x−2)(3x−1)(x+3).p(x)=(x−2)(3x−1)(x+3).

The sign of p(x)p(x) can only change as xx passes over a root of p(x)p(x); these roots are x=−3x=−3, x=13x=13, and x=2x=2.

This divides the xx-axis into four regions.

  • In the first region, with x<−3x<−3, all three factors are negative, so p(x)<0p(x)<0.
  • In the second region, with −3<x<13−3<x<13, two factors are negative and one is positive, so p(x)>0p(x)>0.
  • In the third, with 13<x<213<x<2, one factor is negative and two are positive, and so p(x)<0p(x)<0.
  • Finally, with the fourth, with x>2x>2, all three factors are positive and therefore p(x)>0p(x)>0.

We therefore have that

{x∈ℝ:p(x)≥0}={x∈ℝ:x≥2}∪{x∈ℝ:−3≤x≤13}.{x∈R:p(x)≥0}={x∈R:x≥2}∪{x∈R:−3≤x≤13}.
