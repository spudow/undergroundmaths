Review question

What if the remainder when we divide by x−kx−k is kk?

Ref: R8701

Solution

  1. Given that f(x)=x3+kx2−2x+1f(x)=x3+kx2−2x+1 and that when f(x)f(x) is divided by (x−k)(x−k) the remainder is kk, find the possible values of kk.

Using the remainder theorem, we know that the remainder when we divide by (x−k)(x−k) is the same as f(k)=2k3−2k+1f(k)=2k3−2k+1.

So we need to find solutions for 2k3−2k+1=k2k3−2k+1=k, that is 2k3−3k+1=02k3−3k+1=0.

It is easy to guess and check that k=1k=1 is a solution, so (k−1)(k−1) is a factor; taking out a factor of k−1k−1 gives

2k3−3k+1=(k−1)(2k2+2k−1).2k3−3k+1=(k−1)(2k2+2k−1).
So either k=1k=1 or 2k2+2k−1=02k2+2k−1=0. Solutions for the latter are, using the quadratic formula,
k=−2±22−4×2×(−1)⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√2×2=−1±3⎯⎯√2k=−2±22−4×2×(−1)2×2=−1±32

So the solutions are k=1k=1, k=−1±3⎯⎯√2k=−1±32.

  1. When the polynomial p(x)p(x) is divided by (x−1)(x−1) the remainder is 55 and when p(x)p(x) is divided by (x−2)(x−2) the remainder is 77. Given that p(x)p(x) may be written in the form
    (x−1)(x−2)q(x)+Ax+B,(x−1)(x−2)q(x)+Ax+B,
     where q(x)q(x) is a polynomial and AA and BB are numbers, find the remainder when p(x)p(x) is divided by (x−1)(x−2)(x−1)(x−2).

Using the remainder theorem again, the remainder when we divide p(x)p(x) by (x−1)(x−1) is the same as p(1)=A+Bp(1)=A+B, so A+B=5A+B=5.

Likewise, the remainder when we divide p(x)p(x) by (x−2)(x−2) is the same as p(2)=2A+Bp(2)=2A+B, so 2A+B=72A+B=7.

These simultaneous equations in AA and BB give as solutions A=2A=2 and B=3B=3.

The remainder when p(x)p(x) is divided by (x−1)(x−2)(x−1)(x−2) is Ax+BAx+B, since (x−1)(x−2)(x−1)(x−2) is a factor of (x−1)(x−2)q(x)(x−1)(x−2)q(x), and Ax+BAx+B has a smaller degree than (x−1)(x−2)(x−1)(x−2). So the remainder is 2x+32x+3.
