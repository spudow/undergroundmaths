Solution

Can you find the missing positive numbers aa to dd?

We will work through the integrals in order, but they can be tackled in a variety of ways.

  1. ∫a113xdx=ln2∫1a13xdx=ln⁡2
∫a113xdx=[13ln(3x)]a1=13(ln3a−ln3)=13ln(3a3)=lna13∫1a13xdx=[13ln⁡(3x)]1a=13(ln⁡3a−ln⁡3)=13ln⁡(3a3)=ln⁡a13

So lna13=ln2ln⁡a13=ln⁡2 which gives us a=8a=8.

We have treated this integral as ∫a11(3x)dx∫1a1(3x)dx, but what happens if we think of it as 13∫a11xdx13∫1a1xdx? Do both approaches give the same answer?

  1. ∫30axe2xdx=be6+2∫03axe2xdx=be6+2

What method will we use for integrating this? We have a product of two different functions so it looks like integration by parts will be useful.

∫uv′dx=uv−∫u′vdx∫uv′dx=uv−∫u′vdx
Which function will we integrate and which will we differentiate?

Since we know that a=8a=8, we have two functions, 8x8x and e2xe2x. We can differentiate both these functions, but 8x8x will become a constant so let’s take u=8xu=8x and v′=e2xv′=e2x. That gives us u′=8u′=8 and v=12e2xv=12e2x. Therefore the integral can be written as:

∫308xe2xdx=[4xe2x]30−∫304e2xdx=[4xe2x−2e2x]30=(12e6−2e6)−(0−2e0)=10e6+2.∫038xe2xdx=[4xe2x]03−∫034e2xdx=[4xe2x−2e2x]03=(12e6−2e6)−(0−2e0)=10e6+2.

Comparing the answers we have 10e6+2=be6+210e6+2=be6+2, so b=10b=10.

  1. ∫31cx25−x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=46⎯⎯√−4c∫13cx25−x2dx=46−4c

The function in the numerator is a multiple of the derivative of the function in the square root, so it seems reasonable to think that we should try integration by substitution.

Let u=25−x2u=25−x2, then du=−2x dxdu=−2x dx. Some manipulation is required to get the integral in the form we need.

∫31cx25−x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=−c2∫31−2x25−x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx∫13cx25−x2dx=−c2∫13−2x25−x2dx

Now we can make the substitution.
−c2∫31−2x25−x2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√dx=−c2∫u=16u=241u⎯⎯√du=−c2[2u12]u=16u=24=−c(16⎯⎯⎯⎯√−24⎯⎯⎯⎯√)=2c6⎯⎯√−4c−c2∫13−2x25−x2dx=−c2∫u=24u=161udu=−c2[2u12]u=24u=16=−c(16−24)=2c6−4c

If we compare this to the answer given we have 46⎯⎯√−4c=2c6⎯⎯√−4c46−4c=2c6−4c, so c=2c=2.

When using integration by substitution there is likely to be more than one possible substitution that could be used. What other substitutions could be used here?

  1. ∫baππ2x2sincxdx=πd−π2+ca∫π2baπx2sin⁡cxdx=πd−π2+ca

As in the second integral we have the product of two unrelated functions so it makes sense to try integration by parts. Which way round should the choices for uu and v′v′ be?

We start by substituting in the values of a,ba,b and cc.

∫54ππ2x2sin2xdx∫π254πx2sin⁡2xdx

If u=x2u=x2 and v′=sin2xv′=sin⁡2x then we have u′=2xu′=2x and v=−12cos2xv=−12cos⁡2x.

∫54ππ2x2sin2xdx=[−12x2cos2x]54ππ2−∫54ππ2−xcos2xdx∫π254πx2sin⁡2xdx=[−12x2cos⁡2x]π254π−∫π254π−xcos⁡2xdx

Using integration by parts hasn’t reduced this to an integral that we can now solve but it has reduced the polynomial term from x2x2 to xx.

While we can’t solve this directly, it has got us one step closer. It is now similar to the second integral and we can use integration by parts for a second time where u=xu=x and v′=cos2xv′=cos⁡2x, and u′=1u′=1 and v=12sin2xv=12sin⁡2x.

[−12x2cos2x]54ππ2+∫54ππ2xcos2xdx=[−12x2cos2x+12xsin2x]54ππ2−∫54ππ212sin2xdx=[−12x2cos2x+12xsin2x +14cos2x]54ππ2[−12x2cos⁡2x]π254π+∫π254πxcos⁡2xdx=[−12x2cos⁡2x+12xsin⁡2x]π254π−∫π254π12sin⁡2xdx=[−12x2cos⁡2x+12xsin⁡2x +14cos⁡2x]π254π

Before we substitute in the limits it is worth noticing that all our trigonometric functions have an argument of 2x2x. When substituting x=5π4x=5π4 or x=π2x=π2 into the trigonometric functions we will get the following values.

cos(5π2)=0cos⁡(5π2)=0

sin(5π2)=1sin⁡(5π2)=1

sinπ=0sin⁡π=0

cosπ=−1cos⁡π=−1

Therefore when we evaluate the expression we get

(0+12×5π4×1+0)−(−12×(π2)2×−1+0+14×1)=5π8−π28+14.(0+12×5π4×1+0)−(−12×(π2)2×−1+0+14×1)=5π8−π28+14.

We can use the answer given in the question to check our values for a,ba,b and cc and to find the value for dd.

πd−π2+ca=5π8−π28+14=5π−π2+28πd−π2+ca=5π8−π28+14=5π−π2+28

We can be confident that d=5d=5 since our values for aa and cc match what we have already found.
