Product Rule & Integration by Parts
Review question

Can we find all three integrals?
Add to your resource collection

Ref: R5126

Solution

  1. Prove that
    ∫12π0cosx3+cos2xdx=14ln3.∫012πcos⁡x3+cos2⁡xdx=14ln⁡3.

If the denominator of the integrand were an expression involving sinxsin⁡x we might be able to integrate by substitution…

Since cos2x=1−sin2xcos2⁡x=1−sin2⁡x, we can write
∫12π0cosx3+cos2xdx=∫12π0cosx4−sin2xdx.∫012πcos⁡x3+cos2⁡xdx=∫012πcos⁡x4−sin2⁡xdx.
 Now using partial fractions we can expand, 14−z2=14(12−z+12+z)14−z2=14(12−z+12+z). Then we can integrate using the substitutions u=2±sinxu=2±sin⁡x.
∫12π0cosx4−sin2xdx=14∫12π0cosx2−sinx+cosx2+sinxdx=14[−ln||2−sinx||+ln||2+sinx||]12π0=14[ln||||2+sinx2−sinx||||]12π0=14ln3∫012πcos⁡x4−sin2⁡xdx=14∫012πcos⁡x2−sin⁡x+cos⁡x2+sin⁡xdx=14[−ln⁡|2−sin⁡x|+ln⁡|2+sin⁡x|]012π=14[ln⁡|2+sin⁡x2−sin⁡x|]012π=14ln⁡3
  1. Integrate with respect to xx
    (a)x(x−1)3(a)x(x−1)3

If the numerator were a constant, this would look much more straight-forward, so let’s try rearranging…

We can write,
∫x(x−1)3dx=∫x−1(x−1)3+1(x−1)3dx=∫1(x−1)2+1(x−1)3dx.∫x(x−1)3dx=∫x−1(x−1)3+1(x−1)3dx=∫1(x−1)2+1(x−1)3dx.
 If u=x−1u=x−1, then
∫1(x−1)2+1(x−1)3dx=∫u−2+u−3du=u−1−1+u−2−2+c=11−x−12(x−1)2+c∫1(x−1)2+1(x−1)3dx=∫u−2+u−3du=u−1−1+u−2−2+c=11−x−12(x−1)2+c

where cc is a constant of integration.

  1. Integrate with respect to xx
    (b)(lnx)2.(b)(ln⁡x)2.

The general formula for integration by parts is

∫udvdxdx=uv−∫vdudxdx.∫udvdxdx=uv−∫vdudxdx.

If we let u=(lnx)2u=(ln⁡x)2 and v=xv=x then dvdx=1dvdx=1 and dudx=2lnx×1xdudx=2ln⁡x×1x. Integrating by parts, we see that

∫(lnx)2dx=x(lnx)2−2∫lnxdx.(1)(1)∫(ln⁡x)2dx=x(ln⁡x)2−2∫ln⁡xdx.
 In order to integrate lnxln⁡x, we again integrate by parts using u=lnxu=ln⁡x and v=xv=x. Then dvdx=1dvdx=1 and dudx=1xdudx=1x and we have
∫lnxdx=xlnx−∫x1xdx=xlnx−x.∫ln⁡xdx=xln⁡x−∫x1xdx=xln⁡x−x.
 After substituting this into (1)(1), we can conclude that
∫(lnx)2dx=x(lnx)2−2(xlnx−x)=x(lnx)2−2xlnx+2x+c∫(ln⁡x)2dx=x(ln⁡x)2−2(xln⁡x−x)=x(ln⁡x)2−2xln⁡x+2x+c

where cc is, again, a constant of integration.
