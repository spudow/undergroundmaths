Solution

Below are several statements about the quadratic equation

ax2+bx+c=0,ax2+bx+c=0,
where aa, bb and cc are allowed to be any real numbers except that aa is not 00.

For each statement, decide whether MUST, MAY or CAN’T is the correct word to use in the statement.

To say that something MUST be the case, we need it to be true in all cases; we will need to give a convincing explanation (a proof) of why this must be always true.

To show that something CAN’T be the case, we likewise need to give a convincing explanation (a proof) of why.

To show that something MAY be the case, we need to give an example when it is true and an example when it is false. If you want a harder challenge, can you determine exactly when it is and when it is not true?

You may like to explore some of the scenarios on the cards using graphing software or the interactive graph in this resource.

  1. If a<0a<0, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have real roots.

Knowing that a<0a<0 tells us that this is a “vertex up” parabola. For example, it could look like one of these two graphs:

The graph of the left is a sketch of y=−x2+1y=−x2+1, while the one on the right is y=−x2−1y=−x2−1.

This shows that if a<0a<0, then ax2+bx+c=0ax2+bx+c=0 MAY have real roots.

  1. If b2−4ac=0b2−4ac=0, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have one repeated real root.

From the quadratic formula, we know that the equation ax2+bx+c=0ax2+bx+c=0 (with a≠0a≠0) has roots

x=−b±b2−4ac⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√2a.x=−b±b2−4ac2a.
In this case, we are told that the discriminant b2−4acb2−4ac is 00, so the only real root of the equation is x=−b2ax=−b2a.

Therefore, if b2−4ac=0b2−4ac=0, then ax2+bx+c=0ax2+bx+c=0 MUST have one repeated real root.

  1. If ax2+bx+c=0ax2+bx+c=0 has no real roots, then ax2+bx−c=0ax2+bx−c=0
    MUST / MAY / CAN’T have two distinct real roots.

If the equation has no real roots, then the graph of y=ax2+bx+cy=ax2+bx+c (which intersects the yy-axis at cc) does not cut the xx-axis, and its vertex is on the same side of the xx-axis as cc, as in the following sketches, the left one showing the case of a>0a>0 and the right one showing a<0a<0:

The graph of y=ax2+bx−cy=ax2+bx−c is obtained by translating the first graph in the yy-direction until it intersects the yy-axis at −c−c. Since the direction of the parabola does not change (it remains “vertex up” or “vertex down”), it must now intersect the xx-axis in two distinct points.

Therefore, if ax2+bx+c=0ax2+bx+c=0 has no real roots, then ax2+bx−c=0ax2+bx−c=0 MUST have two distinct real roots.


Alternatively, we can argue algebraically. Since the equation ax2+bx+c=0ax2+bx+c=0 has no real roots, we can say something about the discriminant: we know that b2−4ac<0b2−4ac<0.

Since b2b2 is the square of a real number, it is always non-negative (that is, b2≥0b2≥0), so b2<4acb2<4ac tells us that 4ac4ac is positive.

The equation ax2+bx−c=0ax2+bx−c=0 has discriminant b2+4acb2+4ac, and now this must be positive.

So ax2+bx−c=0ax2+bx−c=0 has two distinct real roots.

  1. If b2a<4cb2a<4c, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have two distinct real roots.

The tempting thing to do is to multiply both sides of the inequality by aa, to obtain b2<4acb2<4ac. This would give b2−4ac<0b2−4ac<0, so that the equation couldn’t have two distinct real roots – it would have no real roots.

Unfortunately, we don’t know whether aa is positive or negative, so when we multiply both sides of the inequality by aa, we may be changing the direction of the inequality.

If aa is positive, then b2<4acb2<4ac and the equation has no real roots.

However, if aa is negative, then multiplying the given inequality by aa will yield b2>4acb2>4ac, so b2−4ac>0b2−4ac>0 and the equation has two distinct real roots.

So if b2a<4cb2a<4c, then ax2+bx+c=0ax2+bx+c=0 MAY have two distinct real roots, depending upon the sign of aa.

  1. If b=0b=0, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have one repeated real root.

If we start by taking a=1a=1, then we have the family of quadratics y=x2+cy=x2+c, which look like this:

We can clearly see that we may have two real roots, one repeated real root (when c=0c=0) or no real roots.

So if b=0b=0, then ax2+bx+c=0ax2+bx+c=0 MAY have one repeated real root.


Alternatively, we could consider this from an algebraic perspective.

The discriminant is b2−4ac=−4acb2−4ac=−4ac, since b=0b=0. For the equation to have one repeated real root, we require the discriminant to be zero. This will be true if and only if c=0c=0 (as a≠0a≠0), so ax2+bx+c=0ax2+bx+c=0 MAY have one repeated real root.

  1. The equation ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have three real roots.

The equation CAN’T have three real roots, as a quadratic equation can have at most two roots. (We can see this from the quadratic formula or from thinking about the shape of the graph.)

  1. If c=0c=0, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have real roots.

If c=0c=0, then the equation is of the form ax2+bx=x(ax+b)=0ax2+bx=x(ax+b)=0, and so x=0x=0 is a real root.

In fact x=−bax=−ba is also a real root, so if b≠0b≠0 then the equation has two distinct real roots, while if b=0b=0, it has one repeated real root.

Alternatively, we could calculate the discriminant, b2−4ac=b2b2−4ac=b2 (since c=0c=0). Since b2b2 is a square, it is never negative, so the equation always has either two distinct or one repeated real root.

So if c=0c=0, then ax2+bx+c=0ax2+bx+c=0 MUST have real roots.

  1. The equation ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have the same number of real roots as ax2−bx+c=0ax2−bx+c=0.

Changing the +bx+bx term to −bx−bx in the equation y=ax2+bx+cy=ax2+bx+c has the same effect as replacing xx by −x−x (since a(−x)2=ax2a(−x)2=ax2).

So the effect on the graph of y=ax2+bx+cy=ax2+bx+c is to reflect it in the yy-axis, as in the following sketches:

So we see that ax2+bx+c=0ax2+bx+c=0 and ax2−bx+c=0ax2−bx+c=0 MUST have the same number of real roots. We can even see more than this: the roots of ax2−bx+c=0ax2−bx+c=0 are the negatives of the roots of ax2+bx+c=0ax2+bx+c=0.


Alternatively, we could think in terms of the discriminant.

The equations ax2+bx+c=0ax2+bx+c=0 and ax2−bx+c=0ax2−bx+c=0 both have discriminant b2−4acb2−4ac, so they must have the same number of real roots (whatever that number is).

  1. If ax2+bx+c=0ax2+bx+c=0 has two distinct real roots, then we
    MUST / MAY / CAN’T have ac<b24ac<b24.

If the equation has two distinct real roots, then its discriminant must be positive, that is, b2−4ac>0b2−4ac>0.

Now we can rearrange that to obtain 4ac<b24ac<b2. We can now divide this inequality by 44 (which is positive) to obtain ac<b24ac<b24.

So if ax2+bx+c=0ax2+bx+c=0 has two distinct real roots, then we MUST have ac<b24ac<b24.

  1. If c>0c>0, then ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have two distinct real roots.

The value of cc tells us where the quadratic crosses the yy-axis. Here are some sketches of graphs with a positive yy-intercept.

So we can see from this that if c>0c>0, then ax2+bx+c=0ax2+bx+c=0 MAY have two distinct real roots.

Explicit examples of quadratic equations would include x2−3x+2=(x−1)(x−2)=0x2−3x+2=(x−1)(x−2)=0, which has two distinct real roots, and x2+1=0x2+1=0, which has no real roots.


Alternatively, if we consider the discriminant b2−4acb2−4ac, if we take a>0a>0, then 4ac4ac is positive. So if we choose bb with b2>4acb2>4ac, we will have b2−4ac>0b2−4ac>0 (giving two distinct real roots), while if we choose bb with 0≤b2<4ac0≤b2<4ac, then b2−4ac<0b2−4ac<0 and there are no real roots.

  1. The equation ax2+bx+c=0ax2+bx+c=0
    MUST / MAY / CAN’T have the same number of real roots as cx2+bx+a=0cx2+bx+a=0.

There is no obvious way of relating the graph of y=ax2+bx+cy=ax2+bx+c to that of y=cx2+bx+ay=cx2+bx+a, though it would be fine to draw them both and compare them.

We consider instead the discriminants of the two equations. The discriminant of ax2+bx+c=0ax2+bx+c=0 is b2−4acb2−4ac, while that of cx2+bx+a=0cx2+bx+a=0 is b2−4cab2−4ca. So the discriminants are equal, and so the equations have the same number of roots.

However, we made an assumption here which may not be valid. The equation ax2+bx+c=0ax2+bx+c=0 is certainly a quadratic, as aa is non-zero. However, cx2+bx+c=0cx2+bx+c=0 is only a quadratic if cc is non-zero. But what if c=0c=0? In this case, cx2+bx+a=0cx2+bx+a=0 is just the linear equation bx+a=0bx+a=0 (assuming bb is non-zero). So if c=0c=0, the equation cx2+bx+a=0cx2+bx+a=0 has one real root, which will not be the same number of real roots as ax2+bx+c=0ax2+bx+c=0.

As explicit examples, the equation x2−3x+2=(x−1)(x−2)=0x2−3x+2=(x−1)(x−2)=0 has two distinct real roots. The equation obtained by reversing the coefficients is 2x2−3x+1=02x2−3x+1=0, and this factorises as 2x2−3x+1=(2x−1)(x−1)=02x2−3x+1=(2x−1)(x−1)=0 so has roots 11 and 1212—it too has two distinct real roots.

On the other hand, the equation x2−x=0x2−x=0 has two distinct real roots (since x2−x=x(x−1)=0x2−x=x(x−1)=0, which has roots x=0x=0 and x=1x=1). The equation obtained by reversing the coefficients is 0x2−x+1=00x2−x+1=0, that is, −x+1=0−x+1=0, and this has just one real root (x=1x=1).

Therefore, the equation ax2+bx+c=0ax2+bx+c=0 MAY have the same number of real roots as cx2+bx+a=0cx2+bx+a=0.

  1. If ax2+bx+c=0ax2+bx+c=0 has no real roots, then −ax2−bx−c=0−ax2−bx−c=0
    MUST / MAY / CAN’T have two distinct real roots.

One way to think about this is that the graph of y=−ax2−bx−cy=−ax2−bx−c is the reflection of the graph y=ax2+bx+cy=ax2+bx+c in the xx-axis. They therefore have the same number of roots as each other.

These two graphs show examples of this. On the left, y=(x−1)(x−2)y=(x−1)(x−2) has two real roots; on the right, y=x2+1y=x2+1 has none. It is clear that their reflections have the same number of real roots.

Algebraically, if we multiply −ax2−bx−c=0−ax2−bx−c=0 by −1−1, we obtain ax2+bx+c=0ax2+bx+c=0, which must therefore have the same roots as the original equation. In some sense, they are essentially the same equation.

Therefore, if ax2+bx+c=0ax2+bx+c=0 has no real roots, then −ax2−bx−c=0−ax2−bx−c=0 CAN’T have two distinct real roots.
