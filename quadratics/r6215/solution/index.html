Quadratics
Review question

How do we solve x+3x⎯⎯√−1/2=0x+3x−1/2=0?
Add to your resource collection

Ref: R6215
Printable/supporting materials Fullscreen mode Teacher notes
  • Question
  • Solution

Solution

  1. Use the substitution x⎯⎯√=yx=y (where y≥0y≥0) to find the real root of the equation
    x+3x⎯⎯√−12=0.x+3x−12=0.

Why must we have y≥0y≥0? It is because when we write x⎯⎯√x, we specifically mean the positive square root (or zero if x=0x=0).

So when we solve the quadratic in yy, we must be careful to only consider positive roots of that quadratic.

As the question suggests, we rewrite the equation with yy for x⎯⎯√x, and therefore y2=xy2=x so

y2+3y−12=0.y2+3y−12=0.

We solve this by completing the square:
⟺⟺⟺(y+32)2−94−12=0(y+32)2=114y+32=±11⎯⎯⎯⎯√2y=−3±11⎯⎯⎯⎯√2.(y+32)2−94−12=0⟺(y+32)2=114⟺y+32=±112⟺y=−3±112.

We could also solve this equation using the quadratic formula or any other method you know.

We need to be a bit careful here—recall that the question told us that we must have y≥0y≥0. So we take

y=−3+11⎯⎯⎯⎯√2,y=−3+112,
and so we have
x=y2=10−311⎯⎯⎯⎯√2.x=y2=10−3112.

We can check that this satisfies the original equation (remembering we know what x⎯⎯√x is since this is the value we found for yy):

x+3x⎯⎯√−12=10−311⎯⎯⎯⎯√2+3−3+11⎯⎯⎯⎯√2−12=12−12=0.x+3x−12=10−3112+3−3+112−12=12−12=0.

  1. Find all real roots of the following equations:

    1. x+10x+2⎯⎯⎯⎯⎯⎯⎯⎯√−22=0x+10x+2−22=0;

A good general tip: if you’re given a suggestion for the first part of the question, and you can’t immediately see how to solve the next part, try using a similar method to that suggested in the suggestion.

You may have to modify the suggestion a little, though!

Following the method for the first question, we try the substitution y=x+2⎯⎯⎯⎯⎯⎯⎯⎯√y=x+2, again for y≥0y≥0. This time, we have y2=x+2y2=x+2, or x=y2−2x=y2−2, so we are now solving the equation

y2+10y−24=0.y2+10y−24=0.
This equation can be factorised as
y2+10y−24=(y+12)(y−2)=0.y2+10y−24=(y+12)(y−2)=0.
 Again, we can disregard the solution y=−12y=−12, since y≥0y≥0, and see that y=2y=2. This implies that x=2x=2 is the only possible solution. We check whether this is a valid solution:
x+10x+2⎯⎯⎯⎯⎯⎯⎯⎯√−22=2+10(2)−22=0.✓x+10x+2−22=2+10(2)−22=0.✓

  1. Find all real roots of the following equations:

    1. x2−4x+2x2−8x−3⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√−9=0x2−4x+2x2−8x−3−9=0.

Let’s try the substitution y=2x2−8x−3⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√y=2x2−8x−3. We can rearrange this to find that x2−4x=(y2+3)/2x2−4x=(y2+3)/2, and so we this time study the equation

y2+32+y−9=0,y2+32+y−9=0,
or equivalently,
y2+2y−15=0.y2+2y−15=0.

We can again factorise to find that

y2+2y−15=(y+5)(y−3)=0.y2+2y−15=(y+5)(y−3)=0.
We must again discard the negative solution, and consider y=3y=3. To find xx, we must now solve x2−4x=(32+3)/2=6x2−4x=(32+3)/2=6, so x2−4x−6=0x2−4x−6=0. Applying the quadratic formula, we deduce that
x=2±10⎯⎯⎯⎯√.x=2±10.

When squaring to solve equations, we may gain extra ‘solutions’ that don’t satisfy the original equation, so we must check them.

But in this case, we have

(x2−4x)+2x2−8x−3⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√−9=6+3−9=0,(x2−4x)+2x2−8x−3−9=6+3−9=0,
so the two solutions are indeed x=2±10⎯⎯⎯⎯√x=2±10.
