Bouncing to nothing

Solution

A ball is thrown horizontally at a speed of 2ms−12ms−1 from a point 1.25m1.25m above OO which is on a flat horizontal surface. After each bounce it reaches a height which is 75%75% of its maximum height after the previous bounce. Ignore air resistance and take the acceleration of gravity, gg, to be 10ms−210ms−2.

  • Sketch the trajectory of the ball.

In between bounces, the ball is a projectile in free-fall so it’s trajectory will follow a parabola. After each bounce it joins a different parabola, with a maximum at 75%75% of the height of the previous one.

This diagram might raise a number of questions which probably can’t be answered without doing some calculations that we’ll get onto later.

  • Are all the parabolic sections the same “width”?
  • If not, how are they related?
  • How does the time taken for each section vary?
  • Does this sequence of parabolas continue indefinitely?
  • Calculate tntn, the time the ball is in the air between the nthnth and (n+1)th(n+1)th bounces.

Between bounces, the ball is a projectile moving under gravity. We can treat the horizontal and vertical motions as independent of one another. The horizontal motion has no effect on the time between bounces.

Vertically, let’s start by thinking about the first section, before the first bounce. The ball starts with zero vertical velocity and we want to know the time taken to fall under gravity through a distance 54m54m.
s54t2=ut+12at2=0+1210t2=14s=ut+12at254=0+1210t2t2=14

So the time for this section, t0=12t0=12.

We can do a similar calculation for tntn (for n≥1n≥1), using the reduced maximum height. The height after the nthnth bounce will be

54×(34)n.54×(34)n.
So the constant acceleration formula for the downward journey gives us
54×(34)n=0+1210t254×(34)n=0+1210t2
 and so the time for the whole section (up and down) will be
tn=2×15×54×(34)n⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(3⎯⎯√2)n.tn=2×15×54×(34)n=(32)n.

Why did we include a factor of 22 at the start of this expression?

This starts to answer some of our earlier questions. Each section of the trajectory takes less time than the previous one, by a factor of 3⎯⎯√/23/2. Assuming the horizontal velocity is a constant 2ms−12ms−1, what is the distance between successive bounces?

  • How far from OO does the ball get before it stops bouncing?

This might seem like a strange question – does it not keep bouncing forever?

We could take a pragmatic approach such as, “in reality a ball won’t appear to be bouncing when the height gets below a few centimetres. Find out how many bounces that will take and add up the times or distances for that many bounces.” Instead, we have considered a theoretical ball for which any bounce can be measured…

The times for the sections of the trajectory form a sequence,

12,3⎯⎯√2,(3⎯⎯√2)2,(3⎯⎯√2)3,⋯12,32,(32)2,(32)3,⋯
Apart from the first term, t0t0, this is a geometric sequence with first term, t1=3√2t1=32 and common ratio 3√232.

As the common ratio is less than one we can calculate the sum to infinity of its terms.

∑n=0∞tn=12+3⎯⎯√/21−3⎯⎯√/2=72+23⎯⎯√≈7.0∑n=0∞tn=12+3/21−3/2=72+23≈7.0

Surprisingly, this tells us that although the ball bounces an infinite number of times, the time between bounces gets shorter and shorter, so that after 77 seconds all the bounces have been completed and the ball no longer moves vertically at all!

Assuming the horizontal velocity is a constant 2ms−12ms−1, the ball will be about 14m14m from OO when it stops bouncing. After that it will be rolling along the ground.

If we had based our calculation on a finite number of bounces we would have found a shorter distance than this.

In reality, the horizontal velocity would probably not be constant, but how it changes at each bounce is hard to model.

Some more questions to think about…

  • What assumptions were made in the model of this situation? How might the answer be different if we had made different assumptions?
  • Can you think of any other effects that might change the outcome?
  • If we tried this experimentally what do you think would actually happen? Do you think this model would be accurate?
  • If we varied the initial height, how would the total bouncing time change?
  • If we drew a curve through the maximum points of all the parabolic sections, what shape would the curve be? [Express xx and yy in terms of nn…]
