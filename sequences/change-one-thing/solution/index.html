Solution

Find the last term of the sequence that starts with 33, has a common difference of 1212 and has 2525 terms.

  • In what ways can this problem be solved?

Let’s write the sequence numerically: 3,312,4,412,...3,312,4,412,...

The first term is 33, and after that there will be 2424 other terms. 1212 is being added for each term, so over 2424 terms, 1212 will be added. Therefore the 2525th term will be 3+12=153+12=15.

How does this example help us explain why the nth term of an arithmetic sequence is an=a1+(n−1)dan=a1+(n−1)d? Why is it n−1n−1 rather than nn?

We could also plot the points of the sequence on a graph.

The sequence is a linear relationship, so we can draw a linear graph through the points with gradient of 1212. The equation of the line is y=12x+cy=12x+c.

What is the value of cc?

The equation of the line is y=12x+212y=12x+212, and so the 2525th term will be,
y=12×25+212=15.y=12×25+212=15.

How does thinking about the sequence graphically and the equation y=12x+212y=12x+212, link to an=a1+(n−1)dan=a1+(n−1)d, the formula for the nth term of an arithmetic sequence?

Change only one aspect of the sequence in order to make the last term 100100.

  • What aspects can be changed?
  • What would they change to?

We are told about three aspects of the sequence: the first term, the number of terms, and the common difference.

The first term

If we change the first term, then we still have a common difference of 1212 and 2525 terms, so difference between the first and last term will still be 1212. Therefore, to make the last term 100100 the first term would be 8888.

If we think about the old and new sequences as linear graphs then we have a translation of 8585 units in the yy direction.

The number of terms

If we change the number of terms then we have the same intitial sequence, with a first term of 33 and a common difference of 1212. This means that we can look at the original graph of y=12x+212y=12x+212, and we want to know when y=100y=100. Solving for xx gives us x=195x=195, i.e. there would be 195195 terms in this sequence.

The common difference

If we change the common difference we want a sequence that starts at 33 and gets to 100100 by the 2525th term. This means we are adding on 9797 in 24 terms. Therefore the common difference will be 97249724. Graphically we have a line that still passes through the point (1,3)(1,3), but also passes through (25,100)(25,100) giving the equation y=9724x−2524y=9724x−2524.

We might have tried to change the common difference in a different way. We could make it a common ratio, and change the sequence from arithmetic to geometric. Why can’t a geometric sequence with a first term of 33, and common ratio of 1212 ever have a nth term of 100100?

Is it possible to have other sequences that have a first term of 33, and a 2525th term of 100100?
