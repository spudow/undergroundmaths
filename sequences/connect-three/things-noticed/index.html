Things you might have noticed

Write down three numbers. Can you find an arithmetic progression containing all three of these numbers (in ANY order)?

Is it always possible to find an arithmetic progression containing any three numbers chosen (in ANY order)?

We will consider these questions in the context of the cards given in the Suggestion.

Cards C, D, E and I

C

44, 88, 1414

D

7878, 3939, 169169

E

−11−11, −3−3, 1919

I

22, 55, 55

ResetShow all hidden cardsExport imagePrint

If we want them to belong to an arithmetic progression (from now on referred to as an AP), we need to identify a common difference for each set of three numbers.

  • How might you do this for card C?
  • Can the same approach be applied to cards D, E & I?
  • Is it possible to find an AP that contains the numbers in card D in their given order?
  • What is different about card I? Is there ever an occasion where an AP could contain a repeated value?

Let’s say we suggest the following AP for card C,

2,4,6,8,…2,4,6,8,…
Are there any other, different APs that could have been suggested?

Cards F and G

F

1212, ππ, 5π−25π−2

G

11, π2π2, ππ

We might choose to look at card F first. Beginning as for cards C, D and E, by looking for a common difference we find that the difference between the first two values is π−12π−12 and the second two values is 4π−24π−2. Now, for us to find an AP we require that

p(π−12)=q(4π−2)p(π−12)=q(4π−2)
where pp and qq are integers.

Can you see any values for pp and qq that could work here?

  • What about if q=1q=1 (i.e. 4π−24π−2 is a multiple of π−12π−12)?

Now let’s think about card G. As before, looking for a common difference we find that the difference between the first two values is π2−1π2−1 and the second two values is π2π2. But what do we do with this information now?

As above, if we have an AP then we can write

p(π2−1)=q(π2)p(π2−1)=q(π2)
where pp and qq are integers.

Expanding the brackets and rearranging we have

π=2pp−q.π=2pp−q.

Why did we choose to rearrange the equation to make ππ the subject?

  • Why might this be useful? What do we know about ππ?

Could pp ever be equal to qq?

  • What would this mean for the progression?
  • What would it mean algebraically?

π=2pp−qπ=2pp−q

As ππ is irrational and all the other terms in the expression are integers, this is impossible and we have reached a contradiction. So it is NOT possible to find an AP containing all three of the terms on card G.

Based on your thinking so far, or otherwise, do you have any feeling about whether or not it will be possible to find APs for the numbers on these cards?

Below is a way to think about card A. It proves that it is impossible to find an AP containing all three of the values.

Does this mean that it is impossible to find an AP for cards B and H too?

The difference between the first two values is 3⎯⎯√−2⎯⎯√3−2 and between the second two values is 5⎯⎯√−3⎯⎯√5−3.

As for cards F and G, if we have an AP then we can write

p(3⎯⎯√−2⎯⎯√)=q(5⎯⎯√−3⎯⎯√)p(3−2)=q(5−3)
where pp and qq are integers.

There are not ‘obvious’ values of pp and qq that will work here so let’s continue as we did for card G.

Expanding the expression and rearranging we have

(p+q)3⎯⎯√=p2⎯⎯√+q5⎯⎯√.(p+q)3=p2+q5.

At this point we have three different irrational terms in our equation. This is different to what happened with card G.

What happens if we square both sides?

Squaring both sides we get

3(p+q)2=2p2+5q2+(p+q)10⎯⎯⎯⎯√.3(p+q)2=2p2+5q2+(p+q)10.

Rearranging to make 10⎯⎯⎯⎯√10 the subject gives us

10⎯⎯⎯⎯√=3(p+q)2−2p2+5q2p+q.10=3(p+q)2−2p2+5q2p+q.
As 10⎯⎯⎯⎯√10 is irrational, but all of the terms on the right-hand side are integers, we have a contradiction. So it is NOT possible to find an AP containing all three of the values on card A.

You might like to think about cards B and H in a similar way before looking at the suggestions below for card H.

The difference between the first two values is 43⎯⎯√−243−2 and between the second two values is 63⎯⎯√−363−3.

As previously, if we have an AP then we can write

p(43⎯⎯√−2)=q(63⎯⎯√−3)p(43−2)=q(63−3)
where pp and qq are integers.

We might now notice that the expressions within the brackets here can be factorised to give

2p(23⎯⎯√−1)=3q(23⎯⎯√−1)2p(23−1)=3q(23−1)
and then simplified to show that
2p=3q.2p=3q.

This tells us that we can find values of pp and qq, and therefore that we can identify a common difference and then an AP that contains the values on this card.

What might have happened if we had continued without factorising the expression

p(43⎯⎯√−2)=q(63⎯⎯√−3)?p(43−2)=q(63−3)?

If we had continued as for card A, we might have expanded the brackets and rearranged to get

(4p−6q)3⎯⎯√=2p−3q(4p−6q)3=2p−3q
but this suggests that
3⎯⎯√=2p−3q2(2p−3q)=12.3=2p−3q2(2p−3q)=12.
 This is a contradiction and suggests that we cannot find an AP containing the values on this card.

Can you identify what has gone wrong with our line of reasoning here?
