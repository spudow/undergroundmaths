Comparing continued fractions

Solutions

Suppose that 0<a<b0<a<b. Which of the following continued fractions is bigger? Why?

12+13+1aor12+13+1b12+13+1aor12+13+1b

We know that 0<a<b0<a<b. So we also know that 1a>1b1a>1b and therefore that 3+1a>3+1b3+1a>3+1b.

It then follows that 13+1a<13+1b.13+1a<13+1b.

We can now say that 2+13+1a<2+13+1b2+13+1a<2+13+1b.

Finally, we can use this to say that

12+13+1a>12+13+1b.12+13+1a>12+13+1b.

Alternative approach

We could try to write the continued fractions as algebraic fractions with the same denominator.

12+13+1a=3a+17a+212+13+1a=3a+17a+2

and similarly,

12+13+1b=3b+17b+2.12+13+1b=3b+17b+2.

Writing these as fractions with a common denominator of (7a+2)(7b+2)(7a+2)(7b+2) gives us the numerators (3a+1)(7b+2)(3a+1)(7b+2) and (3b+1)(7a+2)(3b+1)(7a+2) to compare.

Expanding the brackets we get (3a+1)(7b+2)=21ab+6a+7b+2(3a+1)(7b+2)=21ab+6a+7b+2 and (3b+1)(7a+2)=21ab+6b+7a+2(3b+1)(7a+2)=21ab+6b+7a+2. The only parts of these two expressions that differ are the middle terms 6a+7b6a+7b and 6b+7a6b+7a. We know that a<ba<b so we can reason that 6a+7b>6b+7a6a+7b>6b+7a.

Therefore we can conclude that

12+13+1a>12+13+1b.12+13+1a>12+13+1b.

It is worth noting how much neater the first approach is. Whilst comparing fractions with common denominators sounds reasonable, it is not always necessary.

Suppose the fractions are continued in the same way, then which is bigger in the following pair and why?

12+13+14+1aor12+13+14+1b12+13+14+1aor12+13+14+1b

This problem is a further extension of the first. In the process of repeating the first approach above we can show that

2+13+14+1a>2+13+14+1b2+13+14+1a>2+13+14+1b

And therefore that

12+13+14+1a<12+13+14+1b.12+13+14+1a<12+13+14+1b.

Now compare:

12+13+14+⋯+199+1100+1a12+13+14+⋯+199+1100+1a
and the same thing with bb in place of aa.

Now the continued fractions have been expanded all the way down to 100+1a100+1a and 100+1b100+1b.

If we show the bottom line of the continued fraction as x+1ax+1a or x+1bx+1b then from above we can see that when xx is odd, the continued fraction with aa in it is bigger. However, when xx is even, the continued fraction with bb in it is bigger.

100100 is even, therefore

12+13+14+⋯+199+1100+1a<12+13+14+⋯+199+1100+1b.12+13+14+⋯+199+1100+1a<12+13+14+⋯+199+1100+1b.
