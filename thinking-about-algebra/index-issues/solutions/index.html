Is the left-hand side the same as the right-hand side?

(xy)12=xy⎯⎯⎯⎯√=x⎯⎯√y⎯⎯√=x12y12(xy)12=xy=xy=x12y12

It would seem so from the reasoning above, however what happens if x=−2x=−2 and y=−8y=−8? xy⎯⎯⎯⎯√xy will give us an answer of 44 but x⎯⎯√y⎯⎯√xy will not be defined.

We can only get a real value from a square root when the number is not negative, so x≥0x≥0 and y≥0y≥0.

The overall value of the expression (xy)12(xy)12 (and therefore x12y12x12y12) will also be ≥0≥0.

We have said that the value of the expressions (xy)12(xy)12 and x12y12x12y12 will be ≥0≥0. But will they always be positive? We may think not, as a square root has two solutions: a positive and a negative one.

However, the square root symbol √, and writing something as raised to the power of 1212, denotes the principal root, which is always the positive root. If we take a square root, as part of rearranging an equation then we may need to write ±√± or ±()12±()12 depending on the situation.

Can you think of an equation which involves taking a square root where you will always want the positive root only?

We know we can cube root any number, so x,y∈ℝx,y∈R. The value of (xy)23(xy)23 however, is restricted. How is it different from the value of (xy)53(xy)53 and why is that?

A cube root can give a positive or negative answer. Raising a negative to the power of 55 (or any odd number), will give you a negative answer so (xy)53(xy)53 can take any value. This is unlike (xy)23(xy)23, as squaring any number will always result in a positive answer. So (xy)23(xy)23 is restricted to being greater than or equal to zero.

  1. (xy)12=x13y23(xy)12=x13y23

On first glance, we might easily dismiss this as being incorrect. The question that was asked though, is ‘for what values of xx and yy do these statements make sense?’ Are there any values of xx and yy when this equation would be true?

If x=0x=0 or y=0y=0, then the equation is satisfied, but to see if there are any other possibilities we should try to solve this equation.

(xy)12=x13y23(xy)12=x13y23

For the left hand side to be defined, xy≥0xy≥0. In addition, xx and yy cannot be negative. Can you explain why?

We can write

then divide by x13y12x13y12, assuming x,y≠0x,y≠0,to get

x16=y16.x16=y16.

This is perhaps more solutions than we might have first thought. Can you see why this happens? Try some numbers if you are not sure.
