We know from working with numbers that dividing anything by zero causes problems. We can see this if we try to solve 3x=7x3x=7x by dividing by xx, or in the loss of the solution that was shown in part c.

If you substitute x=4x=4 into each side of the equation at the beginning, you’ll see that neither side has a value that exists, as we would be dividing by zero. We clearly can’t work with things that don’t exist so we must specify that in this case, x≠4x≠4.

If we started again with method 11, then we could multiply both sides by x−4x−4 because as x≠4x≠4, the denominator is not zero. Therefore when we get to x(x−4)=0x(x−4)=0, we already know x≠4x≠4, so the only solution is x=0x=0.

Unlike for c, we do not know anything about our value of xx. As x−2x−2 could be negative, we can’t multiply both sides by it.

Why can’t we multiply both sides by a value that could be negative?

Instead we can add or subtract.

5x+6x−2−2≤05x+6x−2−2≤0

We are now aware that we cannot multiply by x−2x−2 as it may be negative. Instead we need to think about what values this expression takes, and when it changes sign.

One way to do this is to look at the graph of the function y=3x+10x−2y=3x+10x−2 and see when it is positive and when it is negative.

We can see from the graph that there is a root between x=−3x=−3 and x=−4x=−4, and the function changes from positive to negative either side of this. This is where the numerator is equal to zero and so x=−103x=−103.

The graph has an asymptote at x=2x=2, i.e. the function cannot exist for this value of xx as the denominator would be zero. The function changes from positive to negative either side of this value too.

We want to know when 3x+10x−2≤03x+10x−2≤0, i.e. when it’s negative or zero.

So the solution is −103≤x<2−103≤x<2.
