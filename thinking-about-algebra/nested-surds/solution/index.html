Solution

For each card below, determine which non-negative values of aa, bb, cc, and dd, if any, make the equation true.

We have presented the solutions below in an order that seems sensible, starting with the perhaps more familiar statements and making use of these in some of the subsequent solutions.

  1. a⎯⎯√×b⎯⎯√=ab⎯⎯⎯⎯√a×b=ab

This is true for all positive values of aa and bb. It is also true when one or both of aa and bb is zero.

  1. a⎯⎯√b⎯⎯√=ab⎯⎯⎯⎯√ab=ab

This is true for all positive values of aa and bb and when a=0a=0. However, when b=0b=0 then both sides of the equation are undefined.

  1. ab⎯⎯√=ab⎯⎯⎯⎯√ab=ab

In the case where one or both of aa and bb is zero then the equation is true. It is also true if a=1a=1 because then both sides of the equation simplify to b⎯⎯√b.

However, if we select other positive values such as a=2a=2 and b=4b=4, then ab⎯⎯√=2×2=4ab=2×2=4 whereas ab⎯⎯⎯⎯√=2×4⎯⎯⎯⎯⎯⎯⎯⎯√=8⎯⎯√ab=2×4=8, so the equation is not true.

In more general terms, if we take the original equation and square both sides we get a2b=aba2b=ab. This equation can only be true when a2=aa2=a so when a=0a=0 or a=1a=1 as we have already identified.

  1. ab⎯⎯⎯⎯√a⎯⎯√+b⎯⎯√=1aba+b=1

For this to be true we require that ab⎯⎯⎯⎯√=a⎯⎯√+b⎯⎯√ab=a+b.

However, it is not easy to find values of aa and bb that satisfy this requirement. For example, taking a=1a=1 and b=4b=4 gives ab⎯⎯⎯⎯√=2ab=2 but a⎯⎯√+b⎯⎯√=3a+b=3.

It looks as though a=b=0a=b=0 might be a solution. But, we must remember that the original equation looked like ab⎯⎯⎯⎯√a⎯⎯√+b⎯⎯√=1aba+b=1 and taking aa and bb as zero would result in a division by zero, which is undefined.

We know that for the equation to be true we require that ab⎯⎯⎯⎯√=a⎯⎯√+b⎯⎯√ab=a+b. Subtracting a⎯⎯√a and b⎯⎯√b from both sides we get

ab⎯⎯⎯⎯√−a⎯⎯√−b⎯⎯√=0.ab−a−b=0.
Now, this can be factorised as
(a⎯⎯√−1)(b⎯⎯√−1)−1=0,(a−1)(b−1)−1=0,
 and subtracting the negative 11 gives us
(a⎯⎯√−1)(b⎯⎯√−1)=1.(a−1)(b−1)=1.

So if I pick two positive numbers which multiply to 11, say cc and 1c1c, then set a⎯⎯√−1=ca−1=c and b⎯⎯√−1=1cb−1=1c, it should work!

To confirm that this is the case we should work backwards from these solutions and check that they will work in the original equation.

If a⎯⎯√−1=ca−1=c, then a⎯⎯√=c+1a=c+1 and

a=(c+1)2.a=(c+1)2.

Similarly, if b⎯⎯√−1=1cb−1=1c then b⎯⎯√=1+1cb=1+1c and

b=(1+1c)2=(c+1c)2.b=(1+1c)2=(c+1c)2.

Then we have

ab⎯⎯⎯⎯√=(c+1)c+1c=(c+1)2c,ab=(c+1)c+1c=(c+1)2c,
and
a⎯⎯√+b⎯⎯√=c+1+c+1c=c2+2c+1c=(c+1)2c.a+b=c+1+c+1c=c2+2c+1c=(c+1)2c.

Returning to the original equation we now have

ab⎯⎯⎯⎯√a⎯⎯√+b⎯⎯√=((c+1)2c)((c+1)2c)=1.aba+b=((c+1)2c)((c+1)2c)=1.

So we have demonstrated that there are in fact infinitely many solutions to this equation.

Finally, it is sensible to try it out with some numbers. Let’s take c=2c=2 and then from above,

a=(c+1)2=9a=(c+1)2=9
and
b=(c+1c)2=(32)2=94.b=(c+1c)2=(32)2=94.

Substituting these values of aa and bb into the original equation gives

ab⎯⎯⎯⎯√a⎯⎯√+b⎯⎯√=(92)(92)=1aba+b=(92)(92)=1
as required!

  1. a⎯⎯√−b⎯⎯√=a−b⎯⎯⎯⎯⎯⎯⎯⎯⎯√a−b=a−b

When we have b=0b=0 then both sides of the equation will simplify to a⎯⎯√a and the equation holds. However, if we have a=0a=0 and b≠0b≠0, then the equation becomes −b⎯⎯√=−b⎯⎯⎯⎯⎯√−b=−b and, since there is no real solution to −b⎯⎯⎯⎯⎯√−b, the equation is not true.

If we try other positive values for aa and bb we find that the equation is not always true, for instance when a=4a=4 and b=1b=1 then a⎯⎯√−b⎯⎯√=1a−b=1 but a−b⎯⎯⎯⎯⎯⎯⎯⎯⎯√=3⎯⎯√a−b=3.

When we let b>ab>a we also run into problems as a−b⎯⎯⎯⎯⎯⎯⎯⎯⎯√a−b will have no real solutions.

We need to identify exactly when this equation is true. If we take the original equation and square both sides we get

a+b−2ab⎯⎯⎯⎯√=a−b.a+b−2ab=a−b.
Rearranging, this gives
b=ab⎯⎯⎯⎯√b=ab
 and squaring both sides to get rid of the square root gives
b2=ab.b2=ab.
 So we require that a=ba=b.

Substituting this back into the original equation to check shows that a=ba=b gives

a⎯⎯√−b⎯⎯√=0a−b=0
and
a−b⎯⎯⎯⎯⎯⎯⎯⎯⎯√=0,a−b=0,
 so a=ba=b is indeed a solution.

The equation is true when a=ba=b and for any value of aa when b=0b=0.

  1. a⎯⎯√+b⎯⎯√=a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b=a+b+4ab

In the case where one or both of aa and bb is zero the equation is true.

If we take a=1a=1 and b=4b=4 then a⎯⎯√+b⎯⎯√=1⎯⎯√+4⎯⎯√=3a+b=1+4=3. Also, a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=1+4+4×1×4⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=9⎯⎯√=3a+b+4ab=1+4+4×1×4=9=3. So the equation is true.

Taking another example, say a=4a=4 and b=9b=9 we have that a⎯⎯√+b⎯⎯√=4⎯⎯√+9⎯⎯√=5a+b=4+9=5 and a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=4+9+4×4×9⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=25⎯⎯⎯⎯√=5a+b+4ab=4+9+4×4×9=25=5. So the equation is still true.

So far we have selected square numbers for aa and bb because the expressions are much easier to simplify and hence demonstrate whether or not they are equal. If we are to be sure that the equation is true for all positive values of aa and bb then we need to prove it.

Starting with the left-hand side of the equation, squaring gives

(a⎯⎯√+b⎯⎯√)2=a+b+2ab⎯⎯⎯⎯√.(a+b)2=a+b+2ab.

This is already looking promising. Writing 2ab⎯⎯⎯⎯√2ab as 4ab⎯⎯⎯⎯⎯⎯√4ab gives us

(a⎯⎯√+b⎯⎯√)2=a+b+4ab⎯⎯⎯⎯⎯⎯√.(a+b)2=a+b+4ab.

Taking the square root of both sides now gives us the desired result

a⎯⎯√+b⎯⎯√=a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.a+b=a+b+4ab.

So this equation is true for all non-negative values of aa and bb.

  1. 5+26⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=a⎯⎯√+b⎯⎯√5+26=a+b

This question is essentially the same as the one above and we can use the same approach. Writing the left-hand side in the form a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b+4ab gives

5+26⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=5+4×6⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.5+26=5+4×6.

Comparing this to a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b+4ab we have that a+b=5a+b=5 and ab=6ab=6.

By inspection we can quickly see that a=2a=2 and b=3b=3 or vice versa. Therefore

5+26⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=2⎯⎯√+3⎯⎯√.5+26=2+3.

  1. 23−66−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=a⎯⎯√+b⎯⎯√23−66−42=a+b

This is a more complicated-looking version of the previous two problems. Our strategy worked for both of those so let’s apply it again.

Let’s start with the final part of this: 6−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√6−42.

Notice that there is a subtle difference between this and the previous questions. In the previous questions we made use of the expression a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b+4ab but this time we need a+b−4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b−4ab.

If we square this expression we get

a+b−4ab⎯⎯⎯⎯⎯⎯√=a+b−2ab⎯⎯⎯⎯√.a+b−4ab=a+b−2ab.
This is the result of squaring a⎯⎯√−b⎯⎯√a−b so we can write
a+b−4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=a⎯⎯√−b⎯⎯√a+b−4ab=a−b
 (assuming that a≥ba≥b).

We will try to write 6−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√6−42 in the form a⎯⎯√−b⎯⎯√a−b in order to simplify the expression.

Using the approach from above, 6−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=6−4×8⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√6−42=6−4×8 and comparing this to a+b−4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b−4ab tells us that a+b=6a+b=6 and that ab=8ab=8. By inspection we can see that a=2a=2 and b=4b=4 or vice versa. This time however, the order matters as we need to remember that 6−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√6−42 is non-negative.

Therefore we take a=4a=4 and b=2b=2, showing that 6−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=4⎯⎯√−2⎯⎯√=2−2⎯⎯√6−42=4−2=2−2.

Substituting this into the original nested surd gives

23−66−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=23−6(2−2⎯⎯√)⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.23−66−42=23−6(2−2).

We can then simplify the right-hand side by expanding the brackets to give

23−66−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=23−12+62⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=11+62⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.23−66−42=23−12+62=11+62.

We now need to repeat the process to write 11+62⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√11+62 as a⎯⎯√+b⎯⎯√a+b.

Writing 62⎯⎯√62 in the form 4ab⎯⎯⎯⎯⎯⎯√4ab we get

62⎯⎯√=72⎯⎯⎯⎯√=4×18⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.62=72=4×18.

Substituting this back into 11+62⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√11+62 gives

11+4×18⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.11+4×18.

Comparing this to a+b+4ab⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√a+b+4ab tells us that a+b=11a+b=11 and ab=18ab=18. By inspection we can see that a=9a=9 and b=2b=2 or vice versa.

Therefore

11+4×18⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=9⎯⎯√+2⎯⎯√=3+2⎯⎯√,11+4×18=9+2=3+2,

and overall we have

23−66−42⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=3+2⎯⎯√.23−66−42=3+2.

So the equation is true when a=9a=9 and b=2b=2.

  1. a⎯⎯√+bc⎯⎯√+d=(a⎯⎯√+b)(c⎯⎯√−d)a+bc+d=(a+b)(c−d)

As with the other equations, let’s start by considering zero values. If a=b=0a=b=0 then the equation is true. If one of aa and bb is zero then it is less clear whether or not the equation is true – it depends on the values of cc and dd too. If c=d=0c=d=0 then we get a zero divisor on the left-hand side.

Taking a numerical example such as a=1a=1, b=2b=2, c=4c=4 and d=3d=3 we find that

a⎯⎯√+bc⎯⎯√+d=35.a+bc+d=35.
However,
(a⎯⎯√+b)(c⎯⎯√−d)=3×−1=−3.(a+b)(c−d)=3×−1=−3.
 So we have shown that the equation is not true for all positive values of aa, bb, cc and dd.

We now need to consider whether there are any positive values for which the equation does hold. If we multiply both sides of the equation by c⎯⎯√+dc+d we get

a⎯⎯√+b=(a⎯⎯√+b)(c⎯⎯√−d)(c⎯⎯√+d).a+b=(a+b)(c−d)(c+d).
Dealing only with the case where a⎯⎯√+b≠0a+b≠0, we can divide through by a⎯⎯√+ba+b giving
1=(c⎯⎯√−d)(c⎯⎯√+d)=c−d2.1=(c−d)(c+d)=c−d2.

We have used the difference of two squares here.

We now have a required relationship between cc and dd,

c=d2+1.c=d2+1.

So the equation is true either when a=b=0a=b=0 with cc and dd not both 0, or when c=d2+1c=d2+1.
