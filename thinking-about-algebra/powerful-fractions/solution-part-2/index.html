Solution part 2

How might your thinking change if instead we had these two fractions?

Fraction C:10c+110c−1Fraction D:10d+110d−1Fraction C:10c+110c−1Fraction D:10d+110d−1

For what values of cc and dd will fraction C be greater than fraction D?

If we take a step back and compare this pair of fractions to the first pair we will hopefully notice that it is only the denominators that have changed.

  • The second term in each denominator has changed from +2+2 to −1−1.

How does this affect the value of the denominator?

In the first example the denominator was always positive. Is this still the case?

We might try to substitute values in place of cc and dd in order to get a sense of what is happening. There are four scenarios that we will need to consider:

  1. What happens if both cc and dd are positive?

  2. What happens if both cc and dd are negative?

  3. What happens if just one of cc or dd is positive?

  4. What happens if cc or dd is zero?

Now we might choose to take a similar approach as for the first pair of fractions. Re-writing this pair of fractions with a common denominator we have

Fraction C:(10c+1)(10d−1)(10c−1)(10d−1)Fraction D:(10d+1)(10c−1)(10d−1)(10c−1).Fraction C:(10c+1)(10d−1)(10c−1)(10d−1)Fraction D:(10d+1)(10c−1)(10d−1)(10c−1).

If either cc or dd is zero then the fractions are undefined (the denominator will be zero). For each of the other scenarios we might first consider whether or not the resulting fraction is positive or negative:

Scenario Sign of fraction CC Sign of fraction DD Conclusion
c>0c>0, d>0d>0 (+)(+)(+)(+)=+(+)(+)(+)(+)=+ (+)(+)(+)(+)=+(+)(+)(+)(+)=+ Inconclusive
c<0c<0, d<0d<0 (+)(−)(−)(−)=−(+)(−)(−)(−)=− (+)(−)(−)(−)=−(+)(−)(−)(−)=− Inconclusive
c>0c>0, d<0d<0 (+)(−)(+)(−)=+(+)(−)(+)(−)=+ (+)(+)(+)(−)=−(+)(+)(+)(−)=− Fraction CC > Fraction DD
c<0c<0, d>0d>0 (+)(+)(−)(+)=−(+)(+)(−)(+)=− (+)(−)(−)(+)=+(+)(−)(−)(+)=+ Fraction CC < Fraction DD

This means that we only need to investigate the first two scenarios. In both cases the denominators will be positive and so we can consider the numerators alone:

Numerator C:(10c+1)(10d−1)=10c+d−10c+10d−1Numerator C:(10c+1)(10d−1)=10c+d−10c+10d−1
and
Numerator D:(10d+1)(10c−1)=10c+d+10c−10d−1.Numerator D:(10d+1)(10c−1)=10c+d+10c−10d−1.

As before, it is only the middle terms which differ so we consider this part of the numerator of CC:

10d−10c10d−10c
and the equivalent part of the numerator of DD:
10c−10d.10c−10d.

Now, if c>0c>0 and d>0d>0 then fraction CC will be greater than fraction DD when d>cd>c.

We should check this makes sense. If we take c=1c=1 and d=2d=2 then we get

Fraction C:101+1101−1=119Fraction C:101+1101−1=119
and
Fraction D:102+1102−1=10199.Fraction D:102+1102−1=10199.

We can use equivalent fractions to show that 119>10199119>10199 and our solution holds.

If c<0c<0 and d<0d<0 then fraction CC will again be greater than fraction DD when d>cd>c.

Once again, we should check that this makes sense. If we take c=−2c=−2 and d=−1d=−1 then we get

Fraction C:10−2+110−2−1=−10199Fraction C:10−2+110−2−1=−10199
and
Fraction D:10−1+110−1−1=−119.Fraction D:10−1+110−1−1=−119.

We can show that −10199>−119−10199>−119 and our solution holds.

A summary of our conclusions is below, relating the original scenarios proposed.

Question Answer
  1. What happens if both cc and dd are positive?
 Fraction CC > Fraction DD when d>cd>c
  1. What happens if both cc and dd are negative?
 Fraction CC > Fraction DD when d>cd>c
  1. What happens if just one of cc or dd is negative?
 Fraction CC > Fraction DD when c>0c>0 and d<0d<0
  1. What happens if cc or dd is zero?
 Fraction CC or Fraction DD respectively will be undefined.

We might like to think of this problem graphically.

We can see that y=10x+110x−1y=10x+110x−1 is decreasing on the interval (−∞,0)(−∞,0) and also on (0,∞)(0,∞) (this is in contrast to the first problem in which the function was increasing).

Think about how each of the four questions posed above relates to regions of the graph.
