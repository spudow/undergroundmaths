Thinking about Algebra
Review question

Can we prove these inequalities involving a,b,ca,b,c and dd?

Solution

Prove that, if aa, bb are real,

ab≤(a+b2)2,...(∗)(∗)ab≤(a+b2)2,...

Since aa and bb are real, we have that
(a−b)2≥0.(a−b)2≥0.

⟺⟺⟺(a−b)2a2+b2−2ab(a+b)2−4ab4ab≥0,≥0,≥0,≤(a+b)2,(a−b)2≥0,⟺a2+b2−2ab≥0,⟺(a+b)2−4ab≥0,⟺4ab≤(a+b)2,

ab≤(a+b2)2.ab≤(a+b2)2.

Note also that we have equality on the first line of this argument if and only if a=ba=b, so the final inequality (∗)(∗) is also true if and only if a=ba=b.

Alternatively, we could have begun with the given statement (∗)(∗) and worked our way backwards to (a−b)2≥0(a−b)2≥0, ensuring that we use “if and only if” (⟺⟺) arguments the whole way.

… and deduce that, if aa, bb, cc, dd are positive,
abcd≤(a+b+c+d4)4,(†)(†)abcd≤(a+b+c+d4)4,

with equality only when all numbers are equal.

We may apply inequality (∗)(∗) to any two pairs of aa, bb, cc, dd:
ab≤(a+b2)2,(1)(1)ab≤(a+b2)2,
 and
cd≤(c+d2)2,(2)(2)cd≤(c+d2)2,

with equality in (1)(1) if and only if a=ba=b, and equality in (2)(2) if and only if c=dc=d.

Since both sides of (1)(1) and (2)(2) are positive, we have that

abcd≤(a+b2)2(c+d2)2=[(a+b2)(c+d2)]2,abcd≤(a+b2)2(c+d2)2=[(a+b2)(c+d2)]2,
with equality if and only if a=ba=b and c=dc=d, and we may further apply (∗)(∗) to the right hand side of this to find that

Note that “only when” is the same as saying “only if”. We have gone further and shown “when and only when”, or “if and only if”.

By giving dd a suitable value in terms of aa, bb, cc, or otherwise, prove that, if aa, bb, cc are positive,

abc≤(a+b+c3)3.abc≤(a+b+c3)3.

Approach 1

We want to modify our equation in such a way as to make it look more like (†)(†). It would be helpful if we could choose a value of dd that would turn the expression in the bracket into the one we want. Hence we require dd such that

a+b+c3=a+b+c+d4,a+b+c3=a+b+c+d4,
so multiplying by 44 and subtracting a+b+ca+b+c from both sides, we find that we must choose d=a+b+c3d=a+b+c3. Substituting this into (†)(†), we get
abc(a+b+c3)≤a+b+c+a+b+c344=(a+b+c3)4abc(a+b+c3)≤(a+b+c+a+b+c34)4=(a+b+c3)4

As aa, bb and cc are positive, we can divide both sides by a+b+c3a+b+c3 to get the expression we required:

abc≤(a+b+c3)3.abc≤(a+b+c3)3.

This question has covered the first few cases of the arithmetic mean–geometric mean (AM–GM) inequality. This states that if a1a1, …, anan are positive real numbers, then

a1+⋯+ann≥a1…an⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√n,a1+⋯+ann≥a1…ann,
that is, their arithmetic mean is at least as large as their geometric mean.

One of the proofs of this inequality is exactly as shown in this question: one proves the result first for n=2kn=2k for k=1k=1, 22, …, using the method of the first part of the question, and then one fills in the gaps for 2k−1<n<2k2k−1<n<2k using the technique of the second part.
