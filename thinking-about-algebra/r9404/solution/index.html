Review question

Can we factorise (x+y)6−(x−y)6(x+y)6−(x−y)6 completely?

Solution

Factorise completely
(x+y)6−(x−y)6.(x+y)6−(x−y)6.

Since 6=2×36=2×3, we can think of this expression as ((x+y)2)3−((x−y)2)3((x+y)2)3−((x−y)2)3 or as ((x+y)3)2−((x−y)3)2((x+y)3)2−((x−y)3)2.

There are then several identities that are helpful to us here:

  1. a2−b2=(a+b)(a−b)a2−b2=(a+b)(a−b),
  2. a3−b3=(a−b)(a2+ab+b2)a3−b3=(a−b)(a2+ab+b2),
  3. a3+b3=(a+b)(a2−ab+b2)a3+b3=(a+b)(a2−ab+b2),
  4. (a+b)3=a3+3a2b+3ab2+b3(a+b)3=a3+3a2b+3ab2+b3,
  5. (a−b)3=a3−3a2b+3ab2−b3(a−b)3=a3−3a2b+3ab2−b3.

We can check these identities by multiplying out the brackets. Identities (4) and (5) are examples of the binomial theorem.

If we start with (x+y)6−(x−y)6=((x+y)3)2−((x−y)3)2(x+y)6−(x−y)6=((x+y)3)2−((x−y)3)2, we can use (1) to get

(x+y)6−(x−y)6=((x+y)3−(x−y)3)((x+y)3+(x−y)3).(x+y)6−(x−y)6=((x+y)3−(x−y)3)((x+y)3+(x−y)3).

Now we can expand the inner brackets using the binomial theorem to get
(x+y)6−(x−y)6=((x3+3x2y+3xy2+y3)−(x3−3x2y+3xy2−y3))×((x3+3x2y+3xy2+y3)+(x3−3x2y+3xy2−y3))=(6x2y+2y3)(2x3+6xy2)=4xy(3x2+y2)(x2+3y2).(x+y)6−(x−y)6=((x3+3x2y+3xy2+y3)−(x3−3x2y+3xy2−y3))×((x3+3x2y+3xy2+y3)+(x3−3x2y+3xy2−y3))=(6x2y+2y3)(2x3+6xy2)=4xy(3x2+y2)(x2+3y2).

We cannot factorise this any further, as each bracket is the sum of two squares.

Alternatively, if we start with (x+y)6−(x−y)6=((x+y)2)3−((x−y)2)3(x+y)6−(x−y)6=((x+y)2)3−((x−y)2)3, we can use (2) to get

(x+y)6−(x−y)6=((x+y)2−(x−y)2)((x+y)4+(x+y)2(x−y)2+(x−y)4).(x+y)6−(x−y)6=((x+y)2−(x−y)2)((x+y)4+(x+y)2(x−y)2+(x−y)4).

Expanding the inner brackets using the binomial theorem and then simplifying will eventually give the same result, but it seems like a lot more work.

As a third route, we can simply expand the brackets in (x+y)6−(x−y)6(x+y)6−(x−y)6 directly using the binomial theorem. We have
(x+y)6(x−y)6=x6+(61)x5y+⋯+(65)xy5+y6=x6+6x5y+15x4y2+20x3y3+15x2y4+6xy5+y6=x6−6x5y+15x4y2−20x3y3+15x2y4−6xy5+y6,(x+y)6=x6+(61)x5y+⋯+(65)xy5+y6=x6+6x5y+15x4y2+20x3y3+15x2y4+6xy5+y6(x−y)6=x6−6x5y+15x4y2−20x3y3+15x2y4−6xy5+y6,
 so that
(x+y)6−(x−y)6=12x5y+40x3y3+12xy5=4xy(3x4+10x2y2+3y4)=4xy(x2+3y2)(3x2+y2).(x+y)6−(x−y)6=12x5y+40x3y3+12xy5=4xy(3x4+10x2y2+3y4)=4xy(x2+3y2)(3x2+y2).
