Solution

Can you work out the ratio ABOCABOC in each of these cases?

It’s helpful to label some more key points, so that we can talk about them conveniently. Let XX be the lower end point and YY the upper end point of the vertical line segment between AA and BB, as shown.

It looks as though triangles OAXOAX and CADCAD might be similar, so let’s try to prove that.

Opposite angles at a point are the same, so angles OAXOAX and CADCAD are the same. Also, DCDC and OEOE are parallel, so angles AOXAOX and ACDACD are the same. So triangles OAXOAX and CADCAD have the same angles, so they are similar.

You might be familiar with this as the ‘AAA’ reason for two triangles to be similar: all three angles are the same.

Now we can use that to deduce that certain ratios are the same. More precisely,

OACA=OXCD.OACA=OXCD.

Here’s a tip when dealing with similar triangles. When we said that triangles OAXOAX and CADCAD are similar, the points were in the right order. That is, vertex OO in triangle OAXOAX corresponds to vertex CC in triangle CADCAD, vertex AA in triangle OAXOAX corresponds to vertex AA in triangle CADCAD, and vertex XX in triangle OAXOAX corresponds to vertex DD in triangle CADCAD. Being careful about that ordering when we wrote down the similar triangles makes it easier to be clear about which sides of the triangles correspond.

But we know that OX=23CDOX=23CD, so

OACA=23.OACA=23.

This is useful, because we can now find OAOA in terms of OCOC. Clearly OA+AC=OCOA+AC=OC, so we must have OA=25OCOA=25OC (and also AC=35OCAC=35OC, although we don’t need that).

This seems helpful, especially if we can also find the length of OBOB. But that will be OK, by doing something very similar to the argument above.

Similarly, triangle CBYCBY is similar to triangle OBEOBE, so

CBOB=CYOE=13,CBOB=CYOE=13,
and so CB=13OBCB=13OB, so OB=34OCOB=34OC.

Now

ABOC=OB−OAOC=34−25=720.ABOC=OB−OAOC=34−25=720.

At this point, we should behave like mathematicians by trying to answer this question as efficiently as possible. How can we use our work on the first part to help? We can start to see which points to label and where we might look for helpful similar triangles.

We see that triangles AMPAMP and AOXAOX have the same angles and so are similar, so

OAMA=OXMP.OAMA=OXMP.
We know that OX=23OEOX=23OE and MP=12OEMP=12OE, so
OAMA=2312=43,OAMA=2312=43,
 so
OA=43MA.OA=43MA.
 But OA+AM=12OCOA+AM=12OC, so OA=47OM=27OCOA=47OM=27OC.

We would also like to find OBOB, perhaps by finding BCBC.

Triangles BCYBCY and BMQBMQ are similar. In fact, we see that MQ=13DC=YCMQ=13DC=YC, because MM and QQ are each at the midpoints of their respective sides of squares, so triangles BCYBCY and BMQBMQ are congruent.

So BC=BM=12CM=14OCBC=BM=12CM=14OC.

Now

ABOC=OC−OA−BCOC=1−27−14=28−8−728=1328.ABOC=OC−OA−BCOC=1−27−14=28−8−728=1328.
