Thinking about Geometry
Review question

Why is this quadrilateral a rhombus but not a square?
Add to your resource collection

Ref: R6641
Printable/supporting materials Fullscreen mode Teacher notes

  • Suggestion
  • Solution

Solution

Draw a diagram showing points A(−3,1)A(−3,1), B(4,2)B(4,2), C(9,−3)C(9,−3), D(2,−4)D(2,−4).

  1. For the figure ABCDABCD state and prove a fact (or facts) sufficient to ensure that it is a parallelogram.

Drawing a diagram is a good idea.

Approach 1

A parallelogram is a quadrilateral where opposite sides have equal lengths, so all we have to show is that AB=CDAB=CD and AD=BCAD=BC.

The length of the sides can be calculated with the use of Pythagoras’ theorem by constructing right triangles between the points. We then find

ABCDADBC=(−3−4)2+(1−2)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(9−2)2+(−3+4)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(−3−2)2+(1+4)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(4−9)2+(2+3)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=50⎯⎯⎯⎯√,=50⎯⎯⎯⎯√,=50⎯⎯⎯⎯√,=50⎯⎯⎯⎯√.AB=(−3−4)2+(1−2)2=50,CD=(9−2)2+(−3+4)2=50,AD=(−3−2)2+(1+4)2=50,BC=(4−9)2+(2+3)2=50.

This certainly satisfies AB=CDAB=CD and AD=BCAD=BC. The figure therefore is a parallelogram.

Approach 2

A quadrilateral is a parallelogram if and only if its diagonals bisect each other. Midpoint(ACAC) = (3,−13,−1) = Midpoint (BDBD), so ABCDABCD must be a parallelogram.

  1. State and prove an additional fact sufficient to ensure that ABCDABCD is a rhombus.

A rhombus is a quadrilateral with four equal sides. We’ve already calculated all four side lengths, and they’re equal, so ABCDABCD must be a rhombus.

  1. Prove that ABCDABCD is not a square.

Approach 1

If we can prove that any of the angles inside the figure is not a right angle, then this would show that ABCDABCD isn’t a square.

The angle at CC is a right angle if and only if AC2=AD2+CD2AC2=AD2+CD2. In fact, we find

AC2AD2+CD2=(−3−9)2+(1+3)2=50+50=160,=100.AC2=(−3−9)2+(1+3)2=160,AD2+CD2=50+50=100.

The figure is therefore not a square.

Approach 2

A square is a rhombus where diagonals have equal lengths. So all we have to consider is whether AC=BDAC=BD. A short calculation reveals

ACBD=(−3−9)2+(1+3)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=(4−2)2+(2+4)2⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√=160⎯⎯⎯⎯⎯⎯√,=40⎯⎯⎯⎯√.AC=(−3−9)2+(1+3)2=160,BD=(4−2)2+(2+4)2=40.

Once again, we see that ABCDABCD is not a square.

Approach 3

Gradient(ABAB) = 1/71/7, Gradient(BCBC) = −1−1, so their product is not −1−1. Thus the angle at BB is not a right angle, and ABCDABCD is not a square.
