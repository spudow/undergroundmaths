Things you might have done

Without using a calculator, find a reasonable estimate of:

1÷2⎯⎯√1÷2

There are several approaches that could have been used, some of which are discussed below.

1÷2⎯⎯√1÷2 can also be written as 12√12 or 12⎯⎯⎯√.12.

If we call our answer aa, then 12⎯⎯⎯√=a12=a, and 12=a212=a2, so we are looking for a number that squares to 0.50.5. Writing down some decimal versions of square numbers gives us 0.72=0.490.72=0.49 and 0.82=0.640.82=0.64, so we see the answer must be close to 0.70.7. Calculating 0.7120.712 gives us 0.50510.5051, so 12⎯⎯⎯√≈0.71.12≈0.71.

As 1⎯⎯√<2⎯⎯√<4⎯⎯√1<2<4 we know that 2⎯⎯√2 must lie between 11 and 22. Using some square numbers will help in the estimation process as we can write down 1.32=1.691.32=1.69, 1.42=1.961.42=1.96 and 1.52=2.251.52=2.25 showing us that 2⎯⎯√2 must be just slightly bigger than 1.41.4.

If we have an approximate value for 2⎯⎯√2 then we can use it in a division. If we use 2⎯⎯√≈1.42≈1.4 then we can write the approximation as a fraction to use in the division.

12⎯⎯√≈175=5712≈175=57

and some short division gives us 1÷2⎯⎯√≈0.714.1÷2≈0.714.

Our approximation of 2⎯⎯√2 is accurate to 11 decimal place. How accurate do you think the answer of 0.7140.714 will be?

To improve our answer we would need to know a better approximation for 2⎯⎯√2. If we use 2⎯⎯√≈1.4142≈1.414 we could write

12⎯⎯√≈11.414=114141000=10001414.12≈11.414=114141000=10001414.

Although our approximations improve using this method, the division we have to calculate will become harder.

Remember 2⎯⎯√2 is an irrational number so cannot be written as a fraction. Using fractions to approximate the value of 2⎯⎯√2 more accurately, will require larger numbers in the numerators and denominators.

We should see from the method above that we can’t rewrite 2⎯⎯√2 in a way that gives us a simple number to divide by. What can we do to the whole fraction of 12√12 so the denominator becomes an easier number to divide by?

We need to keep the division the same, so we are looking for an equivalent fraction. If we multiply by 2√2√22 then we will get a rational number in the denominator.

12⎯⎯√×2⎯⎯√2⎯⎯√=2⎯⎯√212×22=22

In this form we can see that our division becomes much simpler, and with an approximation of 1.4141.414 for 2⎯⎯√2 we could quickly get an answer of 0.7070.707.

We have done what we call ‘rationalising the denominator’ where any fractions that involve surds in the denominator are rewritten so that the denominator only involves rational numbers. This process makes it easier to approximate the size of numbers as we have just seen, as well as being the standard way that answers are often written in mathematics.

We can rationalise algebraic expressions as well, and this can be a helpful process to give you simpler functions or equations to work with.

What can your thinking about the original problem tell you about other divisions of the form 1÷n⎯⎯√1÷n?

  • Can you select nn such that you can write down the value of the division calculation more easily?

  • Can you select nn such that your answer to the original problem is useful in finding the value of the division?

If we think about the different divisions we would get if we took nn to be a value between 11 and 99 we see there will be some simple calculations when n=1,4n=1,4 and 99, i.e. when nn is a square number. We have left n=3,5,6,7n=3,5,6,7 and 88. We could estimate these as we did with 2⎯⎯√2 above, but if we look at 8⎯⎯√8, we might recognise that it can be written as 4×2⎯⎯⎯⎯⎯⎯⎯⎯√4×2 or 22⎯⎯√22, and then we could use our original answer to help us.

Without using a calculator, find a reasonable estimate of:

50⎯⎯⎯⎯√÷1050÷10

Using square numbers, we know 49⎯⎯⎯⎯√=749=7, so 50⎯⎯⎯⎯√≈7.1.50≈7.1. We could improve this estimate, but already we can see that 50⎯⎯⎯⎯√÷10≈0.71.50÷10≈0.71.

Compare this to your original problem: 1÷2⎯⎯√1÷2.

  • What do you notice about your approach?

  • What do you notice about your answers?

50⎯⎯⎯⎯√÷1050÷10 was a simple calculation to do, as the square root was relatively easy to approximate, and the divisor was a whole number, i.e. this fraction is already in a rationalised form.

You might have also noticed that your answer is very similar, or the same, as your answer for 1÷2⎯⎯√.1÷2. Can you see why this is? What happens if you simplify 50√105010?
