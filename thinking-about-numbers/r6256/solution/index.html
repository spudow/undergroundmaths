Thinking about Numbers
Review question

Which of these numbers does not have a square root of the form x+y2⎯⎯√x+y2?
Add to your resource collection

Ref: R6256

  • Solution

Solution

Which of the following numbers does not have a square root in the form x+y2⎯⎯√x+y2, where xx and yy are positive integers?

  1. 17+122⎯⎯√17+122
  2. 22+122⎯⎯√22+122
  3. 38+122⎯⎯√38+122
  4. 54+122⎯⎯√54+122
  5. 73+122⎯⎯√73+122

Let’s suppose x+y2⎯⎯√x+y2 is the square root of zz.

So z2=(x+y2⎯⎯√)2=(x2+2y2)+(2xy)2⎯⎯√z2=(x+y2)2=(x2+2y2)+(2xy)2.

We need 2xy=122xy=12, and so the only possible pairs (x,y)(x,y) for this to be true are (1,6),(6,1),(2,3)(1,6),(6,1),(2,3) and (3,2)(3,2).

  • (x,y)=(1,6)⟹z2=73+122⎯⎯√(x,y)=(1,6)⟹z2=73+122

  • (x,y)=(6,1)⟹z2=38+122⎯⎯√(x,y)=(6,1)⟹z2=38+122

  • (x,y)=(2,3)⟹z2=22+122⎯⎯√(x,y)=(2,3)⟹z2=22+122

  • (x,y)=(3,2)⟹z2=17+122⎯⎯√(x,y)=(3,2)⟹z2=17+122

Thus the odd one out is 54+122⎯⎯√54+122, and the answer is (D).
