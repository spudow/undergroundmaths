Solution

Evaluate the sum

11⎯⎯√+2⎯⎯√+12⎯⎯√+3⎯⎯√+13⎯⎯√+4⎯⎯√+⋯+115⎯⎯⎯⎯√+16⎯⎯⎯⎯√.11+2+12+3+13+4+⋯+115+16.

(You might want to use a calculator to get an estimate of the answer, but in order to get the exact answer you will have to do it by hand!)

We can think of 3−23−2 as a difference of two squares, since 3−2=(3⎯⎯√)2−(2⎯⎯√)23−2=(3)2−(2)2.

Let the sum we’re interested in be SS, so

S=11⎯⎯√+2⎯⎯√+12⎯⎯√+3⎯⎯√+13⎯⎯√+4⎯⎯√+⋯+115⎯⎯⎯⎯√+16⎯⎯⎯⎯√.S=11+2+12+3+13+4+⋯+115+16.

Then

S=2⎯⎯√−1⎯⎯√2−1+3⎯⎯√−2⎯⎯√3−2+4⎯⎯√−3⎯⎯√4−3+⋯+16⎯⎯⎯⎯√−15⎯⎯⎯⎯√16−15.S=2−12−1+3−23−2+4−34−3+⋯+16−1516−15.

We can simplify the denominators very nicely, to get

S=(−1⎯⎯√+2⎯⎯√)+(−2⎯⎯√+3⎯⎯√)+(−3⎯⎯√+4⎯⎯√)+⋯+(−15⎯⎯⎯⎯√+16⎯⎯⎯⎯√).S=(−1+2)+(−2+3)+(−3+4)+⋯+(−15+16).

The sums on the right-hand side now telescope (lots of adjacent terms cancel), so we are just left with the first and last terms.

So S=16⎯⎯⎯⎯√−1⎯⎯√=4−1=3S=16−1=4−1=3.

Can you find a similar sum that evaluates to 55?

We could construct a similar expression with sum 55 by adding on extra terms up to 135√+36√135+36.

There are other possibilities too: if we construct a similar sum that evaluates to 88, then the difference between this and SS will be 33. So we can do something similar even if we don’t start with 11√+2√11+2.

Can you find a similar sum that evaluates to a number that is not an integer?

We can construct many similar expressions that don’t evaluate to integers. One example would be T=S−115√+16√T=S−115+16 (so just miss out the last term of SS). If we rationalise the denominators then we still get a telescoping sum, but this time T=15⎯⎯⎯⎯√−1⎯⎯√=15⎯⎯⎯⎯√−1T=15−1=15−1, which is not an integer.
