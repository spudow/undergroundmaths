Equation or identity? (II)

Solution

Throughout, AA, BB and CC are the angles of a triangle.

For each of the following, decide whether it is an identity (true for all triangles) or an equation (there is a triangle for which it is not true).

If it is an identity, true for all triangles, then you should prove it (using trigonometric identities that you already know).

If it is an equation, then at the very least you should give an example of a triangle for which it is not true. You could also try to solve the equation (that is, find all triangles for which it is true).

  1. sin(A+2B)=sinA+2sinBcos(A+B)sin⁡(A+2B)=sin⁡A+2sin⁡Bcos⁡(A+B).

We have to decide whether to try to show that this is true for all triangles (because it’s an identity), or to find the triangles for which it’s true (because it’s an equation). To help us to decide which to go for, we could pick some values to try.

For example, if A=π3A=π3 and B=π6B=π6 (because those lead to values that we can easily compute), the left-hand side is
sin(π3+π3)=sin(2π3)=3⎯⎯√2,sin⁡(π3+π3)=sin⁡(2π3)=32,
 and the right-hand side is
sin(π3)+2sin(π6)cos(π3+π6)=3⎯⎯√2+2×12×cos(π2)=3⎯⎯√2,sin⁡(π3)+2sin⁡(π6)cos⁡(π3+π6)=32+2×12×cos⁡(π2)=32,

so the equation is satisfied by that triangle.

We could try another triangle. If A=B=π4A=B=π4, then the left-hand side is

sin(3π4)=12⎯⎯√,sin⁡(3π4)=12,
and the right-hand side is
sin(π4)+2sin(π4)cos(π2)=12⎯⎯√+2×12⎯⎯√×0=12⎯⎯√,sin⁡(π4)+2sin⁡(π4)cos⁡(π2)=12+2×12×0=12,
 so the equation is satisfied by this triangle too.

Of course, it might be that we’ve somehow managed to pick two examples where it works even though it’s not true for all triangles, but that’s not a big problem. We can try to prove the identity, and see what happens. If we succeed, then of course that’s great. If we don’t, then it may be that is because it’s not an identity after all, but in the course of not managing to come up with a proof we’ll hopefully have learned something about the problem that may enable us to construct a triangle that doesn’t satisfy the equation. The important thing at this stage is that we have to dive in (try to prove it), but just have at the back of our minds the idea that it may turn out not to be true after all.

We start with the left-hand side, which we can rewrite using the compound angle formulae. We have

sin(A+2B)=sinAcos(2B)+cosAsin(2B).sin⁡(A+2B)=sin⁡Acos⁡(2B)+cos⁡Asin⁡(2B).

It is always worth pausing in manipulations like this to decide what might be the most helpful next step. We have a very natural way to expand sin(2B)sin⁡(2B) as 2sinBcosB2sin⁡Bcos⁡B, but we could consider for a moment whether we might want to expand cos(2B)cos⁡(2B) as cos2B−sin2Bcos2⁡B−sin2⁡B or 2cos2B−12cos2⁡B−1 or 1−2sin2B1−2sin2⁡B. In order to get a sinAsin⁡A by itself (looking ahead to the right-hand side, to see what we want to show), we might pick 1−2sin2B1−2sin2⁡B, perhaps.

We have
sin(A+2B)=sinA−2sinAsin2B+2cosAsinBcosB.sin⁡(A+2B)=sin⁡A−2sin⁡Asin2⁡B+2cos⁡Asin⁡Bcos⁡B.
 Now we can collect together the last two terms, to get
sin(A+2B)=sinA+2sinB(−sinAsinB+cosAcosB)=sinA+2sinBcos(A+B),sin⁡(A+2B)=sin⁡A+2sin⁡B(−sin⁡Asin⁡B+cos⁡Acos⁡B)=sin⁡A+2sin⁡Bcos⁡(A+B),

as we had hoped.

So this is indeed an identity, and we have just proved it.

Notice that here it was not relevant that AA and BB were two of the three angles of a triangle: our calculations did not rely on them lying in a particular range of values, for example. So this is true for any AA and BB.

  1. tan(A−B)+tan(B−C)+tan(C−A)=0tan⁡(A−B)+tan⁡(B−C)+tan⁡(C−A)=0.
Let’s try this for a triangle. One way to make the left-hand side easy to evaluate would be to pick a triangle where say A=BA=B. Then we have
tan(A−B)+tan(B−C)+tan(C−A)=tan0+tan(A−C)+tan(C−A)=tan(A−C)−tan(A−C)=0.tan⁡(A−B)+tan⁡(B−C)+tan⁡(C−A)=tan⁡0+tan⁡(A−C)+tan⁡(C−A)=tan⁡(A−C)−tan⁡(A−C)=0.

So the equation holds for isosceles triangles.

Let’s try a triangle that is not isosceles. If A=π3A=π3 and B=π6B=π6, then C=π−π3−π6=π2C=π−π3−π6=π2 and so
tan(A−B)+tan(B−C)+tan(C−A)=tan(π6)+tan(−π3)+tan(π6)=13⎯⎯√−3⎯⎯√+13⎯⎯√,tan⁡(A−B)+tan⁡(B−C)+tan⁡(C−A)=tan⁡(π6)+tan⁡(−π3)+tan⁡(π6)=13−3+13,

and this is not 00.

So we see that the equation is not satisfied by this triangle, but is satisfied by all isosceles triangle. Now we need to try to find all triangles for which the equation holds.

We have A−B=(A−C)+(C−B)A−B=(A−C)+(C−B), and so
tan(A−B)+tan(B−C)+tan(C−A)=tan((A−C)+(C−B))+tan(B−C)+tan(C−A)=tan(A−C)+tan(C−B)1−tan(A−C)tan(C−B)+tan(B−C)+tan(C−A).tan⁡(A−B)+tan⁡(B−C)+tan⁡(C−A)=tan⁡((A−C)+(C−B))+tan⁡(B−C)+tan⁡(C−A)=tan⁡(A−C)+tan⁡(C−B)1−tan⁡(A−C)tan⁡(C−B)+tan⁡(B−C)+tan⁡(C−A).

Since all we need to know is whether this is 00, we can put it over a single denominator, which we can then disregard, and save some algebra.

This is 00 if and only if
0=tan(A−C)+tan(C−B)−tan(C−B)−tan(A−C)+[tan(A−C)+tan(C−B)]tan(A−C)tan(C−B)=tan(A−C)tan(C−B)[tan(A−C)+tan(C−B)].0=tan⁡(A−C)+tan⁡(C−B)−tan⁡(C−B)−tan⁡(A−C)+[tan⁡(A−C)+tan⁡(C−B)]tan⁡(A−C)tan⁡(C−B)=tan⁡(A−C)tan⁡(C−B)[tan⁡(A−C)+tan⁡(C−B)].

So we see that tan(A−C)=0tan⁡(A−C)=0, or tan(C−B)=0tan⁡(C−B)=0, or tan(A−C)=−tan(C−B)tan⁡(A−C)=−tan⁡(C−B).

Since AA, BB and CC are angles of a triangle, we are restricted in the range that they can take, and we find that A=CA=C, or C=BC=B, or A−C=−C+BA−C=−C+B. This last possibility is equivalent to A=BA=B.

We have established that the equation holds if and only if the triangle is isosceles.

Note that it was relatively straightforward to show that if the triangle is isosceles then its angles satisfy the equation—that was the first thing we did. The bit that required more thought was showing that this is the only possibility: that if the equation is satisfied, then the triangle must be isosceles.

  1. 2sinAcos2(B2)+2cos2(A2)sinB=sin(A+B)+sin(B+C)+sin(C+A)2sin⁡Acos2⁡(B2)+2cos2⁡(A2)sin⁡B=sin⁡(A+B)+sin⁡(B+C)+sin⁡(C+A).

Let’s try expanding the right-hand side, using compound angle formulae, and hopefully either we’ll get closer to the left-hand side or we’ll learn something about which triangles do or don’t satisfy the equation.

We might notice that the left-hand side does not involve CC, so we could try to eliminate that.

We have
sin(A+B)+sin(B+C)+sin(C+A)=sin(A+B)+sin(B+π−A−B)+sin(π−A−B+A)=sin(A+B)+sin(π−A)+sin(π−B)=sinAcosB+cosAsinB+sinA+sinB=sinA(1+cosB)+sinB(1+cosA).sin⁡(A+B)+sin⁡(B+C)+sin⁡(C+A)=sin⁡(A+B)+sin⁡(B+π−A−B)+sin⁡(π−A−B+A)=sin⁡(A+B)+sin⁡(π−A)+sin⁡(π−B)=sin⁡Acos⁡B+cos⁡Asin⁡B+sin⁡A+sin⁡B=sin⁡A(1+cos⁡B)+sin⁡B(1+cos⁡A).

Here we have collected terms inspired by the left-hand side.

We now want to consider 1+cosB1+cos⁡B, in the hope that it may turn out to be cos2(B2)cos2⁡(B2).

Now
cosB=cos(2×B2)=2cos2(B2)−1,cos⁡B=cos⁡(2×B2)=2cos2⁡(B2)−1,
 and similarly for AA, and so
sin(A+B)+sin(B+C)+sin(C+A)=sinA(2cos2(B2))+sinB(2cos2(A2)),sin⁡(A+B)+sin⁡(B+C)+sin⁡(C+A)=sin⁡A(2cos2⁡(B2))+sin⁡B(2cos2⁡(A2)),

and so we have proved the identity.

  1. sin(A+B)=cosCsin⁡(A+B)=cos⁡C.

We might notice that A+B=π−CA+B=π−C, so this statement starts to look like a more familiar statement. Let’s dive in and try to prove it (and if it turns out that we’re wrong then that will become clear because our proof won’t work).

Since AA, BB and CC are the angles of a triangle, we see that
sin(A+B)=sin(π−C)=sinπcosC−cosπsinC=sinC.sin⁡(A+B)=sin⁡(π−C)=sin⁡πcos⁡C−cos⁡πsin⁡C=sin⁡C.

Oh dear, that’s not quite what we wanted. But this effort isn’t wasted, we can just try to solve the equation using this extra information.

So the triangle satisfies the equation if and only if sinC=cosCsin⁡C=cos⁡C. We cannot have cosC=0cos⁡C=0 (because we cannot have both cosC=0cos⁡C=0 and sinC=0sin⁡C=0 simultaneously), so we can divide through: the triangle satisfies the equation if and only if tanC=1tan⁡C=1.

Since CC is an angle of a triangle, we must have 0<C<π0<C<π, so we see that the only solution is C=π4C=π4.

So the triangle satisfies the equation if and only if one angle is π4π4.

  1. cosC=−cos(A+B)cos⁡C=−cos⁡(A+B).

Sometimes it is a good idea to start with the right-hand side rather than the left-hand side. Let’s try that here, manipulating it to see whether we can make it look more like the left-hand side (either to prove the identity, or to get more information that will help us to solve the equation).

We have
−cos(A+B)=−cos(π−C)=sinπsinC−cosπcosC=cosC,−cos⁡(A+B)=−cos⁡(π−C)=sin⁡πsin⁡C−cos⁡πcos⁡C=cos⁡C,

and so the identity is true for all triangles.

  1. 4(cos2Acos2B+sin2Asin2B)−2sin(2A)sin(2B)=34(cos2⁡Acos2⁡B+sin2⁡Asin2⁡B)−2sin⁡(2A)sin⁡(2B)=3.

It’s hard to know what to make of this one, so let’s try it for a triangle or two to try to get a feel for it.

If A=π3A=π3 and B=π6B=π6, then
4(cos2Acos2B+sin2Asin2B)−2sin(2A)sin(2B)=4(14×34+34×14)−2×3⎯⎯√2×3⎯⎯√2=32−32=0.4(cos2⁡Acos2⁡B+sin2⁡Asin2⁡B)−2sin⁡(2A)sin⁡(2B)=4(14×34+34×14)−2×32×32=32−32=0.

So we have found a triangle for which the equation does not hold, and we know that this is an equation that we should try to solve, rather than an identity that we need to prove.

We have little choice but to think about the left-hand side.

We have
4(cos2Acos2B+sin2A+sin2B)−2sin(2A)sin(2B) =4(cos2Acos2B+sin2A+sin2B)−8sinAcosAsinBcosB.4(cos2⁡Acos2⁡B+sin2⁡A+sin2⁡B)−2sin⁡(2A)sin⁡(2B) =4(cos2⁡Acos2⁡B+sin2⁡A+sin2⁡B)−8sin⁡Acos⁡Asin⁡Bcos⁡B.

We might think of reordering the terms in the final product: we could think of it as sinAsinBcosAcosBsin⁡Asin⁡Bcos⁡Acos⁡B, and it all starts to look a bit as though it involves things like (cosAcosB−sinAsinB)2=cos2(A+B)(cos⁡Acos⁡B−sin⁡Asin⁡B)2=cos2⁡(A+B).

We have
cos2(A+B)=(cosAcosB−sinAsinB)2=cos2Acos2B+sin2Asin2B−2sinAsinBcosAcosB,cos2⁡(A+B)=(cos⁡Acos⁡B−sin⁡Asin⁡B)2=cos2⁡Acos2⁡B+sin2⁡Asin2⁡B−2sin⁡Asin⁡Bcos⁡Acos⁡B,

and so our original equation is equivalent to

4cos2(A+B)=3.4cos2⁡(A+B)=3.
We see that this is equivalent to
cos(A+B)=±3⎯⎯√2.cos⁡(A+B)=±32.

Since AA and BB are angles of a triangle, we know that 0<A+B<π0<A+B<π, and so the possible solutions are

A+B=π6A+B=π6
and
A+B=5π6.A+B=5π6.

This tells us that the angles of a triangle satisfy the original equation if and only if one angle is π6π6 or 5π65π6.

  1. sin(2A)+sin(2B)+sin(2C)=4sinAsinBsinCsin⁡(2A)+sin⁡(2B)+sin⁡(2C)=4sin⁡Asin⁡Bsin⁡C.

If A=π3A=π3 and B=π6B=π6, then C=π2C=π2 and so

sin(2A)+sin(2B)+sin(2C)=3⎯⎯√2+3⎯⎯√2=3⎯⎯√,sin⁡(2A)+sin⁡(2B)+sin⁡(2C)=32+32=3,
while
4sinAsinBsinC=4×3⎯⎯√2×12×1=3⎯⎯√,4sin⁡Asin⁡Bsin⁡C=4×32×12×1=3,
 so the equation holds for this triangle.

If A=B=π4A=B=π4, so C=π2C=π2, then the left-hand side is

sin(2A)+sin(2B)+sin(2C)=1+1+0=2,sin⁡(2A)+sin⁡(2B)+sin⁡(2C)=1+1+0=2,
and the right-hand side is
4sinAsinBsinC=4×12⎯⎯√×12⎯⎯√×1=2,4sin⁡Asin⁡Bsin⁡C=4×12×12×1=2,
 so the equation holds for this triangle too.

Prompted by these two examples, let’s see whether we can prove the identity in general. It seems slightly more natural to expand the left-hand side, so let’s do that and see whether we can make it look more like the right-hand side. We want to keep sines, but if we get a cosine then we should try to eliminate it, perhaps using the fact that the angles sum to ππ.

We have
sin(2A)+sin(2B)+sin(2C)=2sinAcosA+2sinBcosB+2sinCcos(π−(A+B))=2sinAcosA+2sinBcosB−2sinCcos(A+B)=2sinAcos(π−(B+C))+2sinBcos(π−(A+C))−2sinCcosAcosB+2sinCsinAsinB=2sinA(sinBsinC−cosBcosC)+2sinB(sinAsinC−cosAcosC)−2cosAcosBsinC+2sinAsinBsinC=6sinAsinBsinC−2sinAcosBcosC−2cosAsinBcosC−2cosAcosBsinC=6sinAsinBsinC−2cosCsin(A+B)−2cosAcosBsinC=6sinAsinBsinC−2cosCsinC−2cosAcosBsinC=6sinAsinBsinC−2sinC(−cos(A+B)+cosAcosB)=6sinAsinBsinC−2sinC(sinAsinB)=4sinAsinBsinC.sin⁡(2A)+sin⁡(2B)+sin⁡(2C)=2sin⁡Acos⁡A+2sin⁡Bcos⁡B+2sin⁡Ccos⁡(π−(A+B))=2sin⁡Acos⁡A+2sin⁡Bcos⁡B−2sin⁡Ccos⁡(A+B)=2sin⁡Acos⁡(π−(B+C))+2sin⁡Bcos⁡(π−(A+C))−2sin⁡Ccos⁡Acos⁡B+2sin⁡Csin⁡Asin⁡B=2sin⁡A(sin⁡Bsin⁡C−cos⁡Bcos⁡C)+2sin⁡B(sin⁡Asin⁡C−cos⁡Acos⁡C)−2cos⁡Acos⁡Bsin⁡C+2sin⁡Asin⁡Bsin⁡C=6sin⁡Asin⁡Bsin⁡C−2sin⁡Acos⁡Bcos⁡C−2cos⁡Asin⁡Bcos⁡C−2cos⁡Acos⁡Bsin⁡C=6sin⁡Asin⁡Bsin⁡C−2cos⁡Csin⁡(A+B)−2cos⁡Acos⁡Bsin⁡C=6sin⁡Asin⁡Bsin⁡C−2cos⁡Csin⁡C−2cos⁡Acos⁡Bsin⁡C=6sin⁡Asin⁡Bsin⁡C−2sin⁡C(−cos⁡(A+B)+cos⁡Acos⁡B)=6sin⁡Asin⁡Bsin⁡C−2sin⁡C(sin⁡Asin⁡B)=4sin⁡Asin⁡Bsin⁡C.

So we have proved the identity.
