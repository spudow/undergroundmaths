Trigonometry: Compound Angles
Review question

Can we write cosθ−2cos3θ+cos5θcos⁡θ−2cos⁡3θ+cos⁡5θ using sinsin?

Solution

  1. Prove the identity
    cosθ−2cos3θ+cos5θ=2sinθ(sin2θ−sin4θ).cos⁡θ−2cos⁡3θ+cos⁡5θ=2sin⁡θ(sin⁡2θ−sin⁡4θ).

We can always write the expressions cosA+cosB,cosA−cosB,sinA+sinBcos⁡A+cos⁡B,cos⁡A−cos⁡B,sin⁡A+sin⁡B, and sinA−sinBsin⁡A−sin⁡B as the product of two sin/cossin⁡/cos terms.

This can be immensely useful to us in lots of situations. Here we are interested in cosA−cosBcos⁡A−cos⁡B.

We begin with

cos(X+Y)=cosXcosY−sinXsinYandcos(X−Y)=cosXcosY+sinXsinY.cos⁡(X+Y)=cos⁡Xcos⁡Y−sin⁡Xsin⁡Yandcos⁡(X−Y)=cos⁡Xcos⁡Y+sin⁡Xsin⁡Y.
Adding these equations gives
cos(X−Y)−cos(X+Y)=2sinXsinY.cos⁡(X−Y)−cos⁡(X+Y)=2sin⁡Xsin⁡Y.
 Now setting A=X−Y,B=X+YA=X−Y,B=X+Y we find
cosA−cosB=2sinA+B2sinB−A2.cos⁡A−cos⁡B=2sin⁡A+B2sin⁡B−A2.

Applying this to both cosθ−cos3θcos⁡θ−cos⁡3θ and cos5θ−cos3θcos⁡5θ−cos⁡3θ, we see the left-hand side (LHS) is equal to

2sin2θsinθ+2sin4θsin(−2θ)=2sinθ(sin2θ−sin4θ),2sin⁡2θsin⁡θ+2sin⁡4θsin⁡(−2θ)=2sin⁡θ(sin⁡2θ−sin⁡4θ),
as required.

Alternatively, we can show by expanding that if c=cosθ,s=sinθc=cos⁡θ,s=sin⁡θ, then

cos(3θ)=4c3−3c,cos(5θ)=16c5−20c3+5c,sin(2θ)=2sc,sin(4θ)=8sc3−4sc.cos⁡(3θ)=4c3−3c,cos⁡(5θ)=16c5−20c3+5c,sin⁡(2θ)=2sc,sin⁡(4θ)=8sc3−4sc.

Now the LHS is

c−2(4c3−3c)+16c5−20c3+5c=16c5−28c3+12c,c−2(4c3−3c)+16c5−20c3+5c=16c5−28c3+12c,
while the RHS is
2s(2sc−(8sc3−4sc))=2(1−c2)(−8c3+6c)=16c5−28c3+12c,2s(2sc−(8sc3−4sc))=2(1−c2)(−8c3+6c)=16c5−28c3+12c,
 which are the same, and so we are done.

    1. cos2x=sinxcos⁡2x=sin⁡x,

    2. 3sec2x=tanx+53sec2⁡x=tan⁡x+5

    giving in each case all solutions between 0∘0∘ and 360∘360∘.

  1. For the first equation, we use one of the double angle formulae for coscos, that’s cos2x=1−2sin2xcos⁡2x=1−2sin2⁡x.

Now everything’s written in terms of sinxsin⁡x, which makes the equation much simpler.

Writing u=sinxu=sin⁡x, we have to solve the equation cos2x=sinx,cos⁡2x=sin⁡x, which gives

1−2u2=u, or2u2+u−1=0, or(2u−1)(u+1)=0,1−2u2=u, or2u2+u−1=0, or(2u−1)(u+1)=0,

with solutions
sinx=u=12,−1.sin⁡x=u=12,−1.

We need all solutions between 0∘0∘ and 360∘360∘. These are

x=30∘,150∘,270∘.x=30∘,150∘,270∘.

It’s helpful to sketch a graph, to check we’ve got all the solutions.

  1. For the second, recall the identity tan2x+1=sec2xtan2⁡x+1=sec2⁡x (given by dividing sin2x+cos2x=1sin2⁡x+cos2⁡x=1 by cos2xcos2⁡x).

Define v=tanxv=tan⁡x, so we need to solve

3sec2x=tanx+5, or3(1+v2)−v−5=0, or3v2−v−2=0.3sec2⁡x=tan⁡x+5, or3(1+v2)−v−5=0, or3v2−v−2=0.
This is (3v+2)(v−1)=0(3v+2)(v−1)=0, so v=−23,1v=−23,1, which (since tanxtan⁡x has period ππ) gives the four solutions
x=45∘,225∘,146.3∘,326.3∘.x=45∘,225∘,146.3∘,326.3∘.
