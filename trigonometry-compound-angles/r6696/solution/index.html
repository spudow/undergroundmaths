Trigonometry: Compound Angles
Review question

Can we write sinθsin⁡θ and cosθcos⁡θ in terms of tan(θ/2)tan⁡(θ/2)?

Solution

Prove that, if t=tan12θt=tan⁡12θ, then

sinθ=2t1+t2...sin⁡θ=2t1+t2...

Approach 1: Starting with the left hand side

We can rewrite sinθsin⁡θ by using the double angle formula:

sinθ=sin(2(12θ))=2sin12θcos12θ.sin⁡θ=sin⁡(2(12θ))=2sin⁡12θcos⁡12θ.

In order to get tt (which we want in the numerator) into the identity, we can rewrite

sin12θ=tan12θcos12θ,sin⁡12θ=tan⁡12θcos⁡12θ,
which is tcos12θtcos⁡12θ (where we have used t=tan12θt=tan⁡12θ). Therefore,
sinθ=2tcos212θ.sin⁡θ=2tcos2⁡12θ.

Since we want tt in the denominator as well, we write

cos212θ=1sec212θ.cos2⁡12θ=1sec2⁡12θ.
Using the identity
1+tan212θ=sec212θ,1+tan2⁡12θ=sec2⁡12θ,
 we deduce that
cos212θ=11+tan212θ=11+t2.cos2⁡12θ=11+tan2⁡12θ=11+t2.
 Therefore
sinθ=2t1+t2.sin⁡θ=2t1+t2.

Approach 2: Starting with the right hand side

We will rewrite the right hand side in terms of sin12θsin⁡12θ and cos12θcos⁡12θ. We have
2t1+t2=2sin12θcos12θ1+sin212θcos212θ=2sin12θcos12θcos212θ+sin212θ,2t1+t2=2sin⁡12θcos⁡12θ1+sin2⁡12θcos2⁡12θ=2sin⁡12θcos⁡12θcos2⁡12θ+sin2⁡12θ,

where we reached the second line by multiplying the numerator and denominator of the large fraction by cos212θcos2⁡12θ.

Since sin212θ+cos212θ=1sin2⁡12θ+cos2⁡12θ=1, the denominator is 11 and hence

2t1+t2=2sin12θcos12θ=sinθ,2t1+t2=2sin⁡12θcos⁡12θ=sin⁡θ,
as required.

...andcosθ=1−t21+t2....andcos⁡θ=1−t21+t2.

As in Approach 1, we write tanθ=tan(2(12θ))tan⁡θ=tan⁡(2(12θ)), and use the double-angle formula for tantan to get

tanθ=2tan12θ1−tan212θ=2t1−t2.tan⁡θ=2tan⁡12θ1−tan2⁡12θ=2t1−t2.

We also know that tanθ=sinθcosθtan⁡θ=sin⁡θcos⁡θ, so that cosθ=sinθtanθcos⁡θ=sin⁡θtan⁡θ.

Therefore, using our expressions for sinθsin⁡θ and tanθtan⁡θ in terms of tt, we deduce

cosθ=(2t1+t2)(2t1−t2)=1−t21+t2.cos⁡θ=(2t1+t2)(2t1−t2)=1−t21+t2.

We can also prove that cosθ=1−t21+t2cos⁡θ=1−t21+t2 using Approaches 1 and 2 above.

By expressing 3+cosθsinθ3+cos⁡θsin⁡θ in terms of tt, or otherwise, show that this expression cannot have any value between −22⎯⎯√−22 and +22⎯⎯√+22.

Method 1

We have
3+cosθsinθ=3+1−t21+t22t1+t2=3(1+t2)+1−t22t=4+2t22t=2+t2t.3+cos⁡θsin⁡θ=3+1−t21+t22t1+t2=3(1+t2)+1−t22t=4+2t22t=2+t2t.

We need to show therefore that 2+t2t=t+2t2+t2t=t+2t cannot have any value in the given range, where tt is any real number except for zero.

The last part is because t=tan12θt=tan⁡12θ can take any real value, but we cannot allow t=0t=0 in this fraction, just as we cannot allow sinθ=0sin⁡θ=0 in the original fraction. We have

sinθ≠0⟹tan12θ≠0.sin⁡θ≠0⟹tan⁡12θ≠0.

Let t+2t=kt+2t=k. Then t2−kt+2=0t2−kt+2=0, and for this to have real roots, the discriminant must be non-negative.

This means that k2≥8k2≥8, and so k≥22⎯⎯√k≥22, or k≤−22⎯⎯√k≤−22, as required.

Method 2

If we sketch a graph of y=t+2ty=t+2t, we see that it has an asymptote at t=0t=0, is positive for t>0t>0 and negative for t<0t<0.

For t>0t>0, it tends to ∞∞ as t→∞t→∞ and as t→0t→0; for t<0t<0, it tends to −∞−∞ as t→−∞t→−∞ and as t→0t→0. So the graph looks something like this:

We can find the stationary points using calculus.

Writing y=t+2ty=t+2t, we have dydt=1−2t2dydt=1−2t2, sodydt=0dydt=0 has solutions t=±2⎯⎯√t=±2.

When t=2⎯⎯√t=2, y=2⎯⎯√+22⎯⎯√=22⎯⎯√y=2+22=22; whent=−2⎯⎯√t=−2, y=−22⎯⎯√y=−22.

So referring back to our sketch graph, we see that yy cannot take anyvalues between −22⎯⎯√−22 and +22⎯⎯√+22.

Method 3

Another calculus-free approach is to use the arithmetic mean–geometric mean (AM–GM) inequality.

This says that if a1a1 and a2a2 are positive real numbers, then their arithmetic mean is greater than or equal to their geometric mean.

In other words, a1+a22≥a1a2⎯⎯⎯⎯⎯⎯⎯√a1+a22≥a1a2 where a1,a2≥0a1,a2≥0, with equality if and only if a1=a2a1=a2.

In our case, if we take n=2n=2 and set a1=ta1=t and a2=2ta2=2t, we find that

t+2t2≥t×2t⎯⎯⎯⎯⎯⎯⎯⎯⎯√=2⎯⎯√,t+2t2≥t×2t=2,
with equality if and only if t=2tt=2t.

Thus for positive values of tt, t+2t≥22⎯⎯√t+2t≥22 with equality if and only if t=2tt=2t, that is, if t=2⎯⎯√t=2.

For negative values of tt, let t=−ut=−u, with uu positive. Then t+2t=−u−2u=−(u+2u)t+2t=−u−2u=−(u+2u).

Since u+2u≥22⎯⎯√u+2u≥22, it follows that for negative values of tt, t+2t≤−22⎯⎯√t+2t≤−22 with equality if and only if t=−2⎯⎯√t=−2.

Thus t+2tt+2t cannot take any values between −22⎯⎯√−22 and+22⎯⎯√+22.

Method 4

As a final method, if tt is positive, we can write t+2tt+2t as

t+2t=(t√−2t⎯⎯⎯⎯√)2+22⎯⎯√.t+2t=(t−2t)2+22.
Since a square is always positive or zero, the right hand side must be at least 22⎯⎯√22, with equality if and only ift√=2t⎯⎯⎯⎯√t=2t.

We can deal with the case of tt negative as above.

What could make us think to write t+2tt+2t in such a bizarre way? And what made us think that it would be useful?

To prove the AM–GM inequality for n=2n=2, we first observe that

(x−y)2=x2+y2−2xy,(x−y)2=x2+y2−2xy,
so that
x2+y2=(x−y)2+2xy≥2xyx2+y2=(x−y)2+2xy≥2xy
 with equality if and only if x=yx=y.

If we now write x=0a1⎯⎯⎯⎯√x=0a1 andy=0a2⎯⎯⎯⎯√y=0a2, we get

a1+a2≥20a1a2⎯⎯⎯⎯⎯⎯⎯√,a1+a2≥20a1a2,
which is, on dividing by 22, the AM–GM inequality for n=2n=2.

So in this approach, we have simply replicated this proof without mentioning the AM–GM inequality!
