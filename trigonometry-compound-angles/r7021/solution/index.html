Solution

ABCDEABCDE is a regular pentagon. By projecting the broken line AEDAED on the line ABAB, or otherwise, show that

cosπ5−cos2π5=12.cos⁡π5−cos⁡2π5=12.

Method 1

The sum of angles inside a nn-sided polygon is (n−2)π(n−2)π, so each angle inside our regular pentagon is 3π53π5. For simplicity we choose each side to be of length 11.

Let D′D′ and E′E′ be the points DD and EE projected onto the line ABAB as shown. Since ABCDEABCDE is a regular pentagon,
D′A=12.D′A=12.
 From the right-angled triangle AEE′AEE′ we see that
AE′=cos2π5,AE′=cos⁡2π5,
 and from the triangle DPEDPE we find that
D′E′=PE=cosπ5.D′E′=PE=cos⁡π5.
 Since
D′E′=D′A+AE′,D′E′=D′A+AE′,
 we get the result
cosπ5−cos2π5=12.(∗)(∗)cos⁡π5−cos⁡2π5=12.

Method 2

The question says ‘or otherwise’, so we’re free to pick another method if we wish.

The regular pentagon is intimately connected with ϕϕ, the Golden Ratio =1.61803...=1.61803..., which is the positive solution to x2−x−1=0x2−x−1=0.

Using the Cosine Rule in ABDABD one way, cos2π5=12ϕcos⁡2π5=12ϕ. Using the Cosine Rule in ABDABD another way, cosπ5=2ϕ2−12ϕ2cos⁡π5=2ϕ2−12ϕ2.

Thus

cosπ5−cos2π5=2ϕ2−12ϕ2−12ϕ=2ϕ2−12ϕ2−ϕ2ϕ2=2ϕ2−ϕ−12ϕ2=ϕ22ϕ2=12.cos⁡π5−cos⁡2π5=2ϕ2−12ϕ2−12ϕ=2ϕ2−12ϕ2−ϕ2ϕ2=2ϕ2−ϕ−12ϕ2=ϕ22ϕ2=12.

Hence, or otherwise, show that cosπ5=5⎯⎯√+14cos⁡π5=5+14.

Let
γ=cosπ5.γ=cos⁡π5.
 Using the cosine double angle formula
cos2θ=2cos2θ−1cos⁡2θ=2cos2⁡θ−1
 with θ=π5,θ=π5, we can use (∗)(∗) to get
γ−(2γ2−1)⟺4γ2−2γ−1=12=0.γ−(2γ2−1)=12⟺4γ2−2γ−1=0.

We then use the quadratic formula to solve for γγ: we have

γ=cosπ5=2±4−4×4×(−1)⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√2×4=1±5⎯⎯√4.γ=cos⁡π5=2±4−4×4×(−1)2×4=1±54.
We know that π5<π2π5<π2 so cosπ5>0cos⁡π5>0 and thus we choose the positive root:
cosπ5=1+5⎯⎯√4.cos⁡π5=1+54.

Show further that cos3π5=−5⎯⎯√−14cos⁡3π5=−5−14.

We use the fact that

cos(π−θ)=−cosθ,cos⁡(π−θ)=−cos⁡θ,
which we can see from the diagram.

This gives
cos3π5=−cos2π5cos⁡3π5=−cos⁡2π5
 so
cos3π5=−cos2π5=12−cosπ5=2−(5⎯⎯√+1)4=−5⎯⎯√−14cos⁡3π5=−cos⁡2π5=12−cos⁡π5=2−(5+1)4=−5−14

as required.
