Trigonometry: Compound Angles
Review question

Can we solve cosθ−sin(2θ)+cos(3θ)−sin(4θ)=0cos⁡θ−sin⁡(2θ)+cos⁡(3θ)−sin⁡(4θ)=0?

Solution

Give the general solutions of the following equations

  1. 2sin3θ−7cos2θ+sinθ+1=02sin⁡3θ−7cos⁡2θ+sin⁡θ+1=0,

We’ll try to write everything in terms of just sinθsin⁡θ (the single sinθsin⁡θ in the LHS points to this). Let s=sinθs=sin⁡θ, and c=cosθc=cos⁡θ.

We could use the double-angle formula cos2θ=1−2s2.cos⁡2θ=1−2s2. We’d like to expand sin3θsin⁡3θ. Using the angle addition formula,

sin3θ=csin2θ+scos2θ=2sc2+s(1−2s2)=2s(1−s2)+s(1−2s2)=3s−4s3.sin⁡3θ=csin⁡2θ+scos⁡2θ=2sc2+s(1−2s2)=2s(1−s2)+s(1−2s2)=3s−4s3.
 The original equation becomes
2(3s−4s3)−7(1−2s2)+s+1⟹8s3−14s2−7s+6=0=0.2(3s−4s3)−7(1−2s2)+s+1=0⟹8s3−14s2−7s+6=0.

When solving a cubic equation, it is often sensible to try simple numbers like s=±1,±2s=±1,±2 to find the first factor using the Factor Theorem.

If we try small integers for ss, we find that 8s3−14s2−7s+6=08s3−14s2−7s+6=0 when s=2s=2, and so by the Factor Theorem, s−2s−2 is a factor. Thus

8s3−14s2−7s+6=(s−2)(8s2+2s−3)=(s−2)(4s+3)(2s−1),8s3−14s2−7s+6=(s−2)(8s2+2s−3)=(s−2)(4s+3)(2s−1),

which is zero when s=2,−34s=2,−34 or 1212.

We can note that sinθ=2sin⁡θ=2 is impossible.

As the period of sinsin is 2π2π, and sin(θ)=sin(π−θ)sin⁡(θ)=sin⁡(π−θ), the remaining solutions are
θ=θ=θ=θ=sin−1(12)+2πn,π−sin−1(12)+2πn,sin−1(−34)+2πn,π−sin−1(−34)+2πnθ=sin−1⁡(12)+2πn,θ=π−sin−1⁡(12)+2πn,θ=sin−1⁡(−34)+2πn,θ=π−sin−1⁡(−34)+2πn
 where nn is an integer. We can simplify to find
θ=θ=θ=θ=π6+2nπ,5π6+2nπ,−sin−1(34)+2nπ,sin−1(34)+(2n+1)π,θ=π6+2nπ,θ=5π6+2nπ,θ=−sin−1⁡(34)+2nπ,θ=sin−1⁡(34)+(2n+1)π,

where in each case nn is any integer.

  1. cosθ−sin2θ+cos3θ−sin4θ=0cos⁡θ−sin⁡2θ+cos⁡3θ−sin⁡4θ=0.

We might try an approach similar to that above, by writing everything in terms of cosθcos⁡θ and sinθsin⁡θ. It’s probably quicker, however, to use the sum-to-product formulae, which help us to combine two sinsin or two coscos functions.

sinA+sinB=2sin(A+B2)cos(A−B2)sin⁡A+sin⁡B=2sin⁡(A+B2)cos⁡(A−B2)
and
cosA+cosB=2cos(A+B2)cos(A−B2).cos⁡A+cos⁡B=2cos⁡(A+B2)cos⁡(A−B2).

We can derive these simply from
sin(X±Y)cos(X±Y)=sin(X)cos(Y)±cos(X)sin(Y),=cos(X)cos(Y)∓sin(X)sin(Y)sin⁡(X±Y)=sin⁡(X)cos⁡(Y)±cos⁡(X)sin⁡(Y),cos⁡(X±Y)=cos⁡(X)cos⁡(Y)∓sin⁡(X)sin⁡(Y)
Applying these to this part of the question we obtain
cosθ+cos3θ=2cos2θcosθ,sin2θ+sin4θ=2sin3θcosθcos⁡θ+cos⁡3θ=2cos⁡2θcos⁡θ,sin⁡2θ+sin⁡4θ=2sin⁡3θcos⁡θ
 and so we need to solve
2cos2θcosθ−2sin3θcosθ⟺cosθ(cos2θ−sin3θ)=0=0.2cos⁡2θcos⁡θ−2sin⁡3θcos⁡θ=0⟺cos⁡θ(cos⁡2θ−sin⁡3θ)=0.

So our solutions are cosθ=0cos⁡θ=0, which gives θ=π2+nπθ=π2+nπ, or are the solutions to cos2θ=sin3θcos⁡2θ=sin⁡3θ.

Now cos2θ=1−2s2cos⁡2θ=1−2s2, while (as we showed above) sin3θ=3s−4s3sin⁡3θ=3s−4s3, so we need to solve
1−2s2⟹4s3−2s2−3s+1⟹(s−1)(4s2+2s−1)⟹sinθ=1orsinθ=3s−4s3=0=0=−1±5⎯⎯√4.1−2s2=3s−4s3⟹4s3−2s2−3s+1=0⟹(s−1)(4s2+2s−1)=0⟹sin⁡θ=1orsin⁡θ=−1±54.
 So our final solution is
θ=orθ=orθ=orθ=orθ=π2+nπsin−1(−1+5⎯⎯√4)+2nπ,π−sin−1(−1+5⎯⎯√4)+2nπ,sin−1(−1−5⎯⎯√4)+2nπ,π−sin−1(−1−5⎯⎯√4)+2nπ,θ=π2+nπorθ=sin−1⁡(−1+54)+2nπ,orθ=π−sin−1⁡(−1+54)+2nπ,orθ=sin−1⁡(−1−54)+2nπ,orθ=π−sin−1⁡(−1−54)+2nπ,

where nn is an integer.

Alternatively, we could note that

cosα=cosβ⟹α=±β+2nπ.cos⁡α=cos⁡β⟹α=±β+2nπ.

We also have (by considering a right-angled triangle where one angle is γγ) that

sinγ=cos(π2−γ).sin⁡γ=cos⁡(π2−γ).
This is in fact true for all γγ.

Thus the equation cos2θ=sin3θcos⁡2θ=sin⁡3θ becomes
cos2θ⟹2θ⟹θ=cos(π2−3θ)=±(π2−3θ)+2nπ=π2+2nπorπ10+2nπ5,cos⁡2θ=cos⁡(π2−3θ)⟹2θ=±(π2−3θ)+2nπ⟹θ=π2+2nπorπ10+2nπ5,

and so we have the same answers as before, but this time in an explicit form.

Notice we have inadvertently shown here that sin(π10)=−1+5⎯⎯√4sin⁡(π10)=−1+54.
