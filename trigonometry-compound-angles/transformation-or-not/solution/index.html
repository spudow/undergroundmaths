How does this compare with the previous graph?

What function did you use to describe it?

The graphs are identical.

It therefore appears that we can rewrite a function of the form

Asinx+Bcosx,Asin⁡x+Bcos⁡x,
where AA and BB are real numbers, as Rsin(x+α)Rsin⁡(x+α) where R>0R>0 and 0<α<π20<α<π2.

Notice here that we found
5sin(x+α)=3sinx+4cosx,(1)(1)5sin⁡(x+α)=3sin⁡x+4cos⁡x,

with π4<α<3π8π4<α<3π8 from the first problem. The numbers 33, 44 and 55 appear in this equation; does this give you any ideas about a possible relationship between AA, BB and RR?

What happens if we substitute different values of xx into this identity?

Substituting x=0x=0 into equation (1)(1) gives 5sinα=45sin⁡α=4 and thus sinα=45sin⁡α=45.

Below are three graphs of similar functions, namely

y=Asinx+Bcosx,y=Asin⁡x+Bcos⁡x,
where AA and BB are real numbers.

  • Can you work out what AA and BB are for each of these graphs?

  • Can you describe each of them using a single trigonometric function?

The question here is, how can we use the graphs and our knowledge of sine and cosine functions to find the values AA, BB, RR and αα?

What does Asinx+BcosxAsin⁡x+Bcos⁡x tell us when x=0x=0 and when x=π2x=π2?

Graph 1

From the graph we can see that when x=0x=0, Asinx+Bcosx=−1Asin⁡x+Bcos⁡x=−1. This means that B=−1B=−1.

When x=π2x=π2, Asinx−1cosx=1Asin⁡x−1cos⁡x=1 and thus A=1A=1. Hence we have one way of writing this function:

y=sinx−cosx.y=sin⁡x−cos⁡x.

To write it as a single trigonometric function, we note this this looks like a sine graph which has been translated to the right by a little less than π3π3, so we could write it in the form y=Rsin(x−α)y=Rsin⁡(x−α), where R>0R>0 and 0<α<π20<α<π2.

Our initial idea that we could write a function of the form Asinx+BcosxAsin⁡x+Bcos⁡x as Rsin(x+α)Rsin⁡(x+α), where R>0R>0 and 0<α<π20<α<π2, therefore turns out to not be quite correct: we may have to translate the sine graph in the opposite direction. So maybe we can always write Asinx+BcosxAsin⁡x+Bcos⁡x as Rsin(x+α)Rsin⁡(x+α) where R>0R>0 and −π2<α<π2−π2<α<π2?

This new idea is certainly better; is it now correct, or does it need further modification?

Mathematicians frequently have initial ideas and then work to refine them by testing their ideas out with several well-chosen examples. These examples are designed to find the flaws in their initial ideas, which may sound a bit strange at first: don’t you want to show that your ideas work by finding lots of nice examples? Well, yes, but you also need to check that they doesn’t break on nasty ones!

We already know that our function is y=sinx−cosxy=sin⁡x−cos⁡x, and so we could begin by looking to work out αα.

Starting with

Rsin(x−α)=sinx−cosx,Rsin⁡(x−α)=sin⁡x−cos⁡x,
we see that if we substitute x=αx=α, then the left hand side is zero, and so we must have sinα=cosαsin⁡α=cos⁡α. As we are looking for 0<α<π20<α<π2, this gives us α=π4α=π4.

We now have Rsin(x−π4)=sinx−cosxRsin⁡(x−π4)=sin⁡x−cos⁡x. How can we find RR?

If we substitute in x=0x=0, we end up with something very nice, just as we did earlier:

Rsin(−π4)=−1.Rsin⁡(−π4)=−1.
Since sin(−π4)=−12√sin⁡(−π4)=−12, we deduce that R=2⎯⎯√R=2, hence
y=sinx−cosx=2⎯⎯√sin(x−π4).y=sin⁡x−cos⁡x=2sin⁡(x−π4).

Graph 2

From the graph we can see that when x=π2x=π2, Asinx+Bcosx=1Asin⁡x+Bcos⁡x=1, thus A=1A=1.

When x=π6x=π6, sinx+Bcosx=2sin⁡x+Bcos⁡x=2, so 12+B3√2=212+B32=2 and thus B=3⎯⎯√B=3.

Hence we have one way of writing this function:

y=sinx+3⎯⎯√cosx.y=sin⁡x+3cos⁡x.

The graph also looks like a translated sine graph: it has a maximum value of 22, and has been translated π3π3 to the left. So we can also write it as

y=2sin(x+π3).y=2sin⁡(x+π3).

Alternatively, we could think of it as a cosine graph shifted to the right by π6π6, giving

y=2cos(x−π6).y=2cos⁡(x−π6).

Graph 3

We can see that this graph is just a reflection of Graph 2 in the xx-axis. So we can immediately write down equations for the graph from our equations for Graph 2:

y=−sinx−3⎯⎯√cosx=−2sin(x+π3)=−2cos(x−π6).y=−sin⁡x−3cos⁡x=−2sin⁡(x+π3)=−2cos⁡(x−π6).
