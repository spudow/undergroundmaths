Equation or identity? (I)

Solution

Throughout, AA, BB and CC are the angles of a triangle.

For each of the following, decide whether it is an identity (true for all triangles) or an equation (there is a triangle for which it is not true).

If it is an identity, true for all triangles, then you should prove it (perhaps using trigonometric identities that you already know).

If it is an equation, then at the very least you should give an example of a triangle for which it is not true. You could also try to solve the equation (that is, find all triangles for which it is true).

  1. sin(π2−A)=cosAsin⁡(π2−A)=cos⁡A

    We have to decide whether to show that this is an identity or to solve the equation. To help us decide, we could try some particular values. We should choose values that will lead to sines and cosines that we can compute easily, but not such special cases that they run the risk of misleading us.

    If we try A=π3A=π3, the left-hand side becomes sin(π6)=12sin⁡(π6)=12 and the right-hand side is cos(π3)=12cos⁡(π3)=12. So the equation holds in this case.

    Doing that example might have reminded us of a useful diagram.

    We define cosAcos⁡A to be the adjacent over the hypotenuse. But we can see that this is precisely the sine of the other angle that isn’t a right angle, which is π2−Aπ2−A. So we have sin(π2−A)=cosAsin⁡(π2−A)=cos⁡A.

    Alternatively we could think about the connection between the graphs of sine and cosine.

  2. sec2A−tan2A=1sec2⁡A−tan2⁡A=1

    One of our key trigonometric identities is that sin2A+cos2A=1sin2⁡A+cos2⁡A=1 for all angles AA (not just those in a triangle). We might recognise the equation in the question as a common variant.

    We know the familiar identity

    sin2A+cos2A=1.sin2⁡A+cos2⁡A=1.

    Dividing through by cos2Acos2⁡A gives

    tan2A+1=sec2A,tan2⁡A+1=sec2⁡A,
    and we can rearrange this to
    sec2A−tan2A=1.sec2⁡A−tan2⁡A=1.

    We should be slightly more careful, however: we cannot divide by cos2Acos2⁡A if it is zero, and this could happen when AA is an angle of a triangle.

    The equation holds for all values of AA in the range except A=π2A=π2 (when the left-hand side is undefined).

  3. tan(A+B)=−tanCtan⁡(A+B)=−tan⁡C

    We might try some values here.

    We might also remember that AA, BB and CC are the angles of a triangle, so A+B+C=πA+B+C=π.

    We have

    tan(A+B)=tan(π−C)tan⁡(A+B)=tan⁡(π−C)
    whenever both sides are defined (that is, whenever CC is not a right angle).

    It may be convenient to refer to a sketch of y=tanxy=tan⁡x.

    We know that tan is a periodic function, with period ππ, so

    tan(π−C)=tan(−C).tan⁡(π−C)=tan⁡(−C).
    Since tan is an odd function, we have
    tan(−C)=−tanC.tan⁡(−C)=−tan⁡C.

    Putting this together, we find that

    tan(A+B)=−tanCtan⁡(A+B)=−tan⁡C
    for all triangles where CC is not a right angle.

  4. sin2A−cos2B=1sin2⁡A−cos2⁡B=1

    Let’s try some values to try to get a feel for whether this is an equation or an identity.

    If A=B=π4A=B=π4 (note that these are indeed possible angles in a triangle), then

    sin2A−cos2B=0.sin2⁡A−cos2⁡B=0.

    So it is not an identity; we should try to find the values for which the equation does hold.

    One difficulty about the given equation is that the two terms on the left differ in two ways: sine versus cosine, and AA versus BB. We could do something about that by replacing the 11 on the right-hand side by sin2A+cos2Asin2⁡A+cos2⁡A.

    If

    sin2A−cos2B=1,sin2⁡A−cos2⁡B=1,
    then
    sin2A−cos2B=sin2A+cos2A,sin2⁡A−cos2⁡B=sin2⁡A+cos2⁡A,
     so
    −cos2B=cos2A.−cos2⁡B=cos2⁡A.
     The right-hand side is always at least zero, whereas the left-hand side is always at most zero. The only way they can be the same is if they are both zero.

    So if AA and BB are part of a triangle that satisfies the equation, then cosA=cosB=0cos⁡A=cos⁡B=0.

    This means that A=B=π2A=B=π2, and this is not possible in a triangle (at least in a non-degenerate triangle).

    So there are no solutions to the equation.

  5. sin(π−A)=sinBsin⁡(π−A)=sin⁡B

    It’s reasonably clear that this cannot be an identity; it will be easy to think of examples of angles for which it does not hold, because there’s so much flexibility about choosing AA on the left-hand side and BB on the right-hand side. So we should try to solve the equation.

    We know that sin(π−A)=sinAsin⁡(π−A)=sin⁡A, so we are looking to solve

    sinA=sinB.sin⁡A=sin⁡B.
    This has the obvious solutions A=BA=B, but also the slightly less obvious solutions B=π−AB=π−A. The latter cannot happen for a (non-degenerate) triangle, so we see that a triangle satisfies the equation if and only if it is isosceles with A=BA=B.

  6. sin2A−3cosA=cosec2C−cot2Csin2⁡A−3cos⁡A=cosec2⁡C−cot2⁡C

    This is rather complicated. How might we simplify it? We might recognise the right-hand side as something that we could simplify.

    We have

    sin2C+cos2C=1,sin2⁡C+cos2⁡C=1,
    so dividing through by sin2Csin2⁡C gives
    1+cot2C=cosec2C,1+cot2⁡C=cosec2⁡C,
     and so
    cosec2C−cot2C=1.cosec2⁡C−cot2⁡C=1.
     This holds as long as sinCsin⁡C is not zero, and this will be true when CC is an angle of a (non-degenerate) triangle.

    So the equation we’re given is

    sin2A−3cosA=1.sin2⁡A−3cos⁡A=1.

    We can make this more manageable by replacing sin2Asin2⁡A with something involving cosAcos⁡A.

    This is equivalent to

    (1−cos2A)−3cosA=1,(1−cos2⁡A)−3cos⁡A=1,
    that is,
    cos2A+3cosA=0.cos2⁡A+3cos⁡A=0.
     Factorising, this becomes
    cosA(cosA+3)=0,cos⁡A(cos⁡A+3)=0,
     so cosA=0cos⁡A=0 or cosA=−3cos⁡A=−3.

    The latter option is not possible, so if we have a solution then we must have cosA=0cos⁡A=0 and so A=π2A=π2.

    We see that a triangle with angles AA, BB and CC satisfies the equation if and only if it is right-angled with a right angle at AA.

    Notice that although CC appears in the equation we’re given, whether or not there is a solution does not depend on CC.
