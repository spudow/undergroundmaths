  • If tanA=tanBtan⁡A=tan⁡B, what can we say about AA and BB?

Here is a graph of y=tanxy=tan⁡x, showing the solutions of tanx=atan⁡x=a.

You can slide the point on the yy-axis to change the value of aa.

Download GeoGebra file

How does this help you answer the question?

We can see from the graph that y=tanxy=tan⁡x has a period of ππ. This means that tan(x+nπ)=tanxtan⁡(x+nπ)=tan⁡x for all integers nn.

Therefore tanA=tanB=atan⁡A=tan⁡B=a if and only if A=B+nπA=B+nπ for some integer nn. In other words, AA and BB can only differ by an integer multiple of ππ.

If, for example, tanA=tanB=1tan⁡A=tan⁡B=1, then A=π4+nπA=π4+nπ for some integer nn. We’d also have B=π4+mπB=π4+mπ for some integer mm, and so

A=B+kπwhere k is an integer.A=B+kπwhere k is an integer.

More generally, if tanA=tanB=atan⁡A=tan⁡B=a, then A=arctana+nπA=arctan⁡a+nπ and B=arctana+mπB=arctan⁡a+mπ, where nn and mm are integers, so A=B+kπA=B+kπ where kk is an integer.

Please note that arctanarctan and tan−1tan−1 mean the same thing and both are widely used in books and online. Similarly, you may see arcsinarcsin or sin−1sin−1 and arccosarccos or cos−1cos−1.

  • If cosA=cosBcos⁡A=cos⁡B, what can we say about AA and BB?

Here is a graph of y=cosxy=cos⁡x, showing the solutions of cosx=acos⁡x=a.

Download GeoGebra file

We can see from the graph that y=cosxy=cos⁡x has a period of 2π2π. This means that cos(x+2nπ)=cosxcos⁡(x+2nπ)=cos⁡x for all integers nn.

Looking at the graph above, solutions to cosx=acos⁡x=a fall into two types (what we might call the ‘left-hand solutions’ and the ‘right-hand solutions’). For example, if a=12a=12, then we can call π3π3 a right-hand solution, and −π3−π3 a left-hand solution.

This means that any solution to cosx=acos⁡x=a is of the form

(a right-hand solution)+2nπor(a left-hand solution)+2nπfor some integer n.(a right-hand solution)+2nπor(a left-hand solution)+2nπfor some integer n.

For example, if cosx=12cos⁡x=12, then x=π3+2nπx=π3+2nπ or x=−π3+2nπx=−π3+2nπ for some integer nn.

Therefore, if cosA=cosB=acos⁡A=cos⁡B=a, then

A=arccosa+2nπorA=−arccosa+2nπ,A=arccos⁡a+2nπorA=−arccos⁡a+2nπ,
and
B=arccosa+2mπorB=−arccosa+2mπ,B=arccos⁡a+2mπorB=−arccos⁡a+2mπ,

where mm and nn are integers. This means that A=B+2kπA=B+2kπ or A=−B+2kπA=−B+2kπ, where kk is an integer.

  • If sinA=sinBsin⁡A=sin⁡B, what can we say about AA and BB?

Here is a graph of y=sinxy=sin⁡x, showing the solutions of sinx=asin⁡x=a.

The period of sinxsin⁡x is also 2π2π and again the solutions to sinx=asin⁡x=a fall into two types. For example, if a=12a=12, then we can call π6π6 a left-hand solution and π−π6π−π6 a right-hand solution. Any solution to sinx=asin⁡x=a is therefore of the form

(a right-hand solution)+2nπor(a left-hand solution)+2nπfor some integer n.(a right-hand solution)+2nπor(a left-hand solution)+2nπfor some integer n.

If sinA=sinB=asin⁡A=sin⁡B=a, then we can say AA and BB have the following forms

A=arcsina+2nπorA=π−arcsina+2nπ,where n is an integer,A=arcsin⁡a+2nπorA=π−arcsin⁡a+2nπ,where n is an integer,
B=arcsina+2mπorB=π−arcsina+2mπwhere m is an integer.B=arcsin⁡a+2mπorB=π−arcsin⁡a+2mπwhere m is an integer.

This means A=B+2kπA=B+2kπ or A=π−B+2kπA=π−B+2kπ, where kk is an integer.

We can think of y=sinxy=sin⁡x as a translation of y=cosxy=cos⁡x, by π2π2 in the positive xx direction. The symmetry of y=cosxy=cos⁡x about the yy-axis then means y=sinxy=sin⁡x is symmetric about x=π2x=π2, and the solutions for sinx=asin⁡x=a are symmetrical about this line. Therefore we could write the solutions of sinx=asin⁡x=a as

x=(arcsina−π2)+2nπ+π2orx=(π2−arcsina)+2nπ+π2.x=(arcsin⁡a−π2)+2nπ+π2orx=(π2−arcsin⁡a)+2nπ+π2.
