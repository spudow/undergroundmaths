Inverse or not?

Solution

Take a look at the three graphs below.

One of the graphs shows y=arctan(tanx)y=arctan⁡(tan⁡x). Another shows y=tan(arctanx)y=tan⁡(arctan⁡x). Which graphs are these, and why?

As tanxtan⁡x, sinxsin⁡x and cosxcos⁡x are periodic functions, there are many values of xx that give the same value of tanxtan⁡x, sinxsin⁡x or cosxcos⁡x. This means that inverse functions such as arctanxarctan⁡x and arcsinxarcsin⁡x have to be very carefully defined. You can read more about this in Inverse trigonometric functions, and these ideas are used in this solution.

Looking at the three graphs, I notice that Graph C looks like a graph of y=xy=x for all real values of x.x. In contrast, Graph B looks like a graph of y=xy=x but only for xx in the interval (−π2,−π2)(−π2,−π2). Having thought about Graph B, I can now think of Graph A as a repeating version of Graph B, with period ππ.

To see which graphs show the functions arctan(tanx)arctan⁡(tan⁡x) and tan(arctanx)tan⁡(arctan⁡x) I can think about domains and ranges.

When composing functions f(x)f(x) and g(x)g(x) to form g(f(x))g(f(x)), I need to think about the domain of f(x)f(x). I need to check that any output of ff is in the domain of gg and what these outputs are, so the range of g(f(x))g(f(x)) depends on the domain and range of ff as well as the range of gg.

Starting with the inner function in arctan(tanx)arctan⁡(tan⁡x), I know that tanxtan⁡x is defined for all real xx except where x=(2n+1)π2x=(2n+1)π2. This is the domain of tanxtan⁡x. The range is the whole of the real numbers. This means the input to the outer arctanarctan function is all the real numbers, so the output for arctanarctan is its principal value range, which is the interval (−π2,π2)(−π2,π2). Therefore, the graph of arctan(tanx)arctan⁡(tan⁡x) has a domain which is the whole of the xx-axis except the points where x=(2n+1)π2x=(2n+1)π2, and the range is (−π2,π2)(−π2,π2), so Graph A shows y=arctan(tanx)y=arctan⁡(tan⁡x).

I’ll now use similar thinking to work out which is the graph of y=tan(arctanx)y=tan⁡(arctan⁡x). This time arctanxarctan⁡x is the inner function. Its domain is the whole of the real numbers, but the range is (−π2,π2)(−π2,π2).

This graph shows y=tanxy=tan⁡x for xx in the interval (−π2,π2)(−π2,π2). Can you use it to explain why Graph C must show y=tan(arctanx)y=tan⁡(arctan⁡x)?

Match these equations to the graphs below and explain your reasoning.

One thing I notice is that all four of the functions map 00 to 00, so the graphs must pass through the origin. This eliminates Graph G.

1

y=arcsin(sinx)y=arcsin⁡(sin⁡x)

ResetShow all hidden cardsExport imagePrint

The domain of sinxsin⁡x is all real xx and the range is the interval [−1,1][−1,1]. The domain of arcsinxarcsin⁡x is the interval [−1,1][−1,1], and the principal value range is [−π2,π2][−π2,π2] (see Inverse trigonometric functions for more explanation).

So the function arcsin(sinx)arcsin⁡(sin⁡x) is defined for all real xx and has range [−π2,π2][−π2,π2]. Within the domain [−π2,π2][−π2,π2] the function will map xx to itself so the graph will look like that of y=xy=x. So this one must be Graph D.

Notice that as xx increases from π2π2 to 3π23π2, sinxsin⁡x decreases from 11 to −1−1 and so arcsin(sinx)arcsin⁡(sin⁡x) decreases from π2π2 to −π2−π2. And then this pattern continues, forming a zig-zag graph.

2

y=sin(arcsinx)y=sin⁡(arcsin⁡x)

ResetShow all hidden cardsExport imagePrint

The domain of arcsinxarcsin⁡x is the interval [−1,1][−1,1] and it is undefined elsewhere. Within this domain it has range [−π2,π2][−π2,π2], and these values as input to sinxsin⁡x produce values in the range [−1,1][−1,1]. So the graph will look like y=xy=x restricted to the domain −1≤x≤1−1≤x≤1, which is Graph E.

3

y=arccos(cosx)y=arccos⁡(cos⁡x)

ResetShow all hidden cardsExport imagePrint

cosxcos⁡x is defined for all real xx and has range [−1,1][−1,1]. arccosxarccos⁡x is defined on the domain [−1,1][−1,1] and its principal value range is [0,π][0,π]. So the domain of this composition is all the real numbers and its range is[0,π][0,π]. The only graph that matches this is J.

By way of a check, as xx increases from 00 to ππ, the value of cosxcos⁡x decreases from 11 to −1−1 and so arccos(cosx)arccos⁡(cos⁡x) increases from 00 to ππ. On this domain, the graph looks like y=xy=x. As xx increases further from ππ to 2π2π, cosxcos⁡x increases from −1−1 to 11 and arccos(cosx)arccos⁡(cos⁡x) decreases from ππ to 00. Hence we get a zig-zag pattern.

4

y=cos(arccosx)y=cos⁡(arccos⁡x)

ResetShow all hidden cardsExport imagePrint

arccosxarccos⁡x is defined only for xx in the interval [−1,1][−1,1]. It’s range is [0,π][0,π] and coscos of these values has range [−1,1][−1,1]. So our graph will look like y=xy=x restricted to the domain [−1,1][−1,1], and it must be Graph E, the same as for equation (2).

It is interesting to note that cosxcos⁡x and arccosxarccos⁡x are decreasing functions on these intervals but the composition of the functions is increasing. Is this the case for any pair of decreasing functions?

For which values of xx is tan(arctanx)=xtan⁡(arctan⁡x)=x? What about arctan(tanx)=xarctan⁡(tan⁡x)=x?

What can you say about the solutions of similar equations, such as sin(arcsinx)=xsin⁡(arcsin⁡x)=x or arccos(cosx)=xarccos⁡(cos⁡x)=x?

As Graph C is y=tan(arctanx)y=tan⁡(arctan⁡x), it must be that tan(arctanx)=xtan⁡(arctan⁡x)=x for all real xx.

Graph A shows y=arctan(tanx)y=arctan⁡(tan⁡x), so arctan(tanx)=xarctan⁡(tan⁡x)=x only for xx in the interval (−π2,π2)(−π2,π2) as this is the only part of the graph for which y=xy=x matches the graph of y=arctan(tanx)y=arctan⁡(tan⁡x).

To illustrate what happens if xx is outside this interval, let’s try x=4π3x=4π3. I know tan4π3=3⎯⎯√tan⁡4π3=3, but arctan3⎯⎯√=π3arctan⁡3=π3. Therefore arctan(tan4π3)=arctan3⎯⎯√=π3arctan⁡(tan⁡4π3)=arctan⁡3=π3. Generalising from this example, I can see that if xx is outside the interval (−π2,π2)(−π2,π2) and tanxtan⁡x is defined, then arctan(tanx)arctan⁡(tan⁡x) takes xx to the corresponding value inside the interval (−π2,π2)(−π2,π2).

We can draw corresponding conclusions for the sinsin and coscos compositions by looking at graphs D, E and J.
