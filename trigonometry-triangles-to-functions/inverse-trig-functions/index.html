Inverse trigonometric functions
Add to your resource collection

Printable/supporting materials Fullscreen mode Teacher notes

Imagine you were told that in triangle ABCABC, the side opposite CC has length 5cm5cm, the side opposite AA has length 3cm3cm and angle A=π6A=π6. Think about how you would calculate angle CC.

At some point in your calculation you might obtain sinC=56sin⁡C=56, but what exactly does this tell you about CC?

You may have heard of inverse trigonometric functions, such as arcsinxarcsin⁡x or sin−1xsin−1⁡x or the phrase “inverse sine”. In principle, inverse trigonometric functions should be functions that undo the effects of functions such as sinxsin⁡x, cosxcos⁡x and tanxtan⁡x. But we need to be careful because these functions are are not one-to-one; in fact they are periodic, so for instance there are infinitely many values of xx that give the same value of sinxsin⁡x.

You might wonder where the names such as ‘arcsin’ come from.

Recall that in a circle of radius rr, the arc length subtended by an angle θθ radians has length rθrθ. So on a unit circle the length of an arc and the size of the angle which subtends it are equal. This is probably where the words come from.

Please note that arcsinarcsin and sin−1sin−1 mean the same thing and both are widely used in books and online, but note that sin−1xsin−1⁡x is not the same as 1sinx1sin⁡x! Similarly, arccosarccos means the same as cos−1cos−1, and arctanarctan means the same as tan−1tan−1. We’ll use arctanarctan, arccosarccos and arcsinarcsin in this resource.

Although we started with a situation where we were thinking about an inverse for sinxsin⁡x, we’ll think briefly about tanxtan⁡x first before moving back to sinxsin⁡x and cosxcos⁡x. Some of the issues that arise for an inverse of tanxtan⁡x also arise for sinxsin⁡x and cosxcos⁡x, but we’ll see that we have to be even more careful for these functions.

Recall from Making inverse functions that if we want to produce an inverse for a many-to-one function, we need to restrict the domain so that the function is one-to-one on its restricted domain. We also need the range of the restricted function to be the same as the range of the original function.

So for tanxtan⁡x,

  • How could we restrict the domain to make the function one-to-one whilst keeping its range unchanged?

  • Is there more than one way to do this?

The function tanxtan⁡x is periodic with period ππ. Restricting the domain to any one period will not change the range of the function.

From the graph of y=tanxy=tan⁡x we can see that the function is always increasing. If we choose a domain between two adjacent asymptotes, such as the interval (−π2,π2)(−π2,π2), the function will be one-to-one and its range will still include all real numbers.

We could choose any interval of the form ((2n−1)π2,(2n+1)π2)((2n−1)π2,(2n+1)π2) for integer values of nn, but it seems most natural to choose the interval that includes zero.

The following graphs shows how we could restrict the domain of tanxtan⁡x and the resulting function arctanxarctan⁡x.

So although the equation tanx=3⎯⎯√tan⁡x=3 has infinitely many solutions, we say arctan3⎯⎯√arctan⁡3 has the single value π3π3, and this is called the principal value for the equation tanx=3⎯⎯√tan⁡x=3. It is the value of xx in the interval (−π2,π2)(−π2,π2) such that tanx=3⎯⎯√tan⁡x=3. The interval (−π2,π2)(−π2,π2) is known as the principal value range of the function arctanarctan.

We can use the symmetry and periodicity of the tanxtan⁡x graph to express all solutions of tanx=3⎯⎯√tan⁡x=3 in terms of arctan3⎯⎯√arctan⁡3.

We’ll now return to thinking about sinxsin⁡x and cosxcos⁡x.

The period of sinxsin⁡x is 2π2π, but within any interval of length 2π2π, there are two values of xx that give the same value of sinx.sin⁡x. In other words, sinxsin⁡x is not one-to-one or injective over its period. So in order to define an inverse we need to restrict to a smaller interval.

If we restrict the domain of sinxsin⁡x to the interval [−π2,π2][−π2,π2], then the resulting function is one-to-one and its range is the same as that of the unrestricted function. Its inverse is arcsinxarcsin⁡x which has domain [−1,1][−1,1].

The situation for cosxcos⁡x is very similar to that for sinxsin⁡x but we need to restrict to a different interval.

If we restrict the domain of cosxcos⁡x to [0,π][0,π], then the range is still [−1,1][−1,1] and cosxcos⁡x is now one-to-one. We use this domain to define arccosxarccos⁡x with domain [−1,1][−1,1] and range [0,π][0,π]. Note that both cosxcos⁡x (restricted to this domain) and arccosxarccos⁡x are decreasing functions.

So arctanaarctan⁡a, arcsinaarcsin⁡a and arccosaarccos⁡a give the principal values for the equations tanx=atan⁡x=a, sinx=asin⁡x=a and cosx=acos⁡x=a respectively, and we can work from these to find all possible solutions. You can see more about this in General solutions and in Inverse or not?.

Going back to our original triangle problem, we had sinC=56sin⁡C=56. The inverse function will give us the principal value (in radians) as

C=arcsin56≈0.985.C=arcsin⁡56≈0.985.
But there is more than one value of CC that would satisfy our equation. We must always consider other possibilities when using inverse trigonometric functions.

Since we are talking about an angle in a triangle we need only consider values in the interval [0,π][0,π]. There is one more possible value which is

π−arcsin56≈2.156.π−arcsin⁡56≈2.156.
