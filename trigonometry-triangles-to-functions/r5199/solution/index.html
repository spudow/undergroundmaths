Review question

When is sinxsin⁡x or sin2xsin⁡2x larger than a half?

Ref: R5199

Solution

The fraction of the interval 0≤x≤2π0≤x≤2π, for which one (or both) of the inequalities

sinx≥12,sin2x≥12sin⁡x≥12,sin⁡2x≥12
is true, equals

  1. 1313;

  2. 13241324;

  3. 712712;

  4. 5858.

In the range 0≤x≤2π0≤x≤2π we have sinx≥12sin⁡x≥12 if and only if π6≤x≤5π6,π6≤x≤5π6,

and sin2x≥12sin⁡2x≥12 if and only if

π12≤x≤5π12or13π12≤x≤17π12.π12≤x≤5π12or13π12≤x≤17π12.

So at least one of the inequalities holds for π12≤x≤5π6π12≤x≤5π6 and 13π12≤x≤17π1213π12≤x≤17π12.

The first of these intervals has length 9π129π12 and the second has length 4π124π12.

Therefore the fraction of the interval 0≤x≤2π0≤x≤2π, for which one (or both) of the inequalities is true, is 13π1213π12 out of 2π2π, which is 13241324. Hence, the solution is (b).
