Trigonometry: Triangles to Functions
Review question

Can we prove this trig identity to solve this equation?

Solution

  1. Prove that
    sinA1+cosA+1+cosAsinA=2sinAsin⁡A1+cos⁡A+1+cos⁡Asin⁡A=2sin⁡A
     for all values of AA.
We will begin by writing the left-hand side as a single fraction.
sinA1+cosA+1+cosAsinA=sin2A+(1+cosA)2(1+cosA)sinA=sin2A+1+2cosA+cos2A(1+cosA)sinA.sin⁡A1+cos⁡A+1+cos⁡Asin⁡A=sin2⁡A+(1+cos⁡A)2(1+cos⁡A)sin⁡A=sin2⁡A+1+2cos⁡A+cos2⁡A(1+cos⁡A)sin⁡A.
 For any AA, we have the trigonometric identity
sin2A+cos2A=1sin2⁡A+cos2⁡A=1
 and so we can write
sin2A+1+2cosA+cos2A(1+cosA)sinA=2+2cosA(1+cosA)sinA=2(1+cosA)(1+cosA)sinA=2sinAsin2⁡A+1+2cos⁡A+cos2⁡A(1+cos⁡A)sin⁡A=2+2cos⁡A(1+cos⁡A)sin⁡A=2(1+cos⁡A)(1+cos⁡A)sin⁡A=2sin⁡A

as we had hoped to show.

  1. Find the values of xx between 0∘0∘ and 360∘360∘ if
    sinx1+cosx+1+cosxsinx=1+3sinx.sin⁡x1+cos⁡x+1+cos⁡xsin⁡x=1+3sin⁡x.

From the previous part of the question, xx satisfies

sinx1+cosx+1+cosxsinx=1+3sinx.sin⁡x1+cos⁡x+1+cos⁡xsin⁡x=1+3sin⁡x.
exactly when
2sinx=1+3sinx.2sin⁡x=1+3sin⁡x.
 Since this never holds when sinx=0sin⁡x=0, we can multiply through by sinxsin⁡x and rearrange to produce
3sin2x+sinx−2=0.3sin2⁡x+sin⁡x−2=0.
 This is a quadratic equation in sinxsin⁡x; we can factorise it as
(3sinx−2)(sinx+1)=0.(3sin⁡x−2)(sin⁡x+1)=0.
 Thus xx satisfies the original equation given in the question exactly when
sinx=−1orsinx=23.sin⁡x=−1orsin⁡x=23.

The unique solution to the first of these possibilities with 0∘≤x≤360∘0∘≤x≤360∘ is x=270∘x=270∘.

The second possibility gives two solutions with 0∘≤x≤360∘0∘≤x≤360∘.

We have sin−1(23)sin−1⁡(23), which lies between 0∘0∘ and 90∘90∘, and 180∘−sin−1(23)180∘−sin−1⁡(23), which lies between 90∘90∘ and 180∘180∘.

So there are three possible solutions in total, 41.8∘,138.2∘41.8∘,138.2∘, and 270∘270∘.
