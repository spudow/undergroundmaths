What is the position vector of DD if ABCDABCD is a parallelogram?

  1. Find the value of pp such that AA, BB and CC are collinear.

The points A,BA,B and CC are collinear ⟺AB−→⟺AB→ is parallel to AC−→−⟺AB−→=kAC−→−AC→⟺AB→=kAC→ for some constant kk.

Now

AB−→=−3i+12j+6k,AC−→−=(p−4)i+8j+4k,AB→=−3i+12j+6k,AC→=(p−4)i+8j+4k,

and thus

2AB−→=−6i+24j+12k,3AC−→−=3(p−4)i+24j+12k.2AB→=−6i+24j+12k,3AC→=3(p−4)i+24j+12k.

Comparing coefficients, we can say that A,BA,B and CC are collinear if and only if −6=3(p−4)−6=3(p−4), which is true if and only if p=2p=2.

Alternatively, we could you use the fact that CC lies on the straight line ABAB. We could find the equation of the line ABAB and then substitute CC into this equation to find pp.
