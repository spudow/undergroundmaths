Things you might have noticed

A ball is fired from ground level, with a fixed initial velocity, uu, at an angle αα to the ground. Assuming no one intercepts the ball, describe how the position at which it lands will vary with the angle of projection.

The sketch below is a reasonable summary of the problem. We are interested in how SS varies with αα.

We might like to begin to get a feel for the problem by thinking about some specific cases.

What could you say about the case where u=5ms−1u=5ms−1 and α=30∘,40∘,80∘α=30∘,40∘,80∘?

Can you sketch the three cases on the same axes?

  • What do you expect to stay the same in all three cases?
  • What do you expect to change and in what way?

(The values of SS marked on these graphs are based on g=10ms−2g=10ms−2)

What do you think about these values? Are there any other angles that you would like to try?

In order to address the original problem we need to try to use the above experiences to help us to describe the situation in more general terms of SS, uu and αα.

Thinking carefully about how we calculate the horizontal displacement of the ball for particular values of uu and αα, we can apply the same reasoning to determine that

S=ucosα×2usinαg.S=ucos⁡α×2usin⁡αg.

Where does each of the factors in this expression come from?

This can be simplified to give

S=(u2g)sin2α.S=(u2g)sin⁡2α.

How did we simplify the expression for SS?

It is often useful to consider the units in an expression like this. We know that the distance, SS, should have units m. Does the right-hand side of the formula have the same units?

Consider the landing position that is furthest away:

  • How much faster will the ball need to be fired so that it lands 1m1m further away than this?

(Here we use ww to represent the new initial velocity of the ball)

We know that the landing position of the ball this time will be at w2gw2g and that this must be equal to the original landing position plus 1m1m. We can therefore write

w2g=u2g+1m.w2g=u2g+1m.

We can rearrange this to show the new velocity, ww, in terms of the old velocity,uu,

w=u2+g⎯⎯⎯⎯⎯⎯⎯⎯⎯⎯√.w=u2+g.

How would we adapt this formula if we wanted the ball to land 2m2m away? What about dmdm away?
